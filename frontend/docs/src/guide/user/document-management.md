| 文档列表 | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ |
| 我的文档 | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ |

## 文档列表

**菜单位置**：卷宗管理 → 文档列表

查看和管理项目文档。

### 文档操作

- **上传文档**：上传项目相关文档
- **在线预览**：支持 Word、Excel、PDF 等格式在线预览
- **下载**：下载文档到本地
- **删除**：删除文档（需权限）

### 文档分类

文档按项目组织，可以在项目详情页的「文档」标签页管理。

---

## 我的文档

**菜单位置**：卷宗管理 → 我的文档

只显示自己上传或有权限访问的文档。

---

## 用印管理

**菜单位置**：卷宗管理 → 用印管理

管理印章使用记录（行政人员）。

### 用印记录字段

| 列名 | 说明 |
|------|------|
| 申请编号 | 系统自动生成 |
| 申请人 | 申请用印的人 |
| 印章类型 | 公章/合同章/法人章等 |
| 用印事由 | 用印原因 |
| 用印份数 | 需要盖章的份数 |
| 状态 | 待审批/已审批/已用印/已拒绝 |

---

## 用印申请

**菜单位置**：卷宗管理 → 用印申请

提交用印申请。

### 申请用印

1. 点击「新增申请」
2. 填写申请信息：

| 字段 | 必填 | 说明 |
|------|:----:|------|
| 印章类型 | ✅ | 公章/合同章/法人章 |
| 关联项目 | - | 可选关联项目 |
| 用印事由 | ✅ | 说明用印原因 |
| 用印份数 | ✅ | 需要盖章份数 |
| 备注 | - | 其他说明 |

3. 提交审批

### 用印审批

用印申请需要主任或管理员审批。

## 文档模板

**菜单位置**：卷宗管理 → 文档模板

管理文档模板，用于快速生成标准化文档。

### 模板类型

- 合同模板
- 法律意见书模板
- 起诉状模板
- 答辩状模板
- 其他文书模板

### 使用模板

1. 选择模板
2. 填写变量信息
3. 生成文档
4. 下载或保存到项目

---

## AI 撰写

**菜单位置**：卷宗管理 → AI 撰写

使用 AI 辅助撰写法律文书。

### 撰写模式

- **模板模式**：基于模板生成文档
- **AI 模式**：AI 辅助撰写

### 使用步骤

1. 选择项目或案件
2. 选择文档类型
3. 填写必要信息
4. 生成文档
5. 预览和编辑
6. 保存

---

## 常见问题

**Q: 支持哪些文件格式预览？**

支持 Word (.doc, .docx)、Excel (.xls, .xlsx)、PDF、图片等常见格式。

**Q: 文件大小有限制吗？**

单个文件最大支持 100MB。

**Q: 用印申请需要多久审批？**

审批时间取决于审批人，建议提前申请。

**Q: AI 撰写是否准确？**

AI 生成的内容仅供参考，请务必人工审核后使用。
