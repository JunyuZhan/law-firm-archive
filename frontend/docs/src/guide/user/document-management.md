## 文档列表

**菜单位置**：卷宗管理 → 文档列表

查看和管理项目文档。

### 文档操作

- **上传文档**：上传项目相关文档
- **在线预览**：支持 Word、Excel、PDF 等格式在线预览
- **下载**：下载文档到本地
- **删除**：删除文档（需权限）

### 文档分类

文档按项目组织，可以在项目详情页的「文档」标签页管理。

## 我的文档

**菜单位置**：卷宗管理 → 我的文档

只显示自己上传或有权限访问的文档。

## 印章管理

**菜单位置**：卷宗管理 → 印章管理

管理律所印章信息（行政人员）。

### 查询筛选

| 筛选项   | 说明                                               |
| -------- | -------------------------------------------------- |
| 印章名称 | 模糊搜索印章名称（支持输入后按回车或点击查询按钮） |
| 印章类型 | 公章/合同章/财务章/法人章（支持清空选择）          |
| 状态     | 全部/可用/停用                                     |

### 印章列表字段

| 列名     | 说明                                                      |
| -------- | --------------------------------------------------------- |
| 印章名称 | 印章名称（点击可查看详情）                                |
| 印章类型 | 公章（蓝色）/合同章（绿色）/财务章（橙色）/法人章（紫色） |
| 保管人   | 印章保管人（显示姓名）                                    |
| 使用次数 | 印章使用次数统计（用于用印申请）                          |
| 状态     | 可用（绿色标签）/停用（灰色标签）                         |
| 创建时间 | 印章创建时间（YYYY-MM-DD格式）                            |
| 操作     | 编辑/启用停用/删除                                        |

### 场景一：添加印章

1. 点击「添加印章」按钮
2. 填写印章信息：

| 字段     | 必填 | 说明                      |
| -------- | :--: | ------------------------- |
| 印章名称 |  ✅  | 印章名称                  |
| 印章类型 |  ✅  | 公章/合同章/财务章/法人章 |
| 保管人   |  -   | 选择保管人                |

3. 点击「确定」保存

### 场景二：编辑印章

1. 在印章列表中找到要编辑的印章
2. 点击「编辑」
3. 修改印章信息
4. 点击「确定」保存

### 场景三：启用/停用印章

1. 找到要操作的印章
2. 点击「停用」或「启用」
3. 状态变更成功

### 场景四：删除印章

1. 点击「删除」按钮
2. 确认删除

## 用印申请

**菜单位置**：卷宗管理 → 用印申请

提交用印申请。

### 申请用印

1. 点击「新增申请」
2. 填写申请信息：

| 字段     | 必填 | 说明               |
| -------- | :--: | ------------------ |
| 印章类型 |  ✅  | 公章/合同章/法人章 |
| 关联项目 |  -   | 可选关联项目       |
| 用印事由 |  ✅  | 说明用印原因       |
| 用印份数 |  ✅  | 需要盖章份数       |
| 备注     |  -   | 其他说明           |

3. 提交审批

### 用印审批

用印申请需要主任或管理员审批。

## 文书模板

**菜单位置**：卷宗管理 → 文书模板

管理文书模板库，支持变量替换自动生成文书。

### 查询筛选

| 筛选项   | 说明                              |
| -------- | --------------------------------- |
| 模板名称 | 模糊搜索模板名称                  |
| 模板类型 | Word文档/Excel表格/PDF文档/富文本 |
| 适用业务 | 诉讼案件/非诉项目/通用            |
| 状态     | 启用/停用                         |

### 模板列表字段

| 列名     | 说明                                                  |
| -------- | ----------------------------------------------------- |
| 模板名称 | 模板名称和描述                                        |
| 模板类型 | Word（蓝色）/Excel（绿色）/PDF（红色）/富文本（紫色） |
| 适用业务 | 诉讼案件/非诉项目/通用                                |
| 创建人   | 创建模板的人                                          |
| 创建时间 | 创建时间                                              |
| 使用次数 | 模板被使用的次数                                      |
| 状态     | 启用（绿色）/停用（灰色）                             |

### 场景五：新建模板

1. 点击「新建模板」按钮
2. 填写模板信息：

| 字段     | 必填 | 说明                              |
| -------- | :--: | --------------------------------- |
| 模板名称 |  ✅  | 模板名称                          |
| 模板类型 |  ✅  | Word文档/Excel表格/PDF文档/富文本 |
| 适用业务 |  -   | 诉讼案件/非诉项目/通用            |
| 模板内容 |  ✅  | 模板正文内容，支持变量            |
| 描述     |  -   | 模板说明                          |

3. 点击「确定」保存

### 场景六：预览模板

1. 点击模板名称或「预览」按钮
2. 查看模板内容

### 场景七：编辑模板

1. 点击「编辑」按钮
2. 修改模板信息
3. 点击「确定」保存

### 场景八：复制模板

1. 点击「复制」按钮
2. 系统自动创建一个副本（名称后缀「- 副本」）

### 场景九：启用/停用模板

1. 点击「停用」或「启用」按钮
2. 状态变更成功

### 场景十：删除模板

1. 点击「删除」按钮
2. 确认删除（删除后不可恢复）

## AI 撰写

**菜单位置**：卷宗管理 → AI 撰写

使用 AI 辅助撰写法律文书。

### 撰写模式

- **模板模式**：基于模板生成文档
- **AI 模式**：AI 辅助撰写

### 使用步骤

1. 选择项目或案件
2. 选择文档类型
3. 填写必要信息
4. 生成文档
5. 预览和编辑
6. 保存

## 常见问题

**Q: 支持哪些文件格式预览？**

支持 Word (.doc, .docx)、Excel (.xls, .xlsx)、PDF、图片等常见格式。

**Q: 文件大小有限制吗？**

单个文件最大支持 100MB。

**Q: 用印申请需要多久审批？**

审批时间取决于审批人，建议提前申请。

**Q: AI 撰写是否准确？**

AI 生成的内容仅供参考，请务必人工审核后使用。

**Q: 模板支持哪些变量？**

模板支持项目信息、客户信息、日期等变量，在生成文档时会自动替换为实际值。

**Q: 停用的模板还能使用吗？**

停用的模板不会出现在模板选择列表中，但已使用该模板生成的文档不受影响。
