# 客户管理

客户管理模块用于管理律所的客户信息，包括客户录入、利冲审查、案源管理。

## 谁能使用？

| 功能 | 管理员 | 主任 | 团队负责人 | 财务 | 律师 | 行政 | 实习 |
|------|:-----:|:----:|:--------:|:----:|:----:|:----:|:----:|
| 客户列表 | ✅ | ✅ | ✅ | ✅(查看) | ✅ | ✅(查看) | ❌ |
| 利冲审查 | ✅ | ✅ | ✅ | ❌ | ✅ | ❌ | ❌ |
| 案源管理 | ✅ | ✅ | ✅ | ❌ | ✅ | ❌ | ❌ |

---

## 客户列表

**菜单位置**：客户管理 → 客户列表

### 查询筛选

| 筛选项 | 说明 |
|--------|------|
| 年份 | 按创建年份筛选，默认当前年 |
| 客户名称 | 模糊搜索客户名称 |
| 客户类型 | 企业客户/个人客户/政府机关/其他 |
| 客户级别 | 重要客户(A)/普通客户(B)/一般客户(C) |
| 状态 | 潜在/正式/休眠/黑名单 |

### 客户列表字段

| 列名 | 说明 |
|------|------|
| 客户编号 | 系统自动生成 |
| 客户名称 | 企业名称或个人姓名 |
| 客户类型 | 企业/个人/政府/其他 |
| 联系人 | 联系人姓名 |
| 联系电话 | 联系电话 |
| 客户级别 | A/B/C 级别 |
| 客户分类 | VIP/普通/潜在 |
| 状态 | 当前状态 |
| 案源人 | 引入该客户的律师 |
| 负责律师 | 负责该客户的律师 |
| 创建时间 | 录入时间 |

### 新增客户

1. 点击「新增」按钮
2. 填写客户信息：

**基本信息**：

| 字段 | 必填 | 说明 |
|------|:----:|------|
| 客户名称 | ✅ | 企业全称或个人姓名 |
| 客户类型 | ✅ | 企业/个人/政府/其他 |
| 统一社会信用代码 | - | 企业客户填写 |
| 身份证号 | - | 个人客户填写 |
| 法定代表人 | - | 企业客户填写 |

**联系信息**：

| 字段 | 必填 | 说明 |
|------|:----:|------|
| 联系人 | - | 联系人姓名 |
| 联系电话 | - | 联系电话 |
| 联系邮箱 | - | 电子邮箱 |

**业务信息**：

| 字段 | 必填 | 说明 |
|------|:----:|------|
| 客户级别 | ✅ | 重要(A)/普通(B)/一般(C) |
| 客户分类 | ✅ | VIP/普通/潜在 |
| 行业 | - | 所属行业 |
| 来源 | - | 客户来源渠道 |
| 案源人 | - | 引入客户的律师 |
| 负责律师 | - | 负责该客户的律师 |
| 备注 | - | 其他说明 |

3. 系统会自动进行**快速利冲检查**
4. 如果存在利益冲突风险，会提示确认
5. 点击「确定」保存

### OCR 识别（如已启用）

新增客户时可以使用 OCR 功能自动识别证件信息：
- **身份证识别**：自动填充姓名、身份证号
- **营业执照识别**：自动填充企业名称、信用代码、法人

### 编辑客户

1. 点击客户行的「编辑」按钮
2. 修改客户信息
3. 点击「确定」保存

### 删除客户

1. 点击「删除」按钮
2. 确认删除

::: warning 注意
已关联项目或合同的客户不能删除。
:::

### 批量删除

1. 勾选要删除的客户
2. 点击「批量删除」按钮
3. 确认删除

### 转为正式客户

潜在客户可以转为正式客户：
1. 找到状态为「潜在」的客户
2. 点击「转为正式」按钮

## 利冲审查

**菜单位置**：客户管理 → 利冲审查

利益冲突审查是接案前的必要步骤，确保新案件与现有案件没有利益冲突。

### 利冲检索

1. 输入要检索的关键词（客户名称、身份证号、企业信用代码等）
2. 点击「检索」
3. 系统会自动匹配现有客户和项目，返回可能的冲突信息

### 检索结果

| 风险等级 | 颜色 | 说明 |
|---------|------|------|
| 高风险 | 红色 | 存在明显利益冲突 |
| 中风险 | 橙色 | 需要进一步核实 |
| 低风险 | 绿色 | 基本无冲突 |

### 匹配类型

| 类型 | 说明 |
|------|------|
| 完全匹配 | 名称完全一致 |
| 包含匹配 | 名称包含关键词 |
| 相似匹配 | 名称相似 |

## 案源管理

**菜单位置**：客户管理 → 案源管理

管理案源线索，跟踪潜在业务机会。

### 案源列表

查看所有案源线索，包括：
- 案源编号
- 客户信息
- 案源类型
- 预估金额
- 跟进状态
- 负责人

### 创建案源

1. 点击「新建案源」
2. 填写案源信息
3. 保存

### 跟进状态

| 状态 | 说明 |
|------|------|
| 待跟进 | 新建的案源 |
| 跟进中 | 正在沟通 |
| 已转化 | 已成功转为项目 |
| 已关闭 | 未成功转化 |

## 常见问题

**Q: 客户编号是怎么生成的？**

客户编号由系统自动生成，格式为 `C + 年份 + 序号`，如 `C2026010001`。

**Q: 已删除的客户可以恢复吗？**

删除的客户不能恢复，请谨慎操作。如果需要隐藏客户，可以将状态改为「休眠」。

**Q: 利冲检索会检查哪些内容？**

系统会检查客户名称、身份证号、企业信用代码与现有客户和项目的对方当事人是否匹配。

**Q: 一个客户可以有多个联系人吗？**

目前系统只支持一个主要联系人。如果有多个联系人，可以在备注中说明。
