# 客户管理

在接手任何案件之前，你需要先在系统中录入客户信息。这一步很重要，因为后续的合同、项目都要关联到客户。

## 谁能使用客户管理？

| 功能 | 管理员 | 主任 | 团队负责人 | 财务 | 律师 | 行政 | 实习 |
|------|:-----:|:----:|:--------:|:----:|:----:|:----:|:----:|
| 客户列表 | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ |
| 利冲审查 | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ |
| 案源管理 | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ❌ |

::: tip 说明
客户管理对所有角色开放，包括实习律师。但实习律师不能访问案源管理。
:::

---

## 进入客户管理

点击左侧菜单「客户管理」→「客户列表」

页面标题为「客户管理」，描述为「管理客户信息」。

你会看到一个客户列表页面，显示所有已录入的客户信息。

---

## 场景一：有新客户来咨询，我要录入系统

### 第一步：点击新增

在客户列表页面，点击「新增客户」按钮。

### 第二步：使用智能识别（可选）

::: tip OCR智能识别
系统支持通过OCR技术自动识别证件信息，大幅提升录入效率。
:::

在新增客户弹窗顶部，你会看到「智能识别填充」区域，提供两种识别方式：

| 识别类型 | 适用场景 | 自动填充字段 |
|---------|---------|-------------|
| 营业执照识别 | 企业客户 | 公司名称、统一社会信用代码、法定代表人、联系人 |
| 身份证识别 | 个人客户 | 客户姓名、身份证号、联系人 |

**使用方法：**

1. 点击「营业执照识别」或「身份证识别」按钮
2. 选择证件照片（支持 JPG、PNG 等格式）
3. 等待系统识别（通常几秒钟）
4. 识别成功后，相关字段会自动填充
5. 核对并补充其他信息

**识别结果说明：**
- 识别成功后会显示置信度百分比
- 如果关键字段未能自动提取，系统会显示原始识别文本供参考
- 建议核对自动填充的信息，确保准确性

::: warning 图片质量要求
- 图片应清晰、完整，避免模糊或遮挡
- 证件应平整，避免折痕或反光
- 如识别效果不佳，可尝试重新拍照
:::

### 第三步：填写基本信息

系统会弹出一个表单，根据客户类型填写不同的信息：

::: tip 智能识别提示
如果使用了OCR智能识别，部分字段已自动填充，只需核对并补充其他信息即可。
:::

#### 如果是企业客户

| 字段 | 说明 | 是否必填 |
|------|------|---------|
| 客户类型 | 选择「企业客户」 | 必填 |
| 客户名称 | 公司全称，如「北京某某科技有限公司」 | 必填 |
| 统一社会信用代码 | 18位代码，方便后续利冲检索 | 建议填写 |
| 法定代表人 | 公司法人姓名 | 建议填写 |
| 联系人 | 对接人姓名 | 建议填写 |
| 联系电话 | 对接人电话 | 建议填写 |
| 联系邮箱 | 对接人邮箱 | 选填 |
| 所属行业 | 如「互联网」「制造业」 | 选填 |

#### 如果是个人客户

| 字段 | 说明 | 是否必填 |
|------|------|---------|
| 客户类型 | 选择「个人客户」 | 必填 |
| 客户名称 | 客户姓名 | 必填 |
| 身份证号 | 18位身份证号，方便后续利冲检索 | 建议填写 |
| 联系电话 | 客户电话 | 建议填写 |
| 联系邮箱 | 客户邮箱 | 选填 |

#### 其他客户类型

系统还支持「政府机关」和「其他」类型的客户。

### 第三步：填写业务信息

| 字段 | 说明 | 如何选择 |
|------|------|---------|
| 客户级别 | 重要客户/普通客户/一般客户 | 重要客户=大客户、长期合作<br>普通客户=一般业务<br>一般客户=小额业务 |
| 客户分类 | 重要客户/普通客户/潜在客户 | 根据客户重要程度选择 |
| 案源人 | 谁介绍来的这个客户 | 从部门树选择律师<br>**重要：影响后续提成计算** |
| 负责律师 | 谁负责跟进这个客户 | 从部门树选择律师 |
| 备注 | 其他需要记录的信息 | 选填 |

### 第四步：利冲检索（重要！）

::: danger 为什么利冲检索这么重要？
律师不能同时代理利益对立的双方。比如你不能一边帮张三告李四，一边又帮李四告张三。这是律师执业的基本规则，违反可能导致执业处分。
:::

如果这个客户要委托你打官司，你需要检查是否存在利益冲突：

1. 在表单下方找到「利益冲突检索」区域（分隔线下方）
2. 在「对方当事人」输入框中，填写案件的对方是谁
   - 如果对方是公司，填写公司名称
   - 如果对方是个人，填写姓名
3. 点击「🔍 利冲检索」按钮
4. 等待系统检索结果

**检索结果解读**：

| 结果 | 显示 | 下一步 |
|------|------|--------|
| ✅ 未检测到利益冲突 | 绿色成功提示框 | 可以放心创建客户 |
| ❌ 检测到利益冲突 | 红色错误提示框，显示冲突详情 | 需要谨慎处理 |

**如果检测到利益冲突**：

这意味着我们可能不能接这个案子。你有以下选择：

1. **放弃这个案子**：这是最安全的做法
2. **申请利冲豁免**：点击红色提示框中的「申请利冲豁免」按钮，跳转到利冲申请页面

::: warning 提示
如果你填写了对方当事人但没有进行利冲检索，系统会显示黄色警告提示，提醒你先进行检索。
:::

### 第五步：保存

确认信息无误后，点击「确定」按钮保存客户信息。

::: tip 保存限制
如果填写了对方当事人，必须先进行利冲检索且无冲突，才能保存。
:::

保存成功后：
- 系统会自动生成客户编号
- 客户状态默认为「潜在」
- 你可以在客户列表中看到新创建的客户

## 场景二：我想找一个老客户的信息

### 方法一：按年份筛选

1. 在客户列表页面顶部，找到「创建年份」下拉框
2. 选择年份（默认显示当前年份的客户）
3. 选择「全部年份」可以查看所有客户

### 方法二：搜索

1. 在搜索框中输入客户名称（支持模糊搜索）
2. 点击「查询」按钮

**搜索技巧**：
- 企业客户：可以输入公司名称的关键词，如「某某科技」
- 个人客户：可以输入姓名

### 方法三：筛选

如果你记不清客户名字，可以用筛选条件缩小范围：

| 筛选条件 | 用途 |
|---------|------|
| 创建年份 | 按客户创建年份筛选（默认当前年份） |
| 客户类型 | 企业客户/个人客户/政府机关/其他 |
| 客户级别 | 重要客户/普通客户/一般客户 |
| 状态 | 潜在/正式/休眠/黑名单 |

选择筛选条件后，点击「查询」按钮。点击「重置」可以清空所有筛选条件。

## 场景三：客户信息变了，我要修改

客户的联系方式、联系人等信息可能会变化，需要及时更新。

### 修改步骤

1. 在客户列表中找到要修改的客户
2. 点击该行右侧的「编辑」按钮
3. 在弹出的表单中修改需要更新的信息
4. 点击「确定」保存

### 可以修改的信息

- 联系人、联系电话、联系邮箱
- 法定代表人（企业客户）
- 客户级别、客户分类
- 案源人、负责律师
- 备注

### 不能修改的信息

- 客户编号（系统自动生成）
- 客户名称（如需修改，请联系管理员）
- 统一社会信用代码/身份证号（如需修改，请联系管理员）

## 场景四：潜在客户签约了，要转为正式客户

新录入的客户默认是「潜在」状态。当客户正式签约后，需要将状态改为「正式」。

### 转正步骤

1. 在客户列表中找到这个客户
2. 确认客户状态是「潜在」
3. 点击该行右侧的「转正式」按钮
4. 在确认弹窗中点击「确认」

转正后：
- 客户状态变为「正式」
- 可以为该客户创建合同和项目

---

## 场景五：客户长期没有业务，要设为休眠

如果某个客户长期没有新业务，可以将其设为「休眠」状态，方便管理。

### 设置休眠

1. 在客户列表中找到客户
2. 点击「编辑」
3. 将状态改为「休眠」
4. 保存

休眠客户：
- 仍然保留在系统中
- 不会出现在常用客户列表中
- 如果后续有新业务，可以重新激活

## 场景六：删除客户

::: warning 谨慎操作
删除客户是不可逆的操作。如果客户已经关联了合同或项目，将无法删除。
:::

### 删除步骤

1. 在客户列表中找到要删除的客户
2. 点击该行右侧的「删除」按钮
3. 在确认弹窗中点击「确定」

### 什么情况下可以删除？

- 客户是误录入的
- 客户没有关联任何合同或项目
- 客户没有任何业务记录

### 什么情况下不能删除？

- 客户已经关联了合同
- 客户已经关联了项目
- 客户有收款记录

如果需要删除但系统不允许，请联系系统管理员处理。

---

## 场景七：批量删除客户

如果有多个客户需要删除（比如测试数据），可以批量操作。

### 批量删除步骤

1. 在客户列表中，勾选要删除的客户（每行左侧有复选框）
2. 勾选完成后，点击「批量删除」按钮
3. 在确认弹窗中点击「确定」

## 常见问题

### Q: 客户编号是怎么生成的？

客户编号由系统自动生成，格式为 `C + 年份 + 序号`，如 `C202601001`。你不需要手动填写。

### Q: 案源人是什么意思？

案源人是指介绍这个客户来的人。在律所，案源人通常可以获得案源提成。所以录入客户时，一定要正确填写案源人。

### Q: 为什么我创建的客户别人看不到？

系统默认所有人都可以看到所有客户。如果看不到，可能是：
1. 筛选条件限制了显示范围
2. 权限设置问题，请联系管理员

### Q: 可以导入客户吗？

目前系统不支持批量导入客户。如果有大量客户需要录入，请联系系统管理员。

### Q: 客户名称写错了怎么办？

客户名称创建后不能直接修改。如果写错了：
1. 如果客户没有关联任何业务，可以删除后重新创建
2. 如果客户已有业务记录，请联系系统管理员修改

### Q: 利冲检索没有检测到冲突，是不是就一定没问题？

利冲检索只能检测系统中已录入的客户。如果对方当事人从未在系统中录入过，是检测不到的。所以：
- 利冲检索是辅助工具，不能完全依赖
- 律师仍需要根据自己的了解判断是否存在冲突
- 如有疑问，请咨询律所的利冲审查负责人
