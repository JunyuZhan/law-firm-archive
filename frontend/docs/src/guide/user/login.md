# 登录系统

## 打开系统

1. 打开浏览器（推荐使用 Chrome、Edge 或 Firefox）
2. 在地址栏输入系统地址（由管理员提供，通常类似 `http://your-company.com` 或 `http://192.168.1.100:5173`）
3. 按回车键，等待页面加载

::: tip 收藏网址
建议把系统地址添加到浏览器收藏夹，方便下次快速访问。
:::

---

## 第一次登录

### 获取账号

你的账号和初始密码由系统管理员分配。通常会通过以下方式告知你：
- 入职时由 IT 部门或行政部门提供
- 通过邮件发送
- 通过内部通讯工具发送

如果你不知道自己的账号，请联系：
- 系统管理员
- IT 部门
- 行政部门

### 登录步骤

1. 在登录页面，你会看到两个输入框：
   - **用户名**：输入你的账号（通常是工号或姓名拼音）
   - **密码**：输入你的密码

2. **完成滑块验证**：
   - 在密码框下方有一个滑块验证区域
   - 按住滑块，向右拖动到指定位置
   - 验证成功后会显示绿色的「验证成功」提示
   - **必须完成滑块验证才能登录**

3. 可选：勾选「记住密码」，下次登录时自动填充

4. 点击「登录」按钮

**登录成功后**，系统会自动跳转到工作台首页。

::: tip 滑块验证的作用
滑块验证是为了防止恶意程序自动登录，保护你的账号安全。每次登录都需要完成验证。
:::

**如果登录失败**，请检查：
- 用户名是否输入正确（注意大小写）
- 密码是否输入正确
- 滑块验证是否完成
- 网络是否正常

**如果滑块验证失败**：
- 刷新页面重试
- 确保网络连接正常
- 如果多次失败，稍等片刻再试

## 登录后第一件事：修改密码

::: warning 安全提醒
初始密码通常比较简单，为了账号安全，强烈建议你登录后立即修改密码。
:::

### 修改密码步骤

1. 登录成功后，看页面右上角
2. 点击你的**头像**或**用户名**
3. 在弹出的下拉菜单中，点击「个人设置」或「个人中心」
4. 找到「修改密码」或「安全设置」选项
5. 填写以下信息：
   - **原密码**：输入你当前的密码
   - **新密码**：输入你想设置的新密码
   - **确认新密码**：再次输入新密码
6. 点击「确认」或「保存」

### 密码要求

新密码需要满足以下要求：
- 长度至少 8 位
- 包含字母和数字
- 建议包含大小写字母和特殊字符（如 `@#$%`）

**好密码示例**：`MyLaw2024!`、`Lawyer@123`

**不好的密码**：`123456`、`password`、`abc123`

::: danger 密码安全
- 不要使用生日、手机号等容易被猜到的信息
- 不要把密码写在便签纸上贴在电脑旁
- 不要把密码告诉其他人
- 定期更换密码（建议每 3 个月更换一次）
:::

---

## 忘记密码了怎么办？

别担心，联系你的系统管理员，他可以帮你重置密码。

**重置流程**：
1. 联系管理员，告知你的用户名
2. 管理员会为你重置密码
3. 你会收到一个新的临时密码
4. 使用临时密码登录
5. 登录后立即修改为你自己的密码

---

## 账号被锁定了怎么办？

如果你连续输错密码多次（通常是 5 次），账号会被临时锁定，这是为了防止他人恶意尝试登录你的账号。

**解决方法**：

**方法一：等待自动解锁**
- 账号锁定后，等待 30 分钟会自动解锁
- 解锁后，使用正确的密码重新登录

**方法二：联系管理员**
- 如果着急使用，可以联系系统管理员
- 管理员可以立即为你解锁账号

## 退出登录

工作结束后，建议退出登录，特别是在公共电脑上使用时。

### 退出步骤

1. 点击页面右上角的头像
2. 在下拉菜单中选择「退出登录」
3. 系统会跳转回登录页面

::: tip 为什么要退出登录？
- 防止他人使用你的账号
- 保护你的工作数据
- 保护客户信息安全
:::

## 登录后看到的界面

登录成功后，你会看到系统的主界面，主要包括：

### 顶部导航栏
- **左侧**：系统 Logo 和名称
- **右侧**：消息通知、个人头像、退出登录

### 消息通知功能

系统会实时推送各类通知，帮助你及时了解工作动态：

**通知类型**：
- **审批通知**：有新的审批需要处理，或你的申请有了审批结果
- **任务通知**：有新任务分配给你，或任务状态变更
- **项目通知**：项目状态变更、新增团队成员等
- **系统通知**：系统公告、维护通知等

**通知提醒方式**：
1. **铃铛图标红点**：右上角铃铛图标显示红点，表示有未读通知
2. **声音提示**：收到新通知时会播放提示音
3. **浏览器通知**：如果你允许了浏览器通知权限，会弹出桌面通知

::: tip 开启浏览器通知
首次使用时，浏览器会询问是否允许通知。建议点击「允许」，这样即使你在其他页面工作，也能收到重要通知。
:::

**查看通知**：
1. 点击右上角的铃铛图标
2. 下拉显示最近的通知列表
3. 点击某条通知可以跳转到相关业务页面
4. 点击「全部已读」可以一键标记所有通知为已读

### 左侧菜单栏
根据你的角色，会显示不同的菜单。常见的菜单包括：
- **工作台**：首页、待办事项、审批
- **客户管理**：客户列表、利冲审查
- **项目管理**：项目列表、合同管理、工时管理
- **财务管理**：收款、提成、发票、报销
- **文档管理**：项目文档、文档模板
- **档案管理**：档案列表、借阅管理
- **行政管理**：请假、会议室、考勤
- **知识库**：案例库、法规库
- **系统管理**：用户、角色、部门（仅管理员可见）

### 主内容区
显示当前选中菜单的具体内容。

## 我是什么角色？能做什么？

系统会根据你的角色显示不同的菜单和功能。

### 律师（主办律师/协办律师）
- 查看和管理自己参与的项目
- 记录工作时间
- 上传项目文档
- 查看自己的收款和提成
- 提交费用报销
- 申请请假

### 财务人员
- 登记客户付款
- 管理发票
- 查看财务报表
- 管理提成发放

### 行政人员
- 管理考勤记录
- 审批请假申请
- 管理会议室
- 管理资产和采购

### 管理层（主任、合伙人）
- 审批各类申请（合同、收款、请假等）
- 查看全所业务数据
- 管理提成规则
- 查看统计报表

### 系统管理员
- 管理用户账号
- 配置角色权限
- 管理部门结构
- 系统配置

::: tip 看不到某些菜单？
如果你觉得应该有某个功能的权限但看不到对应菜单，请联系系统管理员检查你的角色配置。
:::

## 常见问题

### Q: 页面一直在加载，进不去系统？

**可能原因和解决方法**：
1. **网络问题**：检查网络连接是否正常
2. **浏览器缓存**：按 `Ctrl+F5`（Mac 是 `Cmd+Shift+R`）强制刷新
3. **浏览器兼容性**：换用 Chrome 浏览器试试
4. **系统维护**：联系管理员确认系统是否正常运行

### Q: 登录后页面显示空白？

**解决方法**：
1. 清除浏览器缓存
2. 关闭浏览器重新打开
3. 如果还是不行，联系管理员

### Q: 可以在手机上使用吗？

系统主要为电脑端设计，在手机上可能显示不完整。建议使用电脑访问。

### Q: 可以同时在多个设备登录吗？

可以，但建议不要这样做。如果在其他设备登录，当前设备可能会被强制退出。
