# 登录系统

## 打开系统

1. 打开浏览器（推荐使用 Chrome、Edge 或 Firefox）
2. 在地址栏输入系统地址（由管理员提供）
3. 按回车键，等待页面加载

::: tip 收藏网址
建议把系统地址添加到浏览器收藏夹，方便下次快速访问。
:::

---

## 登录步骤

1. 在登录页面输入：
   - **用户名**：你的账号（通常是工号或姓名拼音）
   - **密码**：你的密码

2. **完成滑块验证**：
   - 按住滑块，向右拖动到底
   - 验证成功后会显示绿色勾号
   - **必须完成滑块验证才能登录**

3. 可选：勾选「记住密码」，下次登录时自动填充

4. 点击「登录」按钮

**登录成功后**，系统会自动跳转到工作台首页。

::: warning 注意
- 每次登录都需要完成滑块验证
- 登录失败后滑块会重置，需要重新验证
:::

**如果登录失败**，请检查：
- 用户名是否输入正确（注意大小写）
- 密码是否输入正确
- 滑块验证是否完成

---

## 登录后修改密码

::: warning 安全提醒
初始密码通常比较简单，建议登录后立即修改密码。
:::

### 修改密码步骤

1. 登录成功后，点击页面右上角的**头像**
2. 在下拉菜单中点击「个人中心」
3. 在个人中心页面，点击左侧「修改密码」标签
4. 填写以下信息：
   - **原密码**：输入你当前的密码
   - **新密码**：输入你想设置的新密码
   - **确认新密码**：再次输入新密码
5. 点击「确认修改」

### 密码要求

新密码需要满足以下要求：
- 长度至少 8 位
- 包含字母和数字
- 建议包含大小写字母和特殊字符

::: danger 密码安全
- 不要使用生日、手机号等容易被猜到的信息
- 不要把密码告诉其他人
- 定期更换密码
:::

---

## 忘记密码

联系系统管理员，管理员可以帮你重置密码。

---

## 账号被锁定

如果连续输错密码多次，账号会被临时锁定。

**解决方法**：
- **等待自动解锁**：30 分钟后自动解锁
- **联系管理员**：管理员可以立即解锁

---

## 个人中心

点击右上角头像 → 「个人中心」，可以管理你的个人信息：

| 标签 | 功能 |
|------|------|
| 基本设置 | 修改头像、姓名、手机号、邮箱等 |
| 安全设置 | 查看账号安全状态 |
| 修改密码 | 修改登录密码 |
| 新消息提醒 | 设置消息通知偏好 |

---

## 退出登录

1. 点击页面右上角的头像
2. 在下拉菜单中选择「退出登录」

::: tip 安全建议
在公共电脑上使用完系统后，记得退出登录。
:::
