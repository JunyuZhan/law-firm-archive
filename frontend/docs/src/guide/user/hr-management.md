## 薪资管理

**菜单位置**：人力资源 → 薪资管理

管理员工工资表。

### 工资表流程

1. **创建工资表**：财务创建月度工资表
2. **编辑扣减项**：填写各员工的扣减项（社保、个税等）
3. **提交工资表**：提交审批
4. **员工确认**：员工查看并确认自己的工资
5. **财务确认**：财务确认所有员工已确认
6. **主任审批**：主任最终审批
7. **发放工资**：标记工资已发放

### 工资表状态

| 状态 | 说明 |
|------|------|
| DRAFT | 草稿 |
| SUBMITTED | 已提交 |
| CONFIRMED | 已确认 |
| APPROVED | 已审批 |
| ISSUED | 已发放 |

## 我的工资

**菜单位置**：人力资源 → 我的工资

查看和确认自己的工资明细。

### 工资明细

| 项目 | 说明 |
|------|------|
| 基本工资 | 固定薪资 |
| 岗位津贴 | 岗位补贴 |
| 提成收入 | 本月提成（如有） |
| 社保扣除 | 社保个人部分 |
| 公积金扣除 | 公积金个人部分 |
| 个税扣除 | 个人所得税 |
| 实发工资 | 最终到手金额 |

### 确认工资

1. 查看工资明细
2. 如有问题联系财务
3. 确认无误后点击「确认」

## 常见问题

**Q: 工资什么时候发放？**

工资发放时间由律所规定，通常在每月固定日期。

**Q: 工资有问题怎么办？**

在工资表确认前联系财务核实。

**Q: 转正申请什么时候提交？**

建议在试用期结束前一周提交。

**Q: 离职需要提前多久通知？**

按劳动合同规定，通常需要提前30天。
