# 人力资源管理

人力资源模块用于管理员工档案、薪资、绩效、培训、请假、考勤、会议室等人事相关事务。

## 谁能使用人力资源模块？

| 功能 | ADMIN | DIRECTOR | PARTNER | 财务 | 律师 | 行政 | 实习 |
|------|:-----:|:--------:|:-------:|:----:|:----:|:----:|:----:|
| 员工档案 | ✅ | ✅ | ✅ | ❌ | ❌ | ✅ | ❌ |
| 请假管理 | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ |
| 考勤管理 | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ |
| 薪资管理 | ✅ | ✅ | ❌ | ✅ | ❌ | ❌ | ❌ |
| 我的工资 | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ |
| 培训管理 | ✅ | ✅ | ✅ | ❌ | ✅ | ✅ | ✅ |
| 绩效考核 | ✅ | ✅ | ❌ | ❌ | ❌ | ❌ | ❌ |
| 晋升管理 | ✅ | ✅ | ✅ | ❌ | ❌ | ❌ | ❌ |
| 发展计划 | ✅ | ✅ | ✅ | ❌ | ❌ | ❌ | ❌ |
| 转正管理 | ✅ | ✅ | ✅ | ❌ | ❌ | ❌ | ❌ |
| 离职管理 | ✅ | ✅ | ✅ | ❌ | ❌ | ❌ | ❌ |
| 会议室管理 | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ |

::: tip 说明
- 请假管理、考勤管理、我的工资、会议室管理对所有角色开放
- 薪资管理由财务人员和管理层使用
- 培训管理对大部分角色开放，支持报名和学分管理
- 绩效考核由超级管理员和律所主任管理
- 晋升、发展、转正、离职由合伙人及以上角色管理
:::

## 员工档案

### 进入员工档案

点击左侧菜单「人力资源」→「员工档案」

页面分为两个标签页：
- **用户列表**：系统用户列表，可批量创建员工档案
- **员工档案**：已创建的员工档案列表

---

### 场景一：从用户创建员工档案

如果系统中已有用户，但还没有员工档案：

1. 切换到「用户列表」标签页
2. 勾选需要创建档案的用户（已有档案的用户会显示为禁用状态）
3. 点击「批量创建员工档案」
4. 确认后，系统自动为选中的用户创建基础档案
5. 创建完成后，可以在「员工档案」中完善详细信息

---

### 场景二：手动创建员工档案

1. 切换到「员工档案」标签页
2. 点击「新建员工档案」
3. 填写员工信息：

**基本信息**
| 字段 | 说明 |
|------|------|
| 关联用户 | 选择系统用户（必填） |
| 员工编号 | 工号 |
| 性别 | 男/女 |
| 出生日期 | 生日 |
| 身份证号 | 身份证号码 |
| 民族 | 如「汉族」 |
| 籍贯 | 户籍所在地 |
| 政治面貌 | 党员/团员/群众 |

**教育信息**
| 字段 | 说明 |
|------|------|
| 学历 | 博士/硕士/本科/专科/高中 |
| 专业 | 所学专业 |
| 毕业院校 | 学校名称 |
| 毕业日期 | 毕业时间 |

**紧急联系人**
| 字段 | 说明 |
|------|------|
| 紧急联系人 | 联系人姓名 |
| 紧急联系电话 | 联系电话 |
| 联系地址 | 家庭住址 |

**律师执业信息**（律师专用）
| 字段 | 说明 |
|------|------|
| 律师执业证号 | 执业证编号 |
| 执业证状态 | 有效/过期/暂停 |
| 执业证发证日期 | 发证时间 |
| 执业证到期日期 | 到期时间 |
| 执业领域 | 擅长领域 |
| 执业年限 | 从业年数 |

**工作信息**
| 字段 | 说明 |
|------|------|
| 职位 | 岗位名称 |
| 级别 | 职级 |
| 入职日期 | 入职时间 |
| 试用期结束日期 | 转正日期 |
| 工作状态 | 在职/试用期/离职/停薪留职 |

4. 点击「确定」保存

---

### 场景三：查看员工档案详情

1. 在员工档案列表中点击「查看」
2. 弹窗显示员工的完整信息
3. 包括基本信息、教育背景、执业信息、工作信息等

---

### 场景四：编辑员工档案

1. 点击「编辑」按钮
2. 修改需要更新的信息
3. 点击「确定」保存

---

## 考勤管理

考勤管理用于员工签到签退、查看考勤记录。

### 进入考勤管理

点击左侧菜单「人力资源」→「考勤管理」

页面标题为「考勤管理」，描述为「员工考勤记录、签到签退」。

---

### 今日考勤卡片

页面顶部显示「今日考勤」卡片，展示当天的考勤状态：

- **签到时间**：显示签到时间，未签到显示「未签到」
- **签退时间**：显示签退时间，未签退显示「未签退」
- **状态**：正常/迟到/早退/缺勤/请假
- **签到按钮**：点击进行签到（已签到后禁用）
- **签退按钮**：点击进行签退（未签到时禁用，已签退后禁用）

---

### 场景一：签到

1. 进入「考勤管理」页面
2. 在「今日考勤」卡片中点击「签到」按钮
3. 系统记录签到时间
4. 如果迟到，状态会显示为「迟到」

::: tip 签到规则
- 每天只能签到一次
- 签到时间会记录到秒
- 系统会根据设定的上班时间判断是否迟到
:::

---

### 场景二：签退

1. 在「今日考勤」卡片中点击「签退」按钮
2. 系统记录签退时间
3. 如果早退，状态会显示为「早退」

::: warning 签退限制
- 必须先签到才能签退
- 每天只能签退一次
:::

---

### 场景三：查看考勤记录

1. 在搜索区域设置筛选条件：

| 筛选条件 | 说明 |
|---------|------|
| 日期范围 | 选择开始和结束日期 |
| 状态 | 全部/正常/迟到/早退/缺勤/请假 |

2. 点击「查询」按钮搜索，点击「重置」清空筛选条件
3. 表格显示考勤记录

---

### 考勤记录列表字段说明

| 列名 | 说明 |
|------|------|
| 姓名 | 员工姓名 |
| 日期 | 考勤日期 |
| 签到时间 | 签到的时间 |
| 签退时间 | 签退的时间 |
| 状态 | 正常（绿色）/迟到（橙色）/早退（橙色）/缺勤（红色）/请假（蓝色） |
| 工作时长(小时) | 当天工作的小时数 |
| 备注 | 备注信息 |

---

### 考勤状态说明

| 状态 | 代码 | 含义 | 颜色 |
|------|------|------|------|
| 正常 | `NORMAL` | 按时签到签退 | 绿色 |
| 迟到 | `LATE` | 签到时间晚于规定上班时间 | 橙色 |
| 早退 | `EARLY_LEAVE` | 签退时间早于规定下班时间 | 橙色 |
| 缺勤 | `ABSENT` | 未签到或未签退 | 红色 |
| 请假 | `LEAVE` | 当天有请假记录 | 蓝色 |

## 请假管理

请假管理用于员工申请请假、查看假期余额和请假记录。

### 进入请假管理

点击左侧菜单「人力资源」→「请假管理」

页面标题为「请假管理」，描述为「请假申请、审批与假期余额查询」。

---

### 假期余额卡片

页面顶部显示「假期余额」卡片，展示各类假期的剩余天数：

- 显示每种假期类型的名称
- 显示剩余天数（蓝色大字）
- 显示「剩余 / 总天数」

---

### 场景一：查看请假记录

1. 进入「请假管理」页面
2. 使用筛选条件：

| 筛选条件 | 说明 |
|---------|------|
| 请假类型 | 从下拉框选择请假类型 |
| 状态 | 全部/待审批/已通过/已拒绝/已取消 |

3. 点击「查询」按钮搜索，点击「重置」清空筛选条件
4. 表格显示请假记录

---

### 请假记录列表字段说明

| 列名 | 说明 |
|------|------|
| 请假类型 | 年假/事假/病假/调休等 |
| 开始日期 | 请假开始的日期 |
| 结束日期 | 请假结束的日期 |
| 请假天数 | 请假的天数 |
| 请假原因 | 请假的原因说明 |
| 状态 | 待审批（橙色）/已通过（绿色）/已拒绝（红色）/已取消（灰色） |
| 审批人 | 审批请假的人 |

---

### 场景二：申请请假

1. 点击「申请请假」按钮
2. 填写请假信息：

| 字段 | 说明 | 必填 |
|------|------|:----:|
| 请假类型 | 从下拉框选择请假类型 | ✅ |
| 请假日期 | 选择开始和结束日期（日期范围选择器） | ✅ |
| 请假天数 | 系统自动计算，也可手动调整（支持0.5天） | ✅ |
| 请假原因 | 填写请假原因 | ✅ |

3. 点击「确定」提交申请
4. 申请状态变为「待审批」

::: tip 请假天数计算
选择日期范围后，系统会自动计算请假天数。如需请半天假，可以手动调整天数为0.5。
:::

---

### 场景三：取消请假申请

如果不需要请假了：

1. 找到状态为「待审批」的请假记录
2. 点击「取消」链接
3. 申请状态变为「已取消」

::: warning 取消限制
只有「待审批」状态的申请可以取消。已通过或已拒绝的申请不能取消。
:::

---

### 场景四：审批请假（管理层）

有审批权限的用户可以审批请假申请：

1. 找到状态为「待审批」的请假记录
2. 点击「通过」审批通过
3. 点击「拒绝」（红色）审批拒绝

审批后，申请人会收到通知。

---

### 请假状态说明

| 状态 | 含义 | 颜色 |
|------|------|------|
| 待审批 | 等待审批人处理 | 橙色 |
| 已通过 | 审批通过 | 绿色 |
| 已拒绝 | 审批拒绝 | 红色 |
| 已取消 | 申请人主动取消 | 灰色 |

## 财务人员操作指南

### 场景五：查看工资表

1. 进入「薪资管理」页面
2. 选择年份和月份
3. 系统自动显示该月所有员工的工资明细
4. 页面顶部显示汇总信息：
   - 工资表编号
   - 状态（草稿/待确认/已确认/已审批/已发放）
   - 总人数
   - 应发总额
   - 扣减总额
   - 实发总额
   - 已确认人数

5. 表格显示每个员工的工资明细：

| 列名 | 说明 |
|------|------|
| 工号 | 员工工号 |
| 姓名 | 员工姓名 |
| 收入 | 提成总额（自动从提成管理导入） |
| 应发工资 | 收入 - 税费扣减项 |
| 扣减总额 | 所有扣减项的总和 |
| 实发工资 | 应发工资 - 其他扣减项 |
| 确认状态 | 待确认/已确认/已拒绝 |

::: tip 员工显示规则
- 员工属性确认后，自动在当月工资列表中显示
- 员工离职后，次月开始不再显示
:::

---

### 场景六：编辑员工扣减项

收入（提成）由系统自动从提成管理模块导入，财务人员主要编辑扣减项。

1. 在工资列表中找到员工，点击「编辑扣减项」
2. 弹窗显示该员工的工资明细

**收入项明细**（只读，自动导入）：
- 显示每笔提成的来源合同和金额
- 收入类型：提成、绩效奖金等

**扣减项明细**（可编辑）：
- 点击「添加扣减项」
- 选择或输入扣减类型：
  - **税费类**：个人所得税、社保个人部分、公积金个人部分
  - **其他类**：办公室租金、其他欠费、其他扣款
- 输入金额
- 填写说明（必填）

3. 设置确认截止时间（默认每月27日24时）
4. 点击「确定」保存

**工资计算公式：**
```
收入（提成总额）
    ↓
减去：税费扣减项（个税、社保、公积金）
    ↓
= 应发工资（税后）
    ↓
减去：其他扣减项（办公室租金、欠款等）
    ↓
= 实发工资
```

::: warning 工资可以为负
当扣减项大于收入时，工资会显示为负数（红色），表示员工需要补缴费用。
:::

---

### 场景七：提交工资表审批

所有员工的扣减项编辑完成后：

1. 点击「提交工资表」按钮
2. 工资表状态变为「待确认」
3. 系统通知所有员工确认工资
4. 等待员工确认

---

### 场景八：财务确认

所有员工确认完成后（或超过确认截止时间自动确认）：

1. 点击「财务确认」按钮
2. 工资表状态变为「待审批」
3. 提交给主任或合伙人审批

---

### 场景九：发放工资

审批通过后：

1. 点击「发放工资」按钮
2. 选择发放方式：
   - 银行转账
   - 现金
   - 其他
3. 可上传付款凭证
4. 填写备注
5. 点击「确定」完成发放
6. 工资表状态变为「已发放」

---

### 场景十：导出和打印工资表

只有已审批通过的工资表可以导出和打印：

1. 点击「导出Excel」下载工资表
2. 点击「打印」打开打印预览
3. 打印内容包括：
   - 工资表标题和年月
   - 汇总信息
   - 每个员工的工资明细

---

## 员工操作指南

### 场景十一：查看我的工资

1. 点击「人力资源」→「我的工资」
2. 选择年份和月份筛选
3. 表格显示你的工资记录：

| 列名 | 说明 |
|------|------|
| 工资表编号 | 工资表的唯一编号 |
| 年月 | 工资所属月份 |
| 状态 | 工资表状态 |
| 应发工资 | 税后应发金额 |
| 扣减总额 | 所有扣减项 |
| 实发工资 | 最终到手金额 |
| 确认状态 | 待确认/已确认/已拒绝 |

4. 点击「查看」查看详细明细

---

### 场景十二：确认我的工资

财务提交工资表后，你会收到确认通知：

1. 进入「我的工资」页面
2. 找到状态为「待确认」的工资记录
3. 点击「查看」查看详细明细：
   - 收入项明细（每笔提成的来源和金额）
   - 扣减项明细（税费、其他扣款）
   - 应发工资和实发工资

4. 核对无误后，点击「确认无误」
5. 如有问题，点击「有误，拒绝」并填写原因

::: warning 确认截止时间
请在截止时间前完成确认。超过截止时间（默认每月27日24时），系统将自动确认。
:::

---

### 工资表状态说明

| 状态 | 含义 | 下一步 |
|------|------|--------|
| 草稿 | 财务正在编辑 | 财务提交 |
| 待确认 | 等待员工确认 | 员工确认 |
| 已确认 | 员工已确认 | 财务确认 |
| 待审批 | 等待主任/合伙人审批 | 审批 |
| 已审批 | 审批通过 | 发放 |
| 已发放 | 工资已发放 | 完成 |

### 确认状态说明

| 状态 | 含义 |
|------|------|
| 待确认 | 等待员工确认 |

1. 点击「新建计划」按钮
2. 填写计划信息：

|------|------|:----:|

| 会议室名称 | 如「大会议室」「小会议室A」 | ✅ |
| 位置 | 如「3楼东侧」 | ❌ |
| 容纳人数 | 最大容纳人数（默认10人） | ❌ |
| 设备 | 如「投影仪、白板、视频会议设备」 | ❌ |
| 描述 | 会议室说明 | ❌ |

3. 点击「确定」保存

1. 在会议室列表中找到要编辑的会议室
2. 点击「编辑」链接
3. 修改会议室信息
4. 点击「确定」保存

### 场景三十八：删除会议室

2. 确认后删除会议室

::: warning 删除限制
如果会议室有未完成的预约，建议先取消预约再删除。
:::

### 场景三十九：预约会议室

1. 在会议室列表中找到状态为「可用」的会议室
2. 点击「预约」链接
3. 填写预约信息：

| 字段 | 说明 | 必填 |
|------|------|:----:|
| 会议主题 | 会议的主题 | ✅ |
| 开始时间 | 会议开始时间 | ✅ |
| 结束时间 | 会议结束时间 | ✅ |
| 参会人员 | 参加会议的人员，多人用逗号分隔 | ❌ |
| 会议描述 | 会议说明 | ❌ |

4. 点击「确定」提交预约

::: tip 预约限制
- 只能预约状态为「可用」的会议室
- 预约时间不能与其他预约冲突
:::

### 场景四十一：取消预约

如果不需要使用会议室了：

1. 在预约记录中找到状态为「已预约」的记录

::: warning 取消限制
只有「已预约」状态的预约可以取消。进行中或已完成的预约不能取消。
:::

## 常见问题

### 员工档案问题

**Q: 员工档案和用户账号是什么关系？**
A: 用户账号用于登录系统，员工档案记录人事信息。一个用户可以关联一个员工档案。

**Q: 试用期员工可以看到工资吗？**
A: 可以，员工可以查看自己的工资信息。

### 考勤问题

**Q: 每天可以签到几次？**
A: 每天只能签到一次，签退一次。

**Q: 忘记签到怎么办？**
A: 如果忘记签到，当天会显示为缺勤状态。请联系人事部门补录考勤。

**Q: 签到时间是怎么判断迟到的？**
A: 系统根据设定的上班时间判断，超过上班时间签到会标记为迟到。

**Q: 请假当天需要签到吗？**
A: 不需要，请假当天的考勤状态会自动显示为「请假」。

### 薪资问题

**Q: 工资的收入是怎么来的？**
A: 收入（提成）由系统自动从提成管理模块导入。当客户付款后，系统计算提成，这些提成会自动出现在工资表的收入项中。

**Q: 为什么我的工资是负数？**
A: 当扣减项（税费、办公室租金、欠款等）大于收入（提成）时，工资会显示为负数，表示你需要补缴费用。

**Q: 工资确认截止时间是什么意思？**
A: 财务提交工资表后，员工需要在截止时间前确认。超过截止时间（默认每月27日24时），系统会自动确认。

**Q: 我拒绝了工资确认怎么办？**
A: 财务会收到通知，核实后会联系你沟通。如果确实有误，财务会修改后重新提交确认。

### 绩效问题

**Q: 绩效考核结果会影响工资吗？**
A: 绩效结果可以作为绩效工资的计算依据，具体由公司薪酬制度决定。

**Q: 绩效考核有哪些评价类型？**
A: 包括自评（员工对自己的评价）、主管评（主管对下属的评价）、同事评（同事之间的互评）。

### 培训问题

**Q: 培训学分有什么用？**
A: 培训学分是员工参加培训的记录，可以作为年度考核、晋升评估的参考依据。

**Q: 报名培训后可以取消吗？**
A: 可以，在培训开始前可以取消报名。培训进行中或已完成后无法取消。

**Q: 培训学分是自动获得的吗？**
A: 是的，培训完成后系统会自动将学分累加到你的总学分中。

### 发展计划问题

**Q: 发展计划和晋升申请有什么区别？**
A: 发展计划是长期的职业规划，设定目标和里程碑，帮助你逐步成长。晋升申请是当你认为已经达到晋升条件时，正式申请职级提升。

**Q: 发展计划的导师有什么作用？**
A: 导师可以指导你的职业发展，帮助你完成计划中的目标和里程碑。

**Q: 里程碑完成后会自动更新进度吗？**
A: 是的，每完成一个里程碑，系统会自动计算完成进度百分比。

### 晋升问题

**Q: 晋升申请需要满足什么条件？**
A: 具体条件由公司制度决定，通常包括工作年限、业绩表现、能力评估等。建议在申请前与主管沟通。

**Q: 晋升通过后什么时候生效？**
A: 审批时会设置生效日期，默认是下月1日。生效后你的职级会自动更新。

**Q: 晋升申请被拒绝后可以再次申请吗？**
A: 可以，建议根据拒绝原因改进后再次申请。

### 转正问题

**Q: 试用期多久可以申请转正？**
A: 通常在试用期结束前一周提交申请。具体试用期长度以劳动合同为准。

**Q: 转正申请被拒绝会怎样？**
A: 可能会延长试用期或终止劳动关系，具体由公司决定。

### 离职问题

**Q: 离职后账号会怎样？**
A: 离职后账号会被禁用，无法登录系统。离职次月起，你也不会出现在工资表中。

**Q: 离职交接需要多长时间？**
A: 根据工作内容而定，通常需要1-2周完成交接。

**Q: 离职后还能查看之前的工资记录吗？**
A: 离职后账号被禁用，无法登录系统。如需查询，请联系人事部门。

### 会议室问题

**Q: 会议室预约有时间限制吗？**
A: 预约时间不能与其他预约冲突，同一时间段只能有一个预约。

**Q: 预约的会议室可以提前取消吗？**
A: 可以，在会议开始前可以取消预约。

**Q: 会议室显示「使用中」是什么意思？**
A: 表示会议室当前有会议正在进行，暂时无法预约该时间段。

**Q: 会议室显示「维护中」可以预约吗？**
A: 不可以，维护中的会议室暂时不可用，需要等待维护完成。
