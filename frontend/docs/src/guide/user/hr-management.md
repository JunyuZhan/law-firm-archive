## 员工管理

**菜单位置**：人力资源 → 员工管理

管理员工档案信息，包括从系统用户批量创建员工档案、维护员工详细信息等。

### 双标签页设计

页面分为两个标签页：

1. **用户列表**：查看系统用户，可批量创建员工档案
2. **员工档案**：管理已创建的员工档案详细信息

### 用户列表（批量导入）

用于从现有系统用户批量创建员工档案。

#### 查询筛选

| 筛选项 | 说明           |
| ------ | -------------- |
| 用户名 | 模糊搜索用户名 |
| 姓名   | 模糊搜索姓名   |
| 部门   | 按部门筛选     |
| 状态   | 正常/禁用      |

#### 用户列表字段

| 列名     | 说明                      |
| -------- | ------------------------- |
| 用户名   | 登录账号                  |
| 姓名     | 用户真实姓名              |
| 部门     | 所属部门                  |
| 职位     | 职位名称                  |
| 手机号   | 联系电话                  |
| 邮箱     | 电子邮箱                  |
| 工号     | 员工编号                  |
| 入职日期 | 入职时间                  |
| 状态     | 正常（绿色）/禁用（灰色） |

#### 批量创建员工档案

1. 在用户列表页面，使用复选框选择要创建档案的用户
2. 点击「批量创建员工档案」按钮
3. 确认后系统会为选中的用户创建基础员工档案
4. 创建成功后，可以在「员工档案」标签页查看和管理

::: warning 已创建档案的用户已创建员工档案的用户会被标记，不会出现在可选列表中，避免重复创建。:::

### 员工档案

管理详细的员工档案信息，包括基本信息、教育背景、紧急联系人、律师执业信息等。

#### 查询筛选

| 筛选项   | 说明                      |
| -------- | ------------------------- |
| 员工编号 | 模糊搜索员工编号          |
| 姓名     | 模糊搜索姓名              |
| 部门     | 按部门筛选                |
| 工作状态 | 在职/试用期/离职/停薪留职 |
| 职位     | 模糊搜索职位              |

#### 员工列表字段

| 列名     | 说明                                                      |
| -------- | --------------------------------------------------------- |
| 员工编号 | 员工唯一编号                                              |
| 姓名     | 员工姓名                                                  |
| 部门     | 所属部门                                                  |
| 职位     | 职位名称                                                  |
| 手机号   | 联系电话                                                  |
| 邮箱     | 电子邮箱                                                  |
| 入职日期 | 入职时间                                                  |
| 工作状态 | 在职（绿色）/试用期（橙色）/离职（红色）/停薪留职（灰色） |
| 操作     | 查看/编辑/删除                                            |

#### 新建员工档案

支持两种方式创建员工档案：

1. **从用户批量导入**：在「用户列表」标签页批量创建
2. **手动新建**：在「员工档案」标签页点击「新建员工档案」

手动新建时，需要关联系统用户，并填写以下信息：

**基本信息**：

- 关联用户（必填，创建后不可修改）
- 员工编号
- 性别、出生日期、身份证号
- 民族、籍贯、政治面貌

**教育背景**：

- 学历、专业
- 毕业院校、毕业日期

**紧急联系人**：

- 紧急联系人姓名
- 紧急联系电话
- 联系地址

**律师执业信息**（仅限律师）：

- 律师执业证号、执业证状态
- 执业证发证日期、到期日期
- 执业领域、执业年限

**工作信息**：

- 职位、级别
- 入职日期、试用期结束日期
- 工作状态（在职/试用期/离职/停薪留职）

#### OCR身份证识别

系统支持身份证智能识别功能，但目前暂时禁用（优化中）。功能恢复后，可上传身份证正面照片自动识别：

- 姓名、身份证号
- 性别、出生日期
- 民族等信息

#### 查看员工详情

点击「查看」按钮可查看员工完整的档案信息，包括所有基本信息和历史记录。

#### 编辑员工档案

点击「编辑」按钮可修改员工档案信息。注意：关联用户创建后不可修改。

#### 删除员工档案

点击「删除」按钮可删除员工档案。删除后，该用户的员工档案信息将被清除，但系统用户账号不受影响。

## 薪资管理

**菜单位置**：人力资源 → 薪资管理

管理员工工资表。

### 工资表流程

1. **创建工资表**：财务创建月度工资表
2. **编辑扣减项**：填写各员工的扣减项（社保、个税等）
3. **提交工资表**：提交审批
4. **员工确认**：员工查看并确认自己的工资
5. **财务确认**：财务确认所有员工已确认
6. **主任审批**：主任最终审批
7. **发放工资**：标记工资已发放

### 工资表状态

| 状态      | 说明   |
| --------- | ------ |
| DRAFT     | 草稿   |
| SUBMITTED | 已提交 |
| CONFIRMED | 已确认 |
| APPROVED  | 已审批 |
| ISSUED    | 已发放 |

## 我的工资

**菜单位置**：人力资源 → 我的工资

查看和确认自己的工资明细。

### 工资明细

| 项目       | 说明             |
| ---------- | ---------------- |
| 基本工资   | 固定薪资         |
| 岗位津贴   | 岗位补贴         |
| 提成收入   | 本月提成（如有） |
| 社保扣除   | 社保个人部分     |
| 公积金扣除 | 公积金个人部分   |
| 个税扣除   | 个人所得税       |
| 实发工资   | 最终到手金额     |

### 确认工资

1. 查看工资明细
2. 如有问题联系财务
3. 确认无误后点击「确认」

## 常见问题

**Q: 工资什么时候发放？**

工资发放时间由律所规定，通常在每月固定日期。

**Q: 工资有问题怎么办？**

在工资表确认前联系财务核实。

**Q: 转正申请什么时候提交？**

建议在试用期结束前一周提交。

**Q: 离职需要提前多久通知？**

按劳动合同规定，通常需要提前30天。
