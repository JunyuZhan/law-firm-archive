# 档案管理

档案管理模块用于管理律所的业务档案，包括档案归档、借阅、迁移等。

| 档案列表 |   ✅   |  ✅  |     ✅     |  ✅  |  ✅  |  ✅  |  ❌  |

## 档案列表

**菜单位置**：档案管理 → 档案列表

查看所有已归档的业务档案。

### 档案列表字段

| 列名     | 说明           |
| -------- | -------------- |
| 档案编号 | 系统自动生成   |
| 档案名称 | 档案标题       |
| 关联项目 | 所属项目       |
| 归档日期 | 归档时间       |
| 存放位置 | 物理存放位置   |
| 状态     | 在库/借出/销毁 |

### 查询档案

1. 输入档案名称或编号搜索
2. 按项目筛选
3. 按状态筛选

### 查看档案详情

点击档案名称查看详细信息，包括：

- 档案基本信息
- 包含的文档清单
- 借阅记录

## 档案借阅

**菜单位置**：档案管理 → 档案借阅

申请借阅档案。

### 借阅申请

1. 找到要借阅的档案
2. 点击「申请借阅」
3. 填写借阅信息：

| 字段         | 必填 | 说明         |
| ------------ | :--: | ------------ |
| 借阅目的     |  ✅  | 说明借阅原因 |
| 预计归还日期 |  ✅  | 计划归还时间 |
| 备注         |  -   | 其他说明     |

4. 提交申请

### 借阅审批

借阅申请由档案管理员或主任审批。

### 归还档案

借阅到期后，需要归还档案：

1. 将档案归还给档案管理员
2. 系统记录归还信息

### 借阅状态

| 状态     | 说明   |
| -------- | ------ |
| PENDING  | 待审批 |
| APPROVED | 已批准 |
| BORROWED | 借阅中 |
| RETURNED | 已归还 |
| REJECTED | 已拒绝 |
| OVERDUE  | 已逾期 |

## 档案迁移

**菜单位置**：档案管理 → 档案迁移

管理档案的迁移和销毁。

### 迁移申请

当档案需要转移存放位置或销毁时：

1. 点击「申请迁移」
2. 选择迁移类型（转移/销毁）
3. 填写迁移原因
4. 提交审批

### 迁移审批

档案迁移需要主任或管理员审批。

## 常见问题

**Q: 如何将项目归档？**

项目结案后，在项目详情页申请归档，审批通过后自动生成档案。

**Q: 借阅档案有期限吗？**

借阅有期限，具体期限由管理员设定，到期需要归还。

**Q: 档案销毁后可以恢复吗？**

不可以，销毁前请确认档案确实不再需要。

**Q: 可以借阅电子档案吗？**

电子档案可以直接在线查看，无需申请借阅。
