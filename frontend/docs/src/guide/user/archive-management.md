# 档案管理

案件结案后，需要将相关材料整理归档。档案管理帮助你管理实体档案的入库、借阅和销毁。

## 谁能使用档案管理？

| 功能 | ADMIN | DIRECTOR | PARTNER | 财务 | 主办律师 | 协办律师 | 行政 | 实习 |
|------|:-----:|:--------:|:-------:|:----:|:-------:|:-------:|:----:|:----:|
| 档案列表 | ✅ | ✅ | ❌ | ✅ | ✅ | ✅ | ✅ | ❌ |
| 档案借阅 | ✅ | ✅ | ❌ | ✅ | ✅ | ✅ | ✅ | ❌ |
| 档案销毁 | ✅ | ✅ | ❌ | ❌ | ✅ | ❌ | ❌ | ❌ |

::: warning 实习律师限制
实习律师无法访问档案管理模块。如需查阅档案，请联系主办律师协助。
:::

---

## 进入档案管理

点击左侧菜单「档案管理」，可以看到三个子菜单：
- 档案列表：管理所有档案
- 档案借阅：申请和管理借阅记录
- 档案销毁：申请和管理销毁流程

---

## 文档与档案的区别

在开始之前，先了解一下「文档」和「档案」的区别：

| 类型 | 说明 | 存储位置 | 查看方式 |
|------|------|---------|---------|
| 文档 | 电子文件 | 存储在系统服务器 | 随时在线查看、下载 |
| 档案 | 实体材料（纸质文件） | 存放在档案室 | 需要借阅才能查看 |

档案管理主要管理的是**实体档案**，即纸质文件的存放、借阅和销毁。

## 档案列表

### 场景一：案子结了，我要把材料归档

当项目状态变为「已结案」后，就可以创建档案了。

**创建档案步骤：**

1. 进入「档案管理」→「档案列表」
2. 点击「新建档案」按钮
3. 填写档案信息：

| 字段 | 说明 | 示例 |
|------|------|------|
| 档案名称 | 档案的名称 | 「张三诉李四合同纠纷案卷宗」 |
| 所属项目 | 选择对应的项目 | 从下拉框选择 |
| 档案类型 | 案件档案/合同档案/行政档案 | 根据内容选择 |
| 描述 | 备注信息 | 可选 |

4. 点击「确定」创建档案

::: tip 档案编号
系统会自动生成档案编号，格式类似 `DA2026001`，方便后续查找。
:::

---

### 场景二：档案整理好了，要入库

创建档案后，需要把实体材料放到档案室，并在系统中登记存放位置。

**入库步骤：**

1. 在档案列表中找到档案
2. 点击「入库」按钮
3. 填写存放位置，例如：「档案室A区-3排-5层」
4. 点击「确定」

入库后档案状态变为「在库」，系统记录存放位置方便后续查找。

---

### 场景三：我想查找某个档案

**方法一：搜索**
在搜索框输入档案名称，点击「查询」。

**方法二：筛选**
使用筛选条件（档案类型、状态）缩小范围后点击「查询」。

## 档案借阅

进入「档案管理」→「档案借阅」

页面顶部有四个标签页：
- **借阅中**：当前正在借阅的档案
- **已归还**：已经归还的借阅记录
- **逾期未还**：超过预计归还日期还没还的
- **全部**：所有借阅记录

### 场景四：我要借阅一份档案

**申请借阅步骤：**

1. 进入「档案借阅」页面
2. 点击「申请借阅」按钮
3. 填写借阅信息：

| 字段 | 说明 |
|------|------|
| 选择档案 | 从下拉框选择要借阅的档案（支持搜索） |
| 预计归还日期 | 计划什么时候还 |
| 借阅事由 | 为什么要借阅 |

4. 点击「确定」提交申请

::: info 借阅流程
1. **待审批**：等待档案管理员审批
2. **审批通过**：可以去档案室取档案了
3. **借阅中**：档案管理员确认借出后的状态
:::

---

### 场景五：我是档案管理员，要审批借阅申请

**审批步骤：**

1. 在「全部」标签页中找到状态为「待审批」的记录
2. 点击「通过」或「拒绝」
   - **通过**：同意借阅申请
   - **拒绝**：需要输入拒绝原因

**确认借出：**
审批通过后，当借阅人来取档案时，点击「确认借出」，档案状态变为「借阅中」。

---

### 场景六：档案用完了，要归还

**归还步骤：**

1. 在「借阅中」标签页找到要归还的记录
2. 点击「归还」按钮
3. 填写归还日期和档案状况（可选）
4. 点击「确定」完成归还

::: warning 逾期提醒
超过预计归还日期还没归还，记录会出现在「逾期未还」标签页。请及时归还档案！
:::

## 档案销毁

进入「档案管理」→「档案销毁」

::: danger 重要提醒
档案销毁后不可恢复，请谨慎操作！
:::

页面顶部有四个标签页：
- **待审批**：等待审批的销毁申请
- **待销毁**：审批通过，等待执行销毁
- **已销毁**：已经销毁的档案
- **全部**：所有档案

### 场景七：档案过了保管期限，要申请销毁

**申请销毁步骤：**

1. 进入「档案销毁」页面
2. 在列表中选中要销毁的档案
3. 点击「申请销毁」按钮
4. 填写销毁原因（如「已超过保管期限」）
5. 点击「确定」提交申请

---

### 场景八：我是档案管理员，要审批销毁申请

**审批步骤：**

1. 在「待审批」标签页找到销毁申请
2. 点击「审批」按钮
3. 选择审批结果（通过/拒绝）并填写审批意见
4. 点击「确认」

---

### 场景九：销毁申请通过了，要执行销毁

**执行销毁步骤：**

1. 在「待销毁」标签页找到档案
2. 点击「执行销毁」按钮
3. 依次输入销毁方式、销毁地点、见证人
4. 确认后档案状态变为「已销毁」

::: tip 销毁记录
系统会完整记录销毁信息（时间、方式、地点、见证人），以备查证。
:::

## 常见问题

**Q: 借阅申请被拒绝了怎么办？**
A: 查看拒绝原因，根据原因调整后重新申请。

**Q: 借阅档案有时间限制吗？**
A: 有的。申请时需要填写预计归还日期，超过日期未归还会显示为「逾期」。

**Q: 我能直接销毁档案吗？**
A: 不能。销毁档案需要提交申请，经过审批后才能执行销毁。

**Q: 销毁需要见证人吗？**
A: 是的。执行销毁时需要填写见证人信息，确保销毁过程有人监督。
