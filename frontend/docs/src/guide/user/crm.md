# 客户关系管理

客户关系管理（CRM）模块帮助您管理案源线索、进行利益冲突检查，是业务拓展的重要工具。

## 谁能使用 CRM？

| 功能 | ADMIN | DIRECTOR | PARTNER | 财务 | 主办律师 | 协办律师 | 行政 | 实习 |
|------|:-----:|:--------:|:-------:|:----:|:-------:|:-------:|:----:|:----:|
| 案源管理 | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ❌ |
| 利冲审查 | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ |

::: warning 实习律师限制
实习律师可以进行利冲审查，但不能访问案源管理。
:::

---

## 案源管理

### 进入案源管理

点击左侧菜单「客户管理」→「案源管理」

页面标题为「案源管理」，描述为「管理案源线索和跟进」。

案源是指潜在的业务机会，还没有正式签约成为客户的线索。

---

### 场景一：录入新案源

当有新的业务线索时：

1. 点击「新增案源」按钮
2. 填写案源信息：

| 字段 | 说明 | 必填 |
|------|------|:----:|
| 客户名称 | 潜在客户的名称 | ✅ |
| 联系人 | 对接人姓名 | ❌ |
| 联系电话 | 联系方式 | ❌ |
| 来源 | 转介绍/网络/电话咨询/上门咨询/其他 | ❌ |
| 预估金额 | 预估的业务金额 | ❌ |
| 跟进人 | 负责跟进的律师 | ❌ |
| 备注 | 其他备注信息 | ❌ |

3. 点击「确定」保存

---

### 场景二：跟进案源

1. 在案源列表中找到要跟进的案源
2. 点击「编辑」更新跟进情况
3. 可以更新：
   - 联系记录
   - 客户需求
   - 跟进状态

---

### 场景三：将案源转化为客户

当案源确定要签约时：

1. 找到要转化的案源（状态不能是「已转化」）
2. 点击「转化」按钮
3. 在确认弹窗中点击「确认」
4. 系统会：
   - 自动创建客户档案
   - 案源状态变为「已转化」
   - 可以继续创建项目和合同

::: tip 转化时机
建议在客户确定委托意向后再进行转化，避免创建无效的客户档案。
:::

---

### 场景四：筛选和搜索案源

1. 使用搜索框按客户名称搜索
2. 使用下拉框按来源筛选：
   - 全部
   - 转介绍
   - 网络
   - 电话咨询
   - 上门咨询
   - 其他
3. 使用状态筛选：
   - 全部
   - 新建：刚录入的案源
   - 跟进中：正在跟进
   - 已转化：已成为正式客户
   - 已流失：未能成交

4. 点击「查询」按钮搜索，点击「重置」清空筛选条件

---

### 场景五：删除案源

1. 找到要删除的案源
2. 点击「删除」链接
3. 在确认弹窗中点击「确定」

---

## 利冲检查

### 什么是利冲检查？

利益冲突检查（简称「利冲检查」）是律师事务所在接受新客户或新业务前，必须进行的合规检查。目的是确保：
- 不会同时代理对立双方
- 不会损害现有客户的利益
- 符合律师职业道德规范

---

### 进入利冲检查

点击左侧菜单「客户管理」→「利冲检查」

---

### 场景五：申请利冲检查

在接受新客户或新项目前：

1. 点击「申请利冲审查」按钮
2. 填写检查信息：

| 字段 | 说明 |
|------|------|
| 审查类型 | 新客户/新项目（必填） |
| 客户名称 | 委托人名称（必填） |
| 对方当事人 | 案件对方的名称（必填） |
| 项目名称 | 项目名称（可选） |
| 备注 | 其他需要说明的信息 |

3. 点击「确定」提交申请

---

### 场景六：查看利冲检查结果

提交后，系统会自动检查是否存在利益冲突：

1. 在列表中找到你的申请
2. 查看状态：
   - **待检查**：等待系统检查
   - **已通过**：无利益冲突，可以接受委托
   - **存在冲突**：发现利益冲突

3. 点击「查看」可以看到详细的检查结果

---

### 场景七：处理存在冲突的情况

如果检查结果显示「存在冲突」：

**选项一：放弃委托**
- 告知客户无法接受委托
- 案源标记为流失

**选项二：申请豁免**
1. 点击「申请豁免」按钮
2. 填写豁免理由（必填），说明为什么可以豁免
3. 提交后等待审批
4. 审批通过后，可以继续接受委托

::: warning 豁免条件
利冲豁免需要满足特定条件，如双方书面同意、不涉及核心利益等。请谨慎申请。
:::

---

### 场景八：审批利冲豁免（管理层）

1. 在列表中找到状态为「豁免待审批」的申请
2. 点击「批准」或「拒绝」
3. 填写审批意见
4. 确认后完成审批

---

| 状态 | 含义 | 颜色 |
|------|------|------|

## 利冲检查结果说明

| 结果 | 含义 |
|------|------|
| 无冲突 | 系统未发现利益冲突 |
| 存在冲突 | 发现与现有客户或项目存在利益冲突 |
| 已豁免 | 虽有冲突但已获得豁免批准 |

## 常见问题

**Q: 什么时候需要做利冲检查？**
A: 接受新客户委托前、接受新项目前都需要进行利冲检查。

**Q: 利冲检查需要多长时间？**
A: 系统自动检查通常几秒钟完成。如果需要人工审核，可能需要1-2个工作日。

**Q: 案源一定要转化才能签合同吗？**
A: 不一定。如果客户是直接来签约的，可以直接创建客户，不需要先录入案源。案源主要用于管理潜在客户。

**Q: 利冲检查通过后还会变吗？**
A: 利冲检查结果是基于检查时的数据。如果后续有新的客户或项目录入，可能需要重新检查。

**Q: 豁免申请被拒绝怎么办？**
A: 如果豁免被拒绝，说明利益冲突无法解决，需要放弃该委托。
