# 系统管理

系统管理模块用于管理系统的基础配置，包括用户、角色、部门、菜单、系统配置和操作日志等。

## 谁能使用系统管理？

| 功能 | ADMIN | DIRECTOR | PARTNER | 财务 | 律师 | 行政 | 实习 |
|------|:-----:|:--------:|:-------:|:----:|:----:|:----:|:----:|
| 用户管理 | ✅ | ❌ | ❌ | ❌ | ❌ | ❌ | ❌ |
| 角色管理 | ✅ | ❌ | ❌ | ❌ | ❌ | ❌ | ❌ |
| 部门管理 | ✅ | ❌ | ❌ | ❌ | ❌ | ❌ | ❌ |
| 菜单管理 | ✅ | ❌ | ❌ | ❌ | ❌ | ❌ | ❌ |
| 系统配置 | ✅ | ❌ | ❌ | ❌ | ❌ | ❌ | ❌ |
| 操作日志 | ✅ | ❌ | ❌ | ❌ | ❌ | ❌ | ❌ |

::: warning 管理员专属
系统管理功能仅限超级管理员使用，其他角色无法访问。
:::

## 用户管理

### 进入用户管理

点击左侧菜单「系统管理」→「用户管理」

---

### 场景一：创建新用户

1. 点击「新增用户」按钮
2. 填写用户信息：

| 字段 | 说明 | 必填 |
|------|------|:----:|
| 用户名 | 登录账号，创建后不可修改 | ✅ |
| 姓名 | 用户真实姓名 | ✅ |
| 密码 | 初始密码 | ✅ |
| 部门 | 所属部门 | ✅ |
| 职位 | 职位名称 | ❌ |
| 手机号 | 联系电话 | ❌ |
| 邮箱 | 电子邮箱 | ❌ |
| 角色 | 分配角色（可多选） | ✅ |

3. 点击「确定」保存

::: tip 初始密码
建议使用统一的初始密码（如 `123456`），并提醒用户首次登录后修改密码。
:::

---

### 场景二：编辑用户信息

1. 在用户列表中找到要编辑的用户
2. 点击「编辑」
3. 修改用户信息（用户名不可修改）
4. 点击「确定」保存

---

### 场景三：重置用户密码

当用户忘记密码时：

1. 找到该用户
2. 点击「重置密码」
3. 确认后密码会重置为默认密码（`123456`）
4. 通知用户新密码，并提醒修改

---

### 场景四：启用/禁用用户

1. 找到要操作的用户
2. 点击「禁用」或「启用」
3. 确认后用户状态变更

禁用后的用户：
- 无法登录系统
- 不会出现在人员选择列表中
- 历史数据保留

---

### 场景五：删除用户

1. 找到要删除的用户
2. 点击「删除」
3. 确认删除

::: warning 删除限制
如果用户有关联的业务数据（如项目、客户等），建议使用「禁用」而不是删除。
:::

---

## 角色管理

### 进入角色管理

点击左侧菜单「系统管理」→「角色管理」

---

### 场景六：创建新角色

1. 点击「新增角色」按钮
2. 填写角色信息：

| 字段 | 说明 | 必填 |
|------|------|:----:|
| 角色编码 | 角色唯一标识，创建后不可修改 | ✅ |
| 角色名称 | 角色显示名称 | ✅ |
| 数据范围 | 全部数据/本部门及下级/本部门/仅本人 | ✅ |
| 排序 | 角色排序顺序 | ❌ |
| 描述 | 角色说明 | ❌ |

3. 点击「确定」保存

---

### 场景七：分配角色权限

1. 找到要配置的角色
2. 点击「分配权限」
3. 在权限树中勾选该角色可以访问的菜单和功能
4. 点击「确定」保存

::: tip 权限继承
勾选父级菜单会自动勾选所有子菜单。取消父级会取消所有子菜单。
:::

---

### 数据范围说明

| 数据范围 | 说明 |
|---------|------|
| 全部数据 | 可以查看系统中所有数据 |
| 本部门及下级 | 可以查看本部门及下级部门的数据 |
| 本部门 | 只能查看本部门的数据 |
| 仅本人 | 只能查看自己创建或负责的数据 |

---

---

### 场景十二：创建菜单

1. 点击「新增菜单」按钮
2. 填写菜单信息：

1. 在日志列表中找到要查看的记录

   - 操作时间
   - 操作类型
   - 操作模块
   - 请求参数
   - 响应结果
   - IP 地址
   - 浏览器信息

## 外部系统集成

### 进入外部系统集成

点击左侧菜单「系统管理」→「外部集成」

页面标题为「外部系统集成」，描述为「管理档案馆API、AI大模型等外部系统的连接配置」。

页面分为三个标签页：
- **AI大模型**：配置 AI 智能辅助功能
- **档案系统**：配置档案迁移对接
- **其他系统**：其他外部系统集成

---

### 场景十六：配置 AI 大模型

AI 大模型用于系统中的智能辅助功能，如文书生成、合同审查等。

1. 切换到「AI大模型」标签页
2. 找到要配置的 AI 服务（如 OpenAI、Claude、DeepSeek 等）
3. 点击「配置」
4. 填写配置信息：

| 字段 | 说明 | 必填 |
|------|------|:----:|
| API地址 | AI 服务的 API 地址 | ✅ |
| 认证方式 | API Key / Bearer Token / Basic Auth / OAuth 2.0 | ✅ |
| API Key | API 密钥（留空表示不修改） | ❌ |
| API Secret | API 密钥（部分服务需要） | ❌ |
| 描述 | 配置说明 | ❌ |

5. 点击「确定」保存

---

### 场景十七：测试 AI 连接

1. 找到已配置的 AI 服务
2. 点击「测试」
3. 系统会尝试连接 AI 服务
4. 测试结果显示：
   - **成功**（绿色）：连接正常
   - **失败**（红色）：连接失败，查看详情了解原因

---

### 场景十八：启用/禁用 AI 服务

- 点击「启用」：启用该 AI 服务，可在文书制作等功能中使用
- 点击「禁用」：禁用该 AI 服务

::: tip 多模型支持
可以同时启用多个 AI 模型，在使用时可以选择具体使用哪个模型。
:::

---

### 场景十九：配置档案系统

档案系统配置用于档案迁移功能，将律所档案同步到外部档案管理系统。

1. 切换到「档案系统」标签页
2. 找到要配置的档案系统（如市档案馆、区档案馆）
3. 点击「配置」填写 API 地址和认证信息
4. 点击「测试」验证连接
5. 点击「启用」启用该档案系统

---

## 提成规则配置

### 进入提成规则配置

点击左侧菜单「系统管理」→「提成规则配置」

页面标题为「提成规则配置」，描述为「设置案件收入分配预设方案，律师创建合同时可选择并自定义」。

---

### 场景二十：查看提成规则

页面显示所有提成规则方案，包括：

| 列名 | 说明 |
|------|------|
| 方案名称 | 规则名称 |
| 律所 | 律所提成比例 |
| 主办律师 | 主办律师提成比例 |
| 协办律师 | 协办律师提成比例 |
| 辅助人员 | 辅助人员提成比例 |
| 案源人 | 案源人提成比例 |
| 可修改 | 律师创建合同时是否可以自定义比例 |
| 默认 | 是否为默认方案 |
| 状态 | 启用/停用 |

::: tip 比例说明
- 比例可以为 0%，表示该角色不参与分配
- 比例之和不强制等于 100%，根据实际情况灵活设置
:::

---

### 场景二十一：新增提成规则

1. 点击「新增方案」按钮
2. 填写规则信息：

| 字段 | 说明 |
|------|------|
| 方案名称 | 如「标准分配方案」 |
| 律所比例 | 律所提成比例（0-100%） |
| 主办律师比例 | 主办律师提成比例 |
| 协办律师比例 | 协办律师提成比例 |
| 辅助人员比例 | 辅助人员提成比例 |
| 案源人比例 | 案源人提成比例 |
| 允许修改 | 律师创建合同时是否可以自定义 |
| 说明 | 方案说明 |

3. 点击「确定」保存

---

### 场景二十二：管理提成规则

- **编辑**：点击「编辑」修改规则
- **设为默认**：点击「设为默认」，新建合同时默认使用此规则
- **启用/停用**：点击「启用」或「停用」切换状态
- **删除**：点击「删除」删除规则（默认方案不能删除）

---

### 常见提成场景

| 场景 | 律所 | 主办 | 协办 | 辅助 | 案源 |
|------|:----:|:----:|:----:|:----:|:----:|
| 独立办案 | 30% | 70% | 0% | 0% | 0% |
| 团队协作 | 20% | 40% | 30% | 10% | 0% |
| 律所不提成 | 0% | 70% | 30% | 0% | 0% |

## 权限矩阵管理

### 进入权限矩阵管理

点击左侧菜单「系统管理」→「权限矩阵」

页面标题为「权限矩阵管理」，描述为「查看和管理所有角色的权限配置」。

---

### 场景二十三：查看权限矩阵

权限矩阵以表格形式展示所有角色的权限配置：
- 行：各个权限点（菜单/按钮）
- 列：各个角色
- 单元格：✓ 表示有权限，✗ 表示无权限

**筛选功能：**
- 模块筛选：按功能模块筛选（客户管理/项目管理/财务管理等）
- 权限类型：菜单/按钮

---

### 场景二十四：编辑权限（快速模式）

1. 点击「编辑权限」按钮进入编辑模式
2. 点击单元格可快速切换权限
3. 系统会弹出确认对话框
4. 确认后权限立即生效
5. 点击「退出编辑」结束编辑模式

---

### 场景二十五：对比角色权限

1. 点击「对比权限」按钮
2. 勾选要对比的角色（至少2个）
3. 点击「开始对比」
4. 系统显示选中角色的权限差异

---

### 场景二十六：导出权限矩阵

点击「导出Excel」可以将权限矩阵导出为 Excel 文件，方便离线查看和存档。

## 常见问题

**Q: 用户忘记密码怎么办？**
A: 管理员可以在用户管理中重置用户密码为默认密码。

**Q: 如何批量创建用户？**
A: 目前系统不支持批量导入用户，需要逐个创建。

**Q: 角色权限修改后什么时候生效？**
A: 权限修改后，用户需要重新登录才能生效。

**Q: 可以删除系统内置角色吗？**
A: 系统内置角色（如 ADMIN、DIRECTOR 等）不能删除，只能修改权限。

**Q: 操作日志会保存多久？**
A: 操作日志会长期保存，具体保存期限由系统配置决定。

**Q: 如何查看某个用户的所有操作？**
A: 在操作日志中，使用「操作人」筛选条件，选择该用户即可。

**Q: AI 大模型配置后多久生效？**
A: 配置保存后立即生效，可以点击「测试」验证连接是否正常。

**Q: 可以同时启用多个 AI 模型吗？**
A: 可以启用多个，在文书制作时可以选择使用哪个模型。

**Q: 提成规则可以修改吗？**
A: 可以。编辑提成规则后，新创建的合同会使用新规则，已有合同不受影响。

**Q: 权限矩阵中如何快速修改权限？**
A: 点击「编辑权限」进入编辑模式，然后点击单元格即可快速切换权限。
