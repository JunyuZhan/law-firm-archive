## 收款管理

**菜单位置**：财务管理 → 收款管理

登记和管理客户付款。

### 收款列表字段

| 列名 | 说明 |
|------|------|
| 收款编号 | 系统自动生成 |
| 项目名称 | 关联的项目 |
| 客户名称 | 付款客户 |
| 合同编号 | 关联的合同 |
| 收款金额 | 本次收款金额 |
| 收款日期 | 收到款项的日期 |
| 付款方式 | 银行转账/现金/支票等 |
| 状态 | 待确认/已确认 |

### 登记收款

1. 点击「新增收款」
2. 选择关联的项目和合同
3. 填写收款信息：

| 字段 | 必填 | 说明 |
|------|:----:|------|
| 项目 | ✅ | 选择关联项目 |
| 合同 | - | 选择关联合同 |
| 收款金额 | ✅ | 本次收款金额 |
| 收款日期 | ✅ | 收到款项日期 |
| 付款方式 | ✅ | 转账/现金/支票 |
| 备注 | - | 其他说明 |

4. 点击「确定」保存

### 收款确认

新登记的收款需要确认后才会计入统计。

## 提成管理

**菜单位置**：财务管理 → 提成管理

管理律师提成的计算和发放。

### 提成列表字段

| 列名 | 说明 |
|------|------|
| 项目名称 | 关联项目 |
| 律师 | 提成对象 |
| 角色 | 主办/协办/案源人 |
| 提成比例 | 提成百分比 |
| 应提金额 | 计算出的提成金额 |
| 已提金额 | 已发放金额 |
| 状态 | 待审批/已审批/已发放 |

### 提成计算

系统根据以下规则自动计算提成：
1. 收款到账后触发提成计算
2. 根据提成规则计算各角色的提成金额
3. 生成提成记录待审批

### 提成审批

1. 主任/管理员审批提成记录
2. 审批通过后可以发放

### 提成发放

1. 点击「发放」按钮
2. 确认发放金额
3. 记录发放日期

## 费用报销

**菜单位置**：财务管理 → 费用报销

提交和管理费用报销申请。

### 报销列表字段

| 列名 | 说明 |
|------|------|
| 报销编号 | 系统自动生成 |
| 报销人 | 申请人 |
| 报销类型 | 交通/餐饮/住宿/办公等 |
| 报销金额 | 申请金额 |
| 关联项目 | 可选关联项目 |
| 状态 | 待审批/已审批/已拒绝/已报销 |

### 提交报销

1. 点击「新增报销」
2. 选择报销类型
3. 填写报销金额和说明
4. 上传发票/收据附件
5. 可选关联项目
6. 提交审批

### 报销审批

报销申请由财务或主任审批。

### 报销状态

| 状态 | 说明 |
|------|------|
| PENDING | 待审批 |
| APPROVED | 已审批 |
| REJECTED | 已拒绝 |
| REIMBURSED | 已报销 |

## 财务报表

**菜单位置**：财务管理 → 财务报表

查看财务统计数据。

### 报表内容

- **收入统计**：按月/年统计收入
- **项目收支**：各项目的收支情况
- **律师业绩**：律师收入排名
- **应收账款**：待收款明细

---

## 合同变更 / 收款变更

当合同或收款信息需要修改时，可以提交变更申请，变更需要审批后生效。

---

## 常见问题

**Q: 提成是什么时候计算的？**

收款确认后系统自动计算提成。

**Q: 提成比例是怎么确定的？**

提成比例由系统管理员在「提成规则」中配置。

**Q: 报销需要上传附件吗？**

建议上传发票或收据作为凭证。

**Q: 可以查看历史财务数据吗？**

可以在财务报表中按时间范围查询历史数据。
