## 收款管理

**菜单位置**：财务管理 → 收款管理

管理项目收费和收款记录。系统以收费项目（Fee）为基准，针对每个收费项目登记收款。

### 查询筛选

| 筛选项   | 说明                                                |
| -------- | --------------------------------------------------- |
| 创建年份 | 按创建年份筛选，支持选择"全部年份"或最近5年        |
| 收费编号 | 模糊搜索收费编号（支持输入后按回车或点击查询按钮） |
| 客户     | 选择已创建的客户（支持搜索）                        |
| 项目     | 选择关联的项目（支持搜索）                          |
| 状态     | 待收款/部分收款/已收款（支持清空选择）              |

### 收费列表字段

| 列名       | 说明                           |
| ---------- | ------------------------------ |
| 收费编号   | 系统自动生成                   |
| 收费名称   | 收费项目名称                   |
| 项目名称   | 关联的项目                     |
| 客户名称   | 付款客户                       |
| 收费金额   | 应收金额                       |
| 已收金额   | 已收款金额                     |
| 未收金额   | 未收款金额（自动计算）         |
| 收费类型   | 服务费/预付款/其他             |
| 状态       | 待收款/部分收款/已收款         |
| 创建时间   | 收费项目创建时间               |

### 登记收款

收款是针对收费项目进行登记的：

1. 在收费列表中找到需要收款的收费项目（状态为"待收款"或"部分收款"）
2. 点击「登记收款」按钮
3. 系统自动填充收费项目信息，填写收款信息：

| 字段       | 必填 | 说明                                 |
| ---------- | :--: | ------------------------------------ |
| 收费项目   |  ✅  | 系统自动填充（不可修改）             |
| 收款金额   |  ✅  | 本次收款金额（不能超过未收金额）     |
| 货币       |  ✅  | 人民币/美元等（默认人民币）          |
| 收款方式   |  ✅  | 银行转账/现金/支票/其他              |
| 收款日期   |  ✅  | 收到款项的日期                       |
| 银行账户   |  -   | 收款账户信息                         |
| 交易流水号 |  -   | 银行转账的交易流水号                 |
| 备注       |  -   | 其他说明                             |

4. 点击「确定」保存，系统会自动确认收款并更新收费项目状态

### OCR智能识别（暂未开放）

::: warning OCR功能状态当前版本中OCR功能暂时禁用，本地识别服务优化中，后续版本将重新开放。:::

收款管理页面提供OCR智能识别功能（暂未开放），用于识别银行回单自动填充收款信息：

- **银行回单识别**：上传银行回单图片，自动识别收款金额、收款日期、交易流水号、付款方、收款账户等信息
- **自动匹配收费项目**：系统会自动匹配相似的收费项目，根据金额、客户等信息推荐匹配项
- **自动填充表单**：选择匹配的收费项目后，系统自动填充收款表单，只需核对后确认即可

::: tip OCR识别提示当OCR功能重新开放后，可在登记收款时使用OCR功能快速录入收款信息，提高工作效率。:::

### 查看收款明细

对于已收款或部分收款的收费项目，可以查看收款明细：

1. 点击「查看明细」按钮
2. 查看该收费项目的所有收款记录：
   - 收款编号
   - 收款金额
   - 收款日期
   - 收款方式
   - 交易流水号
   - 备注

### 收费状态说明

| 状态     | 含义                         | 颜色 |
| -------- | ---------------------------- | ---- |
| 待收款   | 尚未收到任何款项             | 灰色 |
| 部分收款 | 已收到部分款项               | 橙色 |
| 已收款   | 已收到全部款项               | 绿色 |

### 统计卡片

页面顶部显示统计卡片（当前为占位展示，后续可接入后台统计数据）：

- 本月收款：显示本月收款总额
- 待收款：显示待收款总额
- 已开票：显示已开票金额
- 收款笔数：显示本月收款笔数

## 提成管理

**菜单位置**：财务管理 → 提成管理

管理律师提成的计算和发放。

### 提成列表字段

| 列名     | 说明                 |
| -------- | -------------------- |
| 项目名称 | 关联项目             |
| 律师     | 提成对象             |
| 角色     | 主办/协办/案源人     |
| 提成比例 | 提成百分比           |
| 应提金额 | 计算出的提成金额     |
| 已提金额 | 已发放金额           |
| 状态     | 待审批/已审批/已发放 |

### 提成计算

系统根据以下规则自动计算提成：

1. 收款到账后触发提成计算
2. 根据提成规则计算各角色的提成金额
3. 生成提成记录待审批

### 提成审批

1. 主任/管理员审批提成记录
2. 审批通过后可以发放

### 提成发放

1. 点击「发放」按钮
2. 确认发放金额
3. 记录发放日期

## 费用报销

**菜单位置**：财务管理 → 费用报销

提交和管理费用报销申请。

### 报销列表字段

| 列名     | 说明                        |
| -------- | --------------------------- |
| 报销编号 | 系统自动生成                |
| 报销人   | 申请人                      |
| 报销类型 | 交通/餐饮/住宿/办公等       |
| 报销金额 | 申请金额                    |
| 关联项目 | 可选关联项目                |
| 状态     | 待审批/已审批/已拒绝/已报销 |

### 提交报销

1. 点击「新增报销」
2. 选择报销类型
3. 填写报销金额和说明
4. 上传发票/收据附件
5. 可选关联项目
6. 提交审批

### 报销审批

报销申请由财务或主任审批。

### 报销状态

| 状态       | 说明   |
| ---------- | ------ |
| PENDING    | 待审批 |
| APPROVED   | 已审批 |
| REJECTED   | 已拒绝 |
| REIMBURSED | 已报销 |

## 常见问题

**Q: 提成是什么时候计算的？**

收款确认后系统自动计算提成。

**Q: 提成比例是怎么确定的？**

提成比例由系统管理员在「提成规则」中配置。

**Q: 报销需要上传附件吗？**

建议上传发票或收据作为凭证。

**Q: 可以查看历史财务数据吗？**

可以在财务报表中按时间范围查询历史数据。
