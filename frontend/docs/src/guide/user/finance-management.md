# 财务管理

财务模块是律所日常运营的核心，涵盖收款、提成、发票、报销等功能。根据你的角色，系统会显示不同的菜单和功能。

## 我是什么角色？

系统根据你的角色显示不同的财务菜单：

| 角色 | 能看到的菜单 | 主要职责 |
|------|-------------|---------|
| 管理员 | 全部菜单 | 系统管理，可查看所有财务数据 |
| 主任 | 全部菜单 + 提成规则管理 | 业务最高管理，审批重要事项 |
| 团队负责人 | 我的收款、我的提成、费用报销 + 财务视角菜单 | 部门业务管理 |
| 财务 | 全部财务视角菜单 | 日常财务操作 |
| 律师 | 我的收款、我的提成、费用报销 | 查看自己案子的收款和提成 |
| 行政 | 费用报销 | 提交报销 |
| 实习 | 费用报销 | 提交报销 |

**财务模块分为两个视角：**
- **律师视角**：我的收款、我的提成、费用报销
- **财务视角**：合同管理、收款管理、提成管理、发票管理、财务报表、合同变更处理、收款变更审批

::: tip 找到你的角色
下面的内容按视角分类：
- **律师视角**：适用于律师、实习律师、行政人员
- **财务视角**：适用于财务人员
- **管理层视角**：适用于主任、团队负责人
:::

## 第一部分：律师操作指南

作为律师，你主要关心三件事：案子收了多少钱、我能拿多少提成、报销怎么弄。

### 场景一：查看我的案子收款情况

**我想知道我参与的合同，客户付了多少钱，还欠多少。**

1. 点击左侧菜单「财务管理」→「我的收款」
2. 页面顶部显示四个统计卡片：
   - **参与合同数**：你参与的所有合同数量
   - **合同总金额**：所有参与合同的总金额
   - **已收金额**：已经收到的款项（绿色显示）
   - **待收金额**：还没收到的款项（橙色显示）

3. 下方表格显示每个合同的详情：

| 列名 | 说明 |
|------|------|
| 合同编号 | 合同的唯一编号 |
| 合同名称 | 合同的名称 |
| 客户 | 合同对应的客户 |
| 我的角色 | 你在合同中的角色（主办律师/协办律师/案源人/律师助理） |
| 合同金额 | 这个合同的总金额 |
| 已收金额 | 已经收到多少（绿色显示） |
| 收款进度 | 进度条显示收款百分比 |
| 最近收款 | 最近一笔收款的日期 |

::: tip 数据说明
「我的收款」显示你参与的所有合同（包括作为主办律师、协办律师、案源人等角色）。表格中会显示你在每个合同中的具体角色。
:::

### 场景二：查看我的提成

**客户付款后，我想看看我能拿多少提成。**

1. 点击「财务管理」→「我的提成」
2. 页面顶部显示四个统计卡片：
   - **提成记录数**：你的所有提成记录数量
   - **提成总额**：所有提成的总金额
   - **已发放**：已经发放的金额（绿色显示）
   - **待发放**：还没发放的金额（橙色显示）

3. 可以按状态筛选：
   - **全部**：所有提成记录
   - **待发放**：已计算但还没发到手的（PENDING）
   - **已发放**：已经发放的（PAID）

4. 每条提成记录显示：

| 列名 | 说明 |
|------|------|
| 合同编号 | 产生提成的合同 |
| 合同名称 | 合同名称 |
| 客户 | 客户名称 |
| 收款金额 | 这笔收款的金额 |
| 提成比例 | 你的提成比例（百分比显示） |
| 提成金额 | 你能拿到的提成（蓝色高亮显示） |
| 状态 | 待发放（橙色）/已发放（绿色） |
| 计算时间 | 提成计算的时间 |
| 发放时间 | 如果已发放，显示发放日期 |

**提成是怎么算的？**

简单来说，客户付款后，系统按照提成规则自动计算：

```
客户付款 100,000 元
    ↓
律所留存 30,000 元（假设留存比例 30%）
    ↓
可分配金额 70,000 元
    ↓
├── 案源提成 7,000 元（给介绍案子的人，假设 10%）
└── 办案提成 63,000 元（给主办和协办律师，假设 90%）
        ├── 主办律师 50,400 元（假设占 80%）
        └── 协办律师 12,600 元（假设占 20%）
```

::: tip 提成比例
具体比例由律所管理层设定，不同类型的业务可能使用不同的规则。如有疑问请咨询财务。
:::

::: info 提成与工资的关系
提成审批通过后，会自动出现在「人力资源」→「薪资管理」的工资表中作为收入项。每月财务会汇总提成，生成工资表，你可以在「我的工资」中查看详细明细。
:::

::: warning 授薪律师
如果你是授薪律师（拿固定月薪），你不参与提成分配，「我的提成」菜单可能不显示或显示为空。
:::

### 场景三：提交费用报销

**我出差花了钱，或者为案子垫付了费用，需要报销。**

1. 点击「财务管理」→「费用报销」
2. 页面顶部显示四个统计卡片：
   - **报销记录数**：你的所有报销记录数量
   - **报销总额**：所有报销的总金额
   - **已审批**：已审批通过的金额（绿色显示）
   - **待审批**：等待审批的金额（橙色显示）

3. 点击「新增报销」按钮
4. **使用发票智能识别（推荐）**：

::: tip 发票OCR识别
系统支持通过OCR技术自动识别发票信息，大幅提升报销效率。
:::

在新增报销弹窗顶部，你会看到「发票智能识别」区域：

| 功能 | 说明 |
|------|------|
| 拍照识别发票 | 上传发票/票据照片，自动识别金额、日期等信息 |

**使用方法：**
1. 点击「拍照识别发票」按钮
2. 选择发票照片（支持 JPG、PNG 等格式）
3. 等待系统识别
4. 识别成功后，自动填充以下字段：
   - **金额**：发票金额
   - **费用日期**：发票日期
   - **费用类型**：根据发票内容自动推断（餐饮费、住宿费、交通费等）
   - **费用说明**：发票类型、销售方、发票号等信息

5. 核对并补充其他信息后提交

**手动填写报销信息：**

**手动填写报销信息：**

| 字段 | 必填 | 说明 |
|------|------|------|
| 费用类型 | 是 | 差旅费/交通费/餐饮费/住宿费/办公费/诉讼费/其他 |
| 金额 | 是 | 实际花费金额 |
| 费用日期 | 是 | 费用发生的日期 |
| 费用说明 | 是 | 简要描述费用用途，如「出差北京见客户」 |

5. 检查无误后点击「确定」提交

**报销审批流程：**

```
提交报销 → 部门负责人审批 → 财务审核 → 打款 → 完成
```

**查看报销进度：**

在「费用报销」页面可以看到你提交的所有报销申请：

| 状态 | 含义 |
|------|------|
| 待审批 (PENDING) | 等待审批 |
| 已审批 (APPROVED) | 审批通过，等待打款 |
| 已拒绝 (REJECTED) | 审批被拒绝，可查看拒绝原因 |
| 已取消 (CANCELLED) | 已撤销的报销申请 |
| 已打款 (PAID) | 已打款到你的账户 |

**撤销报销：**

如果报销申请还在「待审批」状态，你可以点击「撤销」取消这笔报销申请。

::: tip 加快报销
- 费用说明写清楚，避免来回沟通
- 选择正确的费用类型
:::

## 第二部分：财务人员操作指南

作为财务人员，你负责收款登记、提成计算与发放、发票管理、报销审核，以及财务报表。

### 场景四：登记客户收款

**客户付款了，我需要在系统里登记。**

1. 点击「财务管理」→「收款管理」
2. 页面顶部显示统计卡片（本月收款、待收款、已开票、收款笔数）
3. 使用年份筛选（默认显示当前年份，可选择「全部年份」）

**方式一：使用银行回单智能识别（推荐）**

::: tip OCR智能识别
系统支持通过OCR技术自动识别银行回单，并智能匹配待收款项目，大幅提升收款登记效率。
:::

1. 在搜索区域点击「OCR识别」按钮
2. 在弹窗中上传银行回单图片（支持拖拽上传）
3. 系统自动识别回单信息：
   - 银行名称
   - 交易金额
   - 交易日期
   - 交易流水号
   - 付款方/收款方信息
4. 识别成功后，系统自动进行智能匹配：
   - 根据金额、付款方名称等信息匹配待收款项目
   - 显示匹配度百分比和匹配原因
   - 高匹配度项目会标记为「建议自动核销」
5. 选择匹配的收费项目，点击「选择并填充」
6. 系统自动填充收款表单，核对后确认即可

**智能匹配结果说明：**

| 匹配度 | 颜色 | 说明 |
|--------|------|------|
| ≥85% | 绿色 | 高匹配度，建议直接核销 |
| 50%-84% | 橙色 | 中等匹配度，请仔细核对 |
| <50% | 红色 | 低匹配度，可能不是同一笔款项 |

**方式二：手动登记收款**

4. 找到对应的收费记录：
   - 按收费编号搜索
   - 按客户筛选
   - 按状态筛选（待收款/部分收款/已收款）
5. 找到记录后，点击操作列的「登记收款」
6. 填写收款信息：

| 字段 | 必填 | 说明 |
|------|------|------|
| 收费项目 | 自动 | 显示选中的收费项目名称（只读） |
| 收款金额 | 是 | 实际收到的金额（不能超过待收金额） |
| 货币 | 是 | 人民币(CNY)/美元(USD)/欧元(EUR) |
| 收款方式 | 是 | 银行转账/现金/支票/其他 |
| 收款日期 | 是 | 款项到账日期 |
| 银行账户 | 否 | 收款的银行账户 |
| 交易流水号 | 建议 | 银行流水号，方便对账 |
| 备注 | 否 | 其他需要记录的信息 |

7. 点击「确定」保存，系统会自动确认收款

::: tip 自动触发提成计算
收款登记成功后，该收款会出现在「待计算提成」列表中，等待财务计算提成。
:::

### 场景五：查看收款明细

**我想看某个项目的所有收款记录。**

1. 在「收款管理」页面找到项目（状态为「已收款」或「部分收款」）
2. 点击操作列的「查看明细」
3. 弹窗显示该项目的基本信息和所有收款记录：
   - 收费编号、收费名称
   - 应收金额、已收金额、待收金额
   - 状态
   - 每笔收款的详细信息：
     - 收款编号
     - 收款金额
     - 收款方式
     - 收款日期
     - 交易流水号
     - 状态
     - 备注

### 场景六：计算提成

**收款登记后，我需要计算律师的提成。**

1. 点击「财务管理」→「提成管理」
2. 页面有两个主标签页：
   - **提成记录**：查看已计算的提成
   - **待计算提成**：查看需要计算提成的收款

3. 切换到「待计算提成」标签页
4. 找到需要处理的收款记录，点击「计算提成」
5. 系统显示：
   - 收费编号、项目名称、客户名称
   - 收款金额
   - 参与人列表（从合同参与人自动载入）
   - 每个人的提成比例和金额（自动计算）

6. 提成比例说明：
   - 系统优先使用合同参与人设置的比例
   - 如果没有设置则使用提成规则的默认比例
   - 财务可以在计算时修改比例或直接修改提成金额

7. 点击「确定」保存，提成记录会出现在「提成记录」标签页

::: tip 提成计算说明
- 提成比例已自动载入合同参与人设定的比例，财务可以修改
- 修改提成比例后，提成金额会自动计算（收款金额 × 提成比例）
- 也可以直接修改提成金额（财务最终结算时使用）
:::

### 场景七：审批和发放提成

**提成计算完成后，需要审批和发放。**

**查看提成记录：**

1. 在「提成管理」页面，「提成记录」标签页下有多个子标签：
   - **全部**：所有提成记录
   - **待审批**：等待审批的提成
   - **已审批**：审批通过待发放的提成
   - **已发放**：已经发放的提成

**审批提成：**

1. 切换到「待审批」标签
2. 查看待审批的提成记录
3. 点击「审批」按钮
4. 选择「通过」或「拒绝」
   - 拒绝需要填写原因

**发放提成：**

1. 切换到「已审批」标签（或在全部中筛选已审批状态）
2. 找到已审批通过的提成记录
3. 点击「发放」确认已发放
4. 系统记录发放时间

**批量操作：**

- 勾选多条记录
- 点击「批量审批」或「批量发放」
- 一次性处理多条记录

### 场景八：开具发票

**客户需要开发票。**

1. 点击「财务管理」→「发票管理」
2. 页面有多个标签页：全部、待审批、待开票、已开票
3. 使用年份筛选（默认显示当前年份，可选择「全部年份」）
4. 点击「申请开票」
5. 填写发票信息：

| 字段 | 必填 | 说明 |
|------|------|------|
| 客户 | 是 | 选择开票客户 |
| 发票抬头 | 是 | 开票单位名称 |
| 发票类型 | 是 | 增值税普通发票/增值税专用发票/其他 |
| 发票金额 | 是 | 发票金额（不含税） |
| 税率(%) | 否 | 税率百分比，如 6% |
| 税额 | 自动 | 根据金额和税率自动计算 |
| 总金额 | 自动 | 金额 + 税额 |
| 备注 | 否 | 其他信息 |

6. 提交后等待审批
7. 审批通过后，点击「开具」并输入发票号码
8. 开票完成

**发票状态说明：**

| 状态 | 含义 |
|------|------|
| 待审批 (PENDING) | 等待审批 |
| 待开票 (APPROVED) | 审批通过，等待开具 |
| 已开票 (ISSUED) | 发票已开具 |
| 已作废 (CANCELLED) | 发票已作废 |

**作废发票：**

如果发票开错了，可以作废：
1. 找到需要作废的发票记录（状态为「已开票」）
2. 点击「作废」
3. 填写作废原因
4. 确认作废

### 场景九：审核费用报销

**律师提交了报销申请，我需要审核。**

费用报销的审核在「工作台」→「审批中心」进行，详见[审批中心](./workbench.md#审批中心)。

### 场景十：查看财务报表

**我想了解律所的财务状况。**

1. 点击「财务管理」→「财务报表」
2. 页面顶部显示四个关键指标卡片：
   - **年度收入**：选定时间范围内的合同总额
   - **年度支出**：费用报销总额
   - **应收账款**：待收款金额
   - **利润率**：(收入-支出)/收入

3. 选择时间范围：
   - 最近3个月（默认）
   - 最近6个月
   - 本年度
   - 自定义日期范围

4. 查看不同维度的报表（标签页切换）：

**收入概览标签：**
- 合同总数、合同总额、已收款、待收款
- 按月统计的收入表格
- 显示合同金额、收款金额、开票金额、提成支出

**发票统计标签：**
- 发票总数
- 开票总额
- 待开票数量
- 已作废数量

**提成报表标签：**
- 提成总额、已发放、待发放、涉及人数
- 各律师的提成汇总表格
- 显示提成笔数、总额、已发放金额、待发放金额

**费用报销标签：**
- 报销总额
- 报销笔数
- 平均报销金额
- 报销明细列表（费用类型、申请人、金额、状态、申请日期）

5. 点击「导出报表」按钮，导出数据进行进一步分析

### 场景十一：查看合同财务信息

**我想查看某个合同的财务情况。**

1. 点击「财务管理」→「合同管理」
2. 这里显示所有项目的收费合同（只读）
3. 可以搜索和筛选：
   - 按合同编号搜索
   - 按合同名称搜索
   - 按客户筛选
   - 按状态筛选（草稿/待审批/生效中/已终止/已完成）
4. 点击「查看」查看合同详情：
   - 合同基本信息（名称、客户、项目）
   - 合同类型（服务合同/顾问合同/代理合同/其他）
   - 收费类型（固定收费/计时收费/风险代理/混合收费）
   - 合同金额和货币
   - 签订日期、生效日期、到期日期
   - 已收金额
   - 付款条件
   - 备注

::: tip 合同管理
财务模块的合同管理是只读的，仅供查看。如需修改合同信息，请到「合同管理」模块操作。
:::

## 第三部分：管理层操作指南

作为管理层，除了可以使用财务人员的所有功能外，你还可以设置提成规则、审批重要事项。

### 场景十二：设置提成规则

**我想设置或调整律所的提成分配规则。**

::: warning 权限说明
只有拥有 `finance:commission:rule:create` 权限的用户可以管理提成规则。
:::

1. 点击「财务管理」→「提成管理」→「提成规则」
2. 查看现有规则列表，显示：规则编码、规则名称、律所比例、主办律师比例、案源人比例、状态、是否默认
3. 点击「新建规则」创建新规则
4. 填写规则信息：

| 字段 | 必填 | 说明 |
|------|------|------|
| 规则编码 | 是 | 规则的唯一编码，如「RULE001」 |
| 规则名称 | 是 | 如「标准提成方案」「顾问业务提成方案」 |
| 律所比例 (%) | 是 | 律所留存的百分比，如 30% |
| 主办律师比例 (%) | 是 | 主办律师的提成比例，如 50% |
| 协办律师比例 (%) | 否 | 协办律师的提成比例，如 10% |
| 辅助人员比例 (%) | 否 | 辅助人员的提成比例，如 5% |
| 案源人比例 (%) | 否 | 案源人的提成比例，如 20% |
| 允许修改比例 | 否 | 是否允许在计算提成时修改比例 |
| 是否默认规则 | 否 | 设为默认规则后，新合同自动使用此规则 |
| 启用状态 | 是 | 启用/禁用 |
| 规则描述 | 否 | 规则的详细说明 |

5. 点击「确定」保存

**提成计算示例：**

假设使用以下规则：
- 律所比例：30%
- 主办律师比例：50%
- 案源人比例：20%

客户付款 100,000 元：
```
收款金额：100,000 元
律所留存：100,000 × 30% = 30,000 元
主办律师提成：100,000 × 50% = 50,000 元
案源人提成：100,000 × 20% = 20,000 元
```

**编辑规则：**

1. 找到要修改的规则
2. 点击「编辑」
3. 修改相关信息
4. 保存

::: warning 修改规则的影响
修改规则只影响之后的提成计算，已计算的提成不会自动重新计算。
:::

**删除规则：**

1. 找到要删除的规则
2. 点击「删除」
3. 确认删除

::: danger 注意
如果规则已被合同使用，删除可能会影响提成计算。建议改为「禁用」而不是删除。
:::

### 场景十三：处理合同变更

**律师端修改了合同信息，我需要同步到财务数据。**

1. 点击「财务管理」→「合同变更处理」
2. 查看待处理的合同变更列表，显示：
   - 变更编号
   - 合同ID
   - 变更类型（金额变更/参与人变更/付款计划变更/其他变更）
   - 变更说明
   - 变更时间
   - 是否影响收款（红色标签表示会影响已有收款记录）
   - 状态

3. 对于每条变更记录，可以选择：
   - **同步**：将变更同步到财务数据
   - **忽略**：不同步此变更（需要填写忽略原因）

**变更状态说明：**

| 状态 | 含义 |
|------|------|
| 待处理 (PENDING) | 等待财务处理 |
| 已同步 (SYNCED) | 已同步到财务数据 |
| 已忽略 (IGNORED) | 财务选择不同步 |
| 部分同步 (PARTIAL) | 部分内容已同步 |

::: warning 注意
如果变更会影响已有收款记录（「影响收款」显示为红色「是」），请谨慎处理，确认后再同步。
:::

### 场景十四：审批收款变更

**收款记录已锁定，需要变更时需要审批。**

1. 点击「财务管理」→「收款变更审批」
2. 页面顶部显示统计卡片：待审批、已批准、已拒绝、总申请
3. 查看变更申请列表，显示：
   - 收款编号
   - 原金额
   - 新金额
   - 差额（绿色表示增加，红色表示减少）
   - 变更原因
   - 申请人
   - 申请时间
   - 状态
   - 审批人

4. 使用标签页筛选：
   - **全部申请**：所有变更申请
   - **待审批**：等待审批的申请
   - **已处理**：已批准或已拒绝的申请

5. 对于待审批的申请，可以：
   - **查看**：查看变更详情
   - **通过**：批准变更申请
   - **拒绝**：拒绝变更申请（需要填写拒绝原因）

**变更状态说明：**

| 状态 | 含义 |
|------|------|
| 待审批 (PENDING) | 等待审批 |
| 已批准 (APPROVED) | 审批通过，变更已生效 |
| 已拒绝 (REJECTED) | 审批拒绝，变更未生效 |

### 场景十五：查看整体财务状况

作为管理层，你可以通过「财务报表」全面了解律所的财务状况：

1. **收入趋势**：查看收入概览，了解每月收入变化
2. **应收账款**：关注待收款金额，及时催收
3. **提成支出**：查看提成报表，了解人力成本
4. **费用控制**：查看费用报销，控制运营成本
5. **利润分析**：通过利润率指标评估经营效益

## 常见问题

**Q: 为什么我看不到「收款管理」菜单？**

A: 「收款管理」只有财务人员和管理层能看到。如果你是律师，只能看到「我的收款」查看自己项目的收款情况。

**Q: 提成什么时候能看到？**

A: 客户付款 → 财务登记收款 → 财务计算提成 → 财务审批 → 你就能在「我的提成」看到了。整个流程通常需要 1-3 个工作日。

**Q: 报销多久能到账？**

A: 取决于审批流程和财务安排。一般流程：提交 → 审批（1-2天）→ 打款（1-3天）。

**Q: 提成比例是固定的吗？**

A: 不一定。律所可以设置多个提成规则，不同类型的业务可能使用不同的规则。具体比例请咨询财务或管理层。

**Q: 我是授薪律师，为什么看不到提成？**

A: 授薪律师拿固定月薪，不参与提成分配。如果你认为应该有提成，请联系人事确认你的薪酬类型。

**Q: 收款登记错了怎么办？**

A: 如果收款记录已锁定，需要通过「收款变更审批」流程申请变更，管理层审批通过后才能生效。

**Q: 可以分期收款吗？**

A: 可以。客户每次付款，财务都可以登记一笔收款。系统会自动累计已收金额，计算待收金额。

**Q: 发票开错了怎么办？**

A: 联系财务人员作废发票，然后重新开具。

**Q: 报销被拒绝了怎么办？**

A: 查看拒绝原因，修改后重新提交。常见原因：费用说明不详细、费用类型选择错误等。

**Q: 合同变更后财务数据会自动更新吗？**

A: 不会自动更新。律师端修改合同后，财务需要在「合同变更处理」中确认同步，才会更新财务数据。
