# 常见问题

## 登录问题

### 我忘记密码了，怎么办？

联系你的系统管理员，让他帮你重置密码。重置后你会得到一个新的临时密码，登录后记得修改成自己的密码。

### 我的账号被锁定了，怎么办？

如果你连续输错密码多次，账号会被临时锁定。你可以：
- 等待30分钟后自动解锁
- 或者联系管理员立即解锁

### 我登录后看不到某些菜单，是怎么回事？

系统会根据你的角色显示不同的菜单。比如：
- 律师看不到「收款管理」，只能看到「我的收款」
- 只有行政人员能看到「考勤管理」「资产管理」等
- 只有管理员能看到「系统管理」

如果你觉得应该有权限但看不到，请联系管理员检查你的角色配置。

### 页面一直在加载，进不去系统？

可能的原因和解决方法：
1. **网络问题**：检查网络连接是否正常
2. **浏览器缓存**：按 `Ctrl+F5`（Mac 是 `Cmd+Shift+R`）强制刷新
3. **浏览器兼容性**：换用 Chrome 浏览器试试
4. **系统维护**：联系管理员确认系统是否正常运行

## 客户管理问题

### 为什么创建客户时要做利冲检索？

这是律师执业的基本要求。律师不能同时代理利益对立的双方。比如你不能一边帮张三告李四，一边又帮李四告张三。

利冲检索就是检查：新客户的对方当事人，是不是我们的老客户？

### 利冲检索发现冲突了，怎么办？

如果确实存在冲突，通常需要放弃这个案子。

但在特殊情况下（比如双方都书面同意），可以申请「利冲豁免」。这需要主任审批。

### 客户信息填错了，能改吗？

可以。在客户列表中找到客户，点击「编辑」就能修改。

但客户名称和证件号码创建后不能直接修改，如需修改请联系管理员。

### 客户编号是怎么生成的？

客户编号由系统自动生成，格式为 `C + 年份 + 序号`，如 `C202601001`。你不需要手动填写。

## 项目管理问题

### 为什么我不能直接创建项目，必须先创建合同？

这是系统的设计要求，目的是确保每个案件都有正式的委托手续。

正确的流程是：录入客户 → 创建合同 → 合同审批 → 创建项目

### 合同审批要多久？

取决于你们律所的审批流程。小金额合同可能当天就能批，大金额合同可能需要几天。

你可以在「合同管理」中查看审批进度，也可以联系审批人催促。

### 一个合同可以创建多个项目吗？

可以。比如一份法律顾问合同，可能会产生多个具体的服务项目。

### 项目创建后还能修改吗？

可以，在项目结案之前都可以修改。结案后就不能改了。

### 案件有多个委托人怎么办？

创建项目时，可以添加多个客户。点击「添加共同委托人」就行。

### 项目编号是怎么生成的？

项目编号由系统自动生成，格式为 `M + 年份 + 序号`，如 `M202601001`。

## 财务问题

### 我怎么知道客户付款了没有？

进入「财务管理」→「我的收款」，可以看到你负责项目的收款情况。

### 我的提成是怎么算的？

客户付款后，系统会自动计算提成。大致逻辑是：

1. 先扣除律所留存（比如30%）
2. 再计算案源提成（给介绍案子的人）
3. 剩下的是办案提成（给主办律师和协办律师分）

具体比例由律所管理层设定。

### 我是授薪律师，有提成吗？

授薪律师拿固定月薪，不参与提成分配。

### 报销提交后多久能到账？

取决于审批流程和财务安排。你可以在「费用报销」页面查看审批进度。

### 为什么我看不到「收款管理」菜单？

「收款管理」只有财务人员和管理层能看到。如果你是律师，只能看到「我的收款」。

## 薪资问题

### 我的工资是怎么算的？

工资主要由提成构成。计算公式：
- **收入**：你的提成总额（自动从提成管理导入）
- **应发工资** = 收入 - 税费扣减项（个税、社保、公积金）
- **实发工资** = 应发工资 - 其他扣减项（办公室租金、欠款等）

### 为什么我的工资是负数？

当扣减项（税费、办公室租金、欠款等）大于收入（提成）时，工资会显示为负数，表示你需要补缴费用。

### 工资确认截止时间是什么意思？

财务提交工资表后，你需要在截止时间前确认。超过截止时间（默认每月27日24时），系统会自动确认。

### 我拒绝了工资确认怎么办？

财务会收到通知，核实后会联系你沟通。如果确实有误，财务会修改后重新提交确认。

### 我在哪里查看工资明细？

进入「人力资源」→「我的工资」，可以查看每月的工资明细，包括收入项（每笔提成的来源）和扣减项。

### 我是授薪律师，有工资表吗？

授薪律师拿固定月薪，不参与提成分配。你的工资由人事部门单独处理，可能不在这个系统中显示。

## 文档管理问题

### 上传文件失败怎么办？

检查一下：
1. 文件是否超过100MB
2. 网络是否正常
3. 文件格式是否支持

如果还是不行，刷新页面重试。

### 可以批量上传吗？

可以，在上传窗口中可以一次选择多个文件。

### 文件夹是怎么自动生成的？

系统会根据项目类型自动创建标准文件夹。比如刑事案件会有「会见笔录」「辩护词」等文件夹。

### 删除的文件能恢复吗？

不能。删除后文件无法恢复，请谨慎操作。

## 档案管理问题

### 档案和文档有什么区别？

- **文档**：指电子文件，存储在系统中，随时可以查看下载
- **档案**：指实体材料（纸质文件），存放在档案室，需要借阅才能查看

### 保管期限是多久？

根据档案类型和律所规定不同，一般案件档案保管10-30年。具体请咨询档案管理员。

### 什么是档案迁移？

档案迁移是将本系统中的档案转移到外部档案管理系统（如市档案馆、区档案馆等）的过程。迁移后，档案信息会保留在系统中用于查询，但实体文件可以选择删除以节省存储空间。

### 我能直接迁移档案吗？

不能。迁移档案需要提交申请，经过审批后才能执行迁移。只有已入库的档案才能申请迁移。

### 档案迁移后还能在系统中查看吗？

档案信息会保留，可以查看基本信息和迁移记录。但如果审批时选择了删除文件，则无法查看或下载文件内容。

## 行政管理问题

### 请假天数是怎么算的？

系统会根据开始时间和结束时间自动计算请假天数。

### 为什么我看不到「会议室管理」？

「会议室管理」只有行政人员可以看到。普通员工只能预约会议室。

### 会议室显示「维护中」是什么意思？

说明会议室正在维修或保养，暂时无法预约。请选择其他会议室。

## 数据交接问题

### 什么时候需要做数据交接？

当员工离职、项目负责人变更、客户负责人变更时，需要进行数据交接，确保业务连续性。

### 数据交接后原负责人还能看到数据吗？

交接完成后，原负责人将不再是数据的负责人。但如果他仍是项目的参与人，可能仍有查看权限。

### 数据交接可以撤销吗？

已确认执行的交接不能撤销。如果发现问题，需要创建新的交接单进行反向移交。

### 谁可以发起数据交接？

只有超级管理员、律所主任和合伙人可以发起数据交接。

## 其他问题

### 系统太慢了，怎么办？

试试以下方法：
1. 刷新页面
2. 清除浏览器缓存
3. 换个浏览器试试（推荐 Chrome）
4. 检查网络连接

如果还是慢，联系管理员。

### 我发现了系统 bug，怎么反馈？

联系系统管理员，描述清楚：
1. 你在做什么操作
2. 出现了什么问题
3. 如果有报错信息，截图发给他

### 我有功能建议，怎么提？

联系系统管理员或你们的信息化负责人。

### 可以在手机上使用吗？

系统主要为电脑端设计，在手机上可能显示不完整。建议使用电脑访问。

### 数据会丢失吗？

系统会定期备份数据，一般不会丢失。但建议重要文件也在本地保留备份。
