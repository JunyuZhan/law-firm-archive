# 行政管理

行政管理模块用于管理律所的日常行政事务，包括考勤、请假、会议室、资产等。

## 考勤管理

**菜单位置**：行政管理 → 考勤管理

管理员工的考勤记录。

### 考勤列表

查看员工的打卡记录，包括：

- 员工姓名
- 打卡日期
- 上班时间
- 下班时间
- 考勤状态（正常/迟到/早退/缺勤）

---

## 请假管理

**菜单位置**：行政管理 → 请假管理

提交和管理请假申请。

### 申请请假

1. 点击「申请请假」
2. 填写请假信息：

| 字段     | 必填 | 说明                       |
| -------- | :--: | -------------------------- |
| 请假类型 |  ✅  | 年假/事假/病假/婚假/产假等 |
| 开始日期 |  ✅  | 请假开始日期               |
| 结束日期 |  ✅  | 请假结束日期               |
| 请假天数 |  ✅  | 请假天数                   |
| 请假事由 |  ✅  | 说明请假原因               |

3. 提交审批

### 请假审批

请假申请由部门负责人或主任审批。

### 请假状态

| 状态      | 说明   |
| --------- | ------ |
| PENDING   | 待审批 |
| APPROVED  | 已批准 |
| REJECTED  | 已拒绝 |
| CANCELLED | 已取消 |

---

## 会议室

**菜单位置**：行政管理 → 会议室

预约和管理会议室。

### 会议室列表

查看所有会议室及其状态：

- 会议室名称
- 容纳人数
- 设备配置
- 当前状态（空闲/使用中）

### 预约会议室

1. 选择会议室
2. 选择日期和时间段
3. 填写会议主题
4. 添加参会人员（可选）
5. 提交预约

### 查看预约

可以查看自己的预约记录和所有会议室的预约情况。

## 资产管理

**菜单位置**：行政管理 → 资产管理

管理律所的固定资产，包括电脑设备、办公家具、办公设备、车辆等。

### 统计概览

页面顶部显示统计卡片：

- **资产总数**：总资产数量
- **在用资产**：当前被领用的资产数量
- **闲置资产**：可用的资产数量
- **资产总值**：所有资产的总价值（人民币）

### 查询筛选

| 筛选项   | 说明                                 |
| -------- | ------------------------------------ |
| 资产类型 | 电脑设备/办公家具/办公设备/车辆/其他 |
| 状态     | 闲置/在用/维修中/已报废              |
| 关键词   | 搜索资产名称、品牌、型号等           |

### 资产列表字段

| 列名      | 说明                                                    |
| --------- | ------------------------------------------------------- |
| 资产编号  | 系统自动生成                                            |
| 资产名称  | 资产名称（如「ThinkPad笔记本」）                        |
| 资产类型  | 电脑设备/办公家具/办公设备/车辆/其他                    |
| 品牌/型号 | 品牌和型号（如「ThinkPad X1 Carbon」）                  |
| 使用人    | 当前使用人（闲置时显示为空）                            |
| 存放位置  | 资产存放位置（如「行政部仓库」）                        |
| 购入日期  | 资产购买日期                                            |
| 状态      | 闲置（灰色）/在用（绿色）/维修中（橙色）/已报废（红色） |
| 操作      | 查看/编辑/领用/归还/报废                                |

### 场景一：添加资产

1. 点击「添加资产」按钮
2. 填写资产信息：

| 字段     | 必填 | 说明                                 |
| -------- | :--: | ------------------------------------ |
| 资产名称 |  ✅  | 如「ThinkPad笔记本」                 |
| 资产类型 |  ✅  | 电脑设备/办公家具/办公设备/车辆/其他 |
| 品牌     |  ❌  | 品牌                                 |
| 型号     |  ❌  | 型号                                 |
| 规格     |  ❌  | 规格参数                             |
| 序列号   |  ❌  | 序列号/SN                            |
| 购入日期 |  ❌  | 购入日期                             |
| 购入价格 |  ❌  | 购入价格（人民币）                   |
| 供应商   |  ❌  | 供应商名称                           |
| 保修到期 |  ❌  | 保修到期日期                         |
| 存放位置 |  ❌  | 存放位置                             |
| 备注     |  ❌  | 其他说明                             |

3. 点击「确定」保存

### 场景二：领用资产

1. 找到状态为「闲置」的资产
2. 点击「领用」
3. 选择领用人（按部门筛选）
4. 填写预计归还日期（可选）和领用原因
5. 点击「确定」完成领用

### 场景三：归还资产

1. 找到状态为「在用」的资产
2. 点击「归还」
3. 确认归还后，资产状态变为「闲置」

### 场景四：报废资产

1. 找到状态为「闲置」的资产
2. 点击「报废」
3. 确认报废后，资产状态变为「已报废」

### 场景五：查看资产详情

1. 点击「查看」按钮
2. 查看资产详细信息，包括：
   - 基本信息：资产编号、名称、类型、品牌/型号、序列号等
   - 状态信息：当前状态、使用人、存放位置、保修状态等
   - 操作记录：领用、归还、维修等历史记录

### 资产盘点

定期进行资产盘点，核对资产实际情况。系统支持导出资产清单进行实地核对。

::: tip 资产状态说明

- **闲置**：资产可用，未被领用
- **在用**：资产已被领用，正在使用中
- **维修中**：资产正在维修，不可用
- **已报废**：资产已报废，不可用 :::

## 出函管理

**菜单位置**：行政管理 → 出函管理

管理律师函的申请和审批。

### 申请出函

1. 在项目详情页点击「申请出函」
2. 选择函件模板
3. 填写出函信息：

| 字段     | 必填 | 说明         |
| -------- | :--: | ------------ |
| 函件模板 |  ✅  | 选择出函模板 |
| 接收单位 |  ✅  | 函件送达单位 |
| 出函事由 |  ✅  | 出函原因     |
| 出函律师 |  ✅  | 署名律师     |
| 审批人   |  ✅  | 选择审批人   |
| 份数     |  ✅  | 出函份数     |

4. 提交审批

### 出函审批

出函申请需要主任或管理员审批。

### 出函状态

| 状态     | 说明   |
| -------- | ------ |
| PENDING  | 待审批 |
| APPROVED | 已审批 |
| REJECTED | 已拒绝 |
| PRINTED  | 已打印 |
| RECEIVED | 已签收 |

## 常见问题

**Q: 请假可以撤回吗？**

待审批的请假可以取消，已批准的请假需要重新申请销假。

**Q: 会议室可以取消预约吗？**

可以在预约时间前取消。

**Q: 资产借用怎么操作？**

联系行政人员办理资产借用手续。
