## 考勤管理

**菜单位置**：行政管理 → 考勤管理

管理员工的考勤记录。

### 考勤列表

查看员工的打卡记录，包括：
- 员工姓名
- 打卡日期
- 上班时间
- 下班时间
- 考勤状态（正常/迟到/早退/缺勤）

## 请假管理

**菜单位置**：行政管理 → 请假管理

提交和管理请假申请。

### 申请请假

1. 点击「申请请假」
2. 填写请假信息：

| 字段 | 必填 | 说明 |
|------|:----:|------|
| 请假类型 | ✅ | 年假/事假/病假/婚假/产假等 |
| 开始日期 | ✅ | 请假开始日期 |
| 结束日期 | ✅ | 请假结束日期 |
| 请假天数 | ✅ | 请假天数 |
| 请假事由 | ✅ | 说明请假原因 |

3. 提交审批

### 请假审批

请假申请由部门负责人或主任审批。

### 请假状态

| 状态 | 说明 |
|------|------|
| PENDING | 待审批 |
| APPROVED | 已批准 |
| REJECTED | 已拒绝 |
| CANCELLED | 已取消 |

---

## 会议室

**菜单位置**：行政管理 → 会议室

预约和管理会议室。

### 会议室列表

查看所有会议室及其状态：
- 会议室名称
- 容纳人数
- 设备配置
- 当前状态（空闲/使用中）

### 预约会议室

1. 选择会议室
2. 选择日期和时间段
3. 填写会议主题
4. 添加参会人员（可选）
5. 提交预约

### 查看预约

可以查看自己的预约记录和所有会议室的预约情况。

## 出函管理

**菜单位置**：行政管理 → 出函管理

管理律师函的申请和审批。

### 申请出函

1. 在项目详情页点击「申请出函」
2. 选择函件模板
3. 填写出函信息：

| 字段 | 必填 | 说明 |
|------|:----:|------|
| 函件模板 | ✅ | 选择出函模板 |
| 接收单位 | ✅ | 函件送达单位 |
| 出函事由 | ✅ | 出函原因 |
| 出函律师 | ✅ | 署名律师 |
| 审批人 | ✅ | 选择审批人 |
| 份数 | ✅ | 出函份数 |

4. 提交审批

### 出函审批

出函申请需要主任或管理员审批。

### 出函状态

| 状态 | 说明 |
|------|------|
| PENDING | 待审批 |
| APPROVED | 已审批 |
| REJECTED | 已拒绝 |
| PRINTED | 已打印 |
| RECEIVED | 已签收 |

## 常见问题

**Q: 请假可以撤回吗？**

待审批的请假可以取消，已批准的请假需要重新申请销假。

**Q: 会议室可以取消预约吗？**

可以在预约时间前取消。

**Q: 资产借用怎么操作？**

联系行政人员办理资产借用手续。
