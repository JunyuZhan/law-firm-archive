## 考勤管理

### 进入考勤管理

点击左侧菜单「行政管理」→「考勤管理」

页面顶部显示今日打卡状态和打卡按钮，下方显示考勤统计和记录列表。

---

### 场景一：每日打卡

1. 进入「考勤管理」页面
2. 页面顶部显示「今日考勤」卡片
3. **上班打卡**：点击「上班打卡」按钮（打卡后按钮变灰）
4. **下班打卡**：点击「下班打卡」按钮（需先完成上班打卡）

::: tip 打卡状态
- 正常：按时打卡
- 迟到：超过上班时间打卡
- 早退：早于下班时间打卡
- 缺勤：当天未打卡
- 请假：当天有请假记录
- 加班：当天有加班记录
:::

---

### 场景二：查看考勤记录

1. 在考勤管理页面，使用日期选择器选择日期
2. 表格显示当天所有员工的考勤记录：
   - 员工姓名、部门
   - 日期
   - 上班打卡时间、下班打卡时间
   - 工作时长
   - 状态
   - 备注
3. 点击「详情」查看某条记录的详细信息（包括打卡地点）

**统计卡片说明**：
- 今日出勤：正常打卡人数
- 迟到：迟到人数
- 早退：早退人数
- 缺勤：未打卡人数

## 请假管理

### 进入请假管理

点击左侧菜单「行政管理」→「请假管理」

页面有三个标签页：
- **我的申请**：查看自己提交的请假申请
- **待审批**：查看需要你审批的请假申请（仅主管可见）
- **全部申请**：查看所有请假申请（仅管理员可见）

---

### 场景三：我要请假

1. 点击右上角「申请请假」按钮
2. 填写请假信息：

| 字段 | 说明 |
|------|------|
| 请假类型 | 从系统配置的请假类型中选择（如年假、事假、病假等） |
| 开始时间 | 请假开始的日期和时间 |
| 结束时间 | 请假结束的日期和时间 |
| 请假事由 | 简单说明请假原因 |

3. 点击「确定」提交申请

**提交后**：
- 申请会发送给你的上级审批
- 你可以在「我的申请」中查看审批进度
- 系统自动计算请假天数

::: tip 提前申请
建议提前申请请假，给审批人留出处理时间。紧急情况可以先电话沟通，再补申请。
:::

---

### 场景四：审批下属的请假

1. 进入「请假管理」，切换到「待审批」标签页
2. 找到需要审批的申请，点击「审批」
3. 选择「通过」或「拒绝」
4. 填写审批意见（拒绝时建议说明原因）
5. 点击「确定」

### 场景五：取消请假申请

1. 在「我的申请」中找到状态为「待审批」的申请
2. 点击「取消」
3. 确认后申请状态变为「已取消」

---

## 会议室预约

### 进入会议室预约

点击左侧菜单「行政管理」→「会议室预约」

页面分为两个标签页：
- **预约会议室**：预约和查看会议室（所有人可用）
- **会议室管理**：管理会议室（仅管理员和行政人员可见）

---

### 场景六：我要预约会议室

1. 在「预约会议室」标签页，左侧显示所有会议室
2. 点击选中一个会议室（维护中的无法选择）
3. 右侧显示该会议室的预约情况
4. 点击「预约会议室」按钮
5. 填写预约信息：

| 字段 | 说明 |
|------|------|
| 会议室 | 确认选择的会议室（显示容纳人数） |
| 会议主题 | 如「项目讨论会」（必填） |
| 开始时间 | 会议开始时间（必填） |
| 结束时间 | 会议结束时间（必填） |
| 会议说明 | 可选，备注参会人员等 |

6. 点击「确定」完成预约

**会议室状态说明**：
- 可用（绿色）：可以预约
- 使用中（蓝色）：当前有会议进行
- 维护中（黄色）：无法预约

---

### 场景七：取消会议室预约

1. 在预约列表中找到状态为「已预约」的记录
2. 点击「取消」
3. 确认后预约被取消

---

### 场景八：管理会议室（管理员/行政）

1. 切换到「会议室管理」标签页
2. **添加会议室**：点击「添加会议室」，填写：
   - 名称（必填）：如「大会议室A」
   - 位置：如「3楼东侧」
   - 容纳人数：如 20
   - 设备：如「投影仪、视频会议系统」
3. **编辑会议室**：点击「编辑」修改信息
4. **设置维护**：点击「维护」，会议室变为维护中状态，无法被预约
5. **启用会议室**：维护完成后点击「启用」恢复
6. **删除会议室**：点击「删除」移除会议室

---

## 外出管理

### 进入外出管理

点击左侧菜单「行政管理」→「外出管理」

---

### 场景九：登记外出

工作时间需要外出办事时：

1. 点击「外出登记」按钮
2. 填写外出信息：

| 字段 | 说明 |
|------|------|
| 外出时间 | 外出的时间（必填） |
| 预计返回时间 | 预计什么时候回来 |
| 外出地点 | 去哪里，如「XX法院」 |
| 外出事由 | 外出原因（必填），如「开庭」 |
| 同行人员 | 一起外出的同事 |

3. 点击「确定」完成登记

---

### 场景十：登记返回

外出回来后：

1. 在外出记录列表中找到你的外出记录（状态为「外出中」）
2. 点击「登记返回」
3. 确认后，状态变为「已返回」，系统记录实际返回时间

::: tip 查看当前外出
点击「当前外出」按钮可以快速查看目前所有外出中的人员。
:::

---

### 场景十一：查询外出记录

1. 使用日期范围选择器筛选记录
2. 点击「查询」查看指定日期范围的外出记录
3. 点击「重置」清除筛选条件

---

## 资产管理

### 进入资产管理

点击左侧菜单「行政管理」→「资产管理」

页面顶部显示资产统计（总数、在用、闲置、总值），下方是资产列表。

---

### 场景十五：添加新资产

1. 点击「添加资产」按钮
2. 填写资产信息：

| 字段 | 说明 |
|------|------|
| 资产名称 | 如「ThinkPad笔记本」（必填） |
| 资产类型 | 电脑设备/办公家具/办公设备/车辆/其他（必填） |
| 品牌 | 品牌名称 |
| 型号 | 具体型号 |
| 规格 | 规格参数 |
| 序列号 | 设备序列号/SN |
| 购入日期 | 购买日期 |
| 购入价格 | 购买金额 |
| 供应商 | 供应商名称 |
| 保修到期 | 保修截止日期 |
| 存放位置 | 资产存放位置 |
| 备注 | 其他说明 |

3. 点击「确定」保存

---

### 场景十六：资产领用

员工需要使用某个资产时：

1. 在资产列表中找到状态为「闲置」的资产
2. 点击「领用」
3. 填写领用信息：
   - 领用人（必填）：从部门树中选择
   - 预计归还日期
   - 领用原因
4. 点击「确定」

领用后，资产状态变为「在用」，并记录当前使用人。

---

### 场景十七：资产归还

1. 找到状态为「在用」的资产
2. 点击「归还」
3. 确认后，资产状态变回「闲置」

---

### 场景十八：资产报废

资产损坏或达到使用年限需要报废：

1. 找到状态为「闲置」的资产
2. 点击「报废」
3. 确认后，资产状态变为「已报废」

---

### 场景十九：查看资产详情和操作记录

1. 点击资产的「查看」按钮
2. 弹窗显示资产详细信息：
   - 资产编号、名称、类型
   - 品牌/型号、序列号
   - 状态、购入日期、购入价格
   - 使用人、存放位置
   - 保修到期、保修状态（保修中/已过保）
3. 下方显示操作记录时间线（领用、归还、报废等历史）

**资产状态说明**：
| 状态 | 颜色 | 含义 |
|------|------|------|
| 闲置 | 灰色 | 可以领用 |
| 在用 | 绿色 | 已被领用 |
| 维修中 | 黄色 | 正在维修 |
| 已报废 | 红色 | 已报废处理 |

## 资产盘点

### 进入资产盘点

点击左侧菜单「行政管理」→「资产盘点」

---

### 场景二十：创建盘点任务

1. 点击「新建盘点」
2. 填写盘点信息：

| 字段 | 说明 |
|------|------|
| 盘点日期 | 盘点的日期（必填） |
| 盘点类型 | 全盘（所有资产）或抽盘（部分资产）（必填） |
| 盘点部门 | 按部门筛选资产（可选） |
| 资产分类 | 按分类筛选：办公设备/IT设备/家具/车辆/其他 |
| 盘点位置 | 盘点的位置范围，如「3楼办公区」 |
| 备注 | 盘点说明或备注 |

3. 如果选择「抽盘」，需要选择要盘点的资产
4. 底部显示预计盘点资产数量
5. 点击「确定」创建盘点任务

---

### 场景二十一：执行盘点

1. 点击盘点任务的「查看」
2. 查看盘点明细列表，包含：
   - 资产编号、资产名称
   - 预期状态、实际状态
   - 预期位置、实际位置
   - 预期使用人、实际使用人
   - 差异类型
3. 对每个资产点击「更新」，填写：
   - 实际状态
   - 实际位置
   - 差异描述
4. 全部盘点完成后，点击「完成盘点」

**盘点统计**：
- 总数：应盘点资产数
- 实盘数：已盘点资产数
- 盘盈：多出的资产
- 盘亏：缺少的资产

## 出函管理（介绍信）

### 进入出函管理

点击左侧菜单「行政管理」→「出函管理」

页面顶部显示统计卡片（待审批、待打印、待领取、已完成、总申请），下方是申请列表。

页面有三个标签页：
- **全部申请**：所有出函申请
- **待办理**：待审批、待打印、待领取的申请
- **已完成**：已领取、已拒绝、已取消的申请

---

### 场景二十六：审批出函申请

1. 在列表中找到状态为「待审批」的申请
2. 可以执行以下操作：
   - **通过**：点击「通过」直接审批通过
   - **退回**：点击「退回」要求申请人修改，需填写退回原因
   - **拒绝**：点击「拒绝」拒绝申请，需填写拒绝原因

---

### 场景二十七：查看和编辑函件内容

1. 点击「查看」打开详情弹窗
2. 弹窗显示：
   - 基本信息：申请编号、函件类型、项目、项目编号、接收单位、联系人、联系电话、份数、出函律师、期望日期、申请人、状态
   - 出函事由
   - 审批意见（如有）
   - 时间记录：申请时间、审批时间、打印时间、领取时间
3. 下方是函件内容编辑器（富文本），可以修改函件正文
4. 点击「保存内容」保存修改
5. 点击「打印」可以直接打印当前编辑的内容

---

### 场景二十八：打印函件

1. 找到状态为「已审批」的申请
2. 点击「打印」打开打印预览窗口：
   - 函件包含验证二维码，可扫码验证真伪
   - 使用浏览器打印功能打印
3. 或点击「下载」下载为Word文档
4. 打印完成后，点击「确认已打印」，状态变为「已打印」

::: tip 补打功能
已打印或已领取的函件也可以点击「补打」或「下载」重新打印。
:::

---

### 场景二十九：确认领取

1. 找到状态为「已打印」的申请
2. 律师来领取函件时，点击「确认领取」
3. 状态变为「已领取」，流程完成

---

### 出函状态说明

| 状态 | 颜色 | 含义 |
|------|------|------|

## 供应商管理

### 进入供应商管理

点击左侧菜单「行政管理」→「供应商管理」

页面顶部显示统计卡片（供应商总数、启用、停用），下方是供应商列表。

---

### 场景三十：添加供应商

1. 点击「新增供应商」
2. 填写供应商信息：

| 字段 | 说明 |
|------|------|
| 供应商名称 | 公司名称（必填） |
| 供应商类型 | 服务商/供应商/合作伙伴 |
| 联系人 | 对接人姓名 |
| 联系电话 | 联系方式 |
| 联系邮箱 | 邮箱地址 |
| 地址 | 公司地址 |
| 统一社会信用代码 | 营业执照号 |
| 开户银行 | 银行名称 |
| 银行账号 | 付款账号 |
| 供应范围 | 提供什么产品或服务 |
| 评级 | A/B/C/D级 |

3. 点击「确定」保存

---

### 场景三十一：管理供应商状态

- **启用/停用**：点击「启用」或「停用」切换供应商状态
- **编辑**：点击「编辑」修改供应商信息
- **删除**：点击「删除」移除供应商

---

### 场景三十二：搜索和筛选供应商

1. 使用搜索框搜索供应商名称、联系人、电话
2. 使用下拉框筛选：
   - 供应商类型：服务商/供应商/合作伙伴
   - 状态：启用/停用
   - 评级：A/B/C/D级
3. 点击「搜索」执行筛选
4. 点击「重置」清除筛选条件

### 进入合同查询

此功能用于查看已审批通过的合同信息，主要用于司法局报备等行政工作。

### 场景三十五：查询合同信息

1. 使用搜索条件筛选：
   - 合同编号
   - 委托人名称
   - 案件类型（民事/刑事/行政/破产/知识产权/仲裁/执行/法律顾问/专项服务）

   - 合同编号、合同名称
   - 委托人、对方当事人

   - 签约日期、管辖法院、审理阶段

## 常见问题

**Q: 请假天数是怎么算的？**
A: 系统根据开始时间和结束时间自动计算请假天数。

**Q: 为什么我看不到「会议室管理」？**
A: 只有管理员和行政人员才能看到管理功能，普通员工只能预约。

**Q: 资产领用后可以转给别人吗？**
A: 需要先归还，再由新的使用人重新领用。

**Q: 采购申请可以修改吗？**
A: 草稿状态可以修改，提交后不能修改。如需修改，可以取消后重新申请。

**Q: 出函申请被退回怎么办？**
A: 查看退回原因，在项目详情的「出函记录」中修改后重新提交。

**Q: 出函申请在哪里提交？**
A: 出函申请由律师在项目详情页的「出函记录」标签中提交，行政人员在「行政管理」→「出函管理」中审批和打印。

**Q: 打印的函件有二维码是什么？**
A: 函件上的二维码用于验证函件真伪，扫码可查看函件的基本信息和状态。

**Q: 资产盘点的全盘和抽盘有什么区别？**
A: 全盘会盘点所有资产（可按部门、分类、位置筛选），抽盘需要手动选择要盘点的资产。

**Q: 加班申请需要审批吗？**
A: 是的，加班申请提交后需要主管审批通过才生效。
