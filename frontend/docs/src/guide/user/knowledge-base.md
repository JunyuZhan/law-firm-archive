# 知识库

知识库模块用于管理律所的知识文章、案例库、法规库等知识资源。

## 谁能使用？

所有用户都可以查看知识库内容。

| 功能     | 管理员 | 主任 | 团队负责人 | 财务 | 律师 | 行政 | 实习 |
| -------- | :----: | :--: | :--------: | :--: | :--: | :--: | :--: |
| 知识文章 |   ✅   |  ✅  |     ✅     |  ✅  |  ✅  |  ✅  |  ✅  |
| 案例库   |   ✅   |  ✅  |     ✅     |  ✅  |  ✅  |  ✅  |  ✅  |
| 法规库   |   ✅   |  ✅  |     ✅     |  ✅  |  ✅  |  ✅  |  ✅  |

## 知识文章

**菜单位置**：知识库 → 知识文章

管理和查阅律所内部的知识文章。

### 文章分类

- 法律法规
- 实务经验
- 文书模板
- 行业动态

### 查询筛选

| 筛选项   | 说明                                |
| -------- | ----------------------------------- |
| 文章标题 | 模糊搜索文章标题                    |
| 文章分类 | 法律法规/实务经验/文书模板/行业动态 |
| 状态     | 草稿/已发布/已归档                  |

### 文章列表字段

| 列名     | 说明               |
| -------- | ------------------ |
| 文章标题 | 文章标题           |
| 分类     | 文章分类           |
| 作者     | 文章作者           |
| 发布时间 | 发布时间           |
| 浏览量   | 阅读次数           |
| 点赞数   | 点赞数量           |
| 状态     | 草稿/已发布/已归档 |

### 场景一：发布文章

1. 点击「发布文章」按钮
2. 填写文章信息：

| 字段     | 必填 | 说明                                |
| -------- | :--: | ----------------------------------- |
| 文章标题 |  ✅  | 文章标题                            |
| 文章分类 |  -   | 法律法规/实务经验/文书模板/行业动态 |
| 文章内容 |  ✅  | 文章正文                            |

3. 点击「确定」保存

::: tip OCR 智能识别新增文章时支持 OCR 识别功能，上传文章截图可自动提取文字内容填充到文章正文中。:::

### 场景二：编辑文章

1. 在文章列表中找到要编辑的文章
2. 点击「编辑」
3. 修改文章信息
4. 点击「确定」保存

### 场景三：发布草稿

草稿状态的文章可以发布：

1. 找到状态为「草稿」的文章
2. 点击「发布」按钮
3. 文章状态变为「已发布」

### 场景四：归档文章

已发布的文章可以归档：

1. 找到状态为「已发布」的文章
2. 点击「归档」按钮
3. 文章状态变为「已归档」

### 场景五：点赞和收藏

- **点赞**：点击「点赞」增加文章点赞数
- **收藏**：点击「收藏」将文章加入收藏夹，再次点击取消收藏

### 场景六：删除文章

1. 点击「删除」按钮
2. 确认删除

---

## 案例库

**菜单位置**：知识库 → 案例库

管理典型案例，供律师参考学习。

### 查询筛选

| 筛选项   | 说明                                                     |
| -------- | -------------------------------------------------------- |
| 案由类型 | 民事/刑事/行政/破产/知识产权/仲裁/执行/法律顾问/专项服务等（与系统案件类型保持一致，由后台统一维护） |
| 案例分类 | 按分类树筛选                                             |
| 案例名称 | 模糊搜索案例名称                                         |

### 案例列表字段

| 列名     | 说明                         |
| -------- | ---------------------------- |
| 案例名称 | 案例标题                     |
| 案由类型 | 民事/刑事/行政等             |
| 审理法院 | 审理案件的法院               |
| 判决日期 | 判决日期                     |
| 案件结果 | 胜诉/部分胜诉/败诉/调解/撤诉 |
| 经办律师 | 办理该案的律师               |
| 参考价值 | 高/中/低                     |

### 场景七：添加案例

1. 点击「添加案例」按钮
2. 填写案例信息：

| 字段     | 必填 | 说明                                                     |
| -------- | :--: | -------------------------------------------------------- |
| 案例名称 |  ✅  | 如「张某诉李某合同纠纷案」                               |
| 案由类型 |  -   | 民事/刑事/行政/破产/知识产权/仲裁/执行/法律顾问/专项服务等，选项由系统预置，无需手动维护 |
| 案例分类 |  -   | 选择分类                                                 |
| 审理法院 |  -   | 如「北京市朝阳区人民法院」                               |
| 判决日期 |  -   | 判决日期                                                 |
| 案件结果 |  -   | 胜诉/部分胜诉/败诉/调解/撤诉                             |
| 经办律师 |  -   | 选择经办律师                                             |
| 参考价值 |  -   | 高/中/低                                                 |
| 案例摘要 |  -   | 案例摘要、裁判要旨等                                     |

3. 点击「确定」保存

::: tip 裁判文书智能识别添加案例时支持 OCR 识别功能，上传裁判文书截图可自动识别并填充案件名称、法院、判决日期、案件类型、案件结果等信息。:::

### 场景八：查看案例详情

1. 在案例列表中找到要查看的案例
2. 点击「查看」按钮
3. 查看案例详细信息

### 场景九：收藏案例

- 点击「收藏」将案例加入收藏夹
- 点击「取消收藏」移出收藏夹

### 场景十：编辑和删除案例

- **编辑**：点击「编辑」修改案例信息
- **删除**：点击「删除」删除案例

## 法规库

**菜单位置**：知识库 → 法规库

查询和管理法律法规。

### 页面布局

页面分为左右两栏：

- **左侧**：法规分类树，点击分类筛选法规
- **右侧**：法规列表

### 查询筛选

| 筛选项   | 说明                                       |
| -------- | ------------------------------------------ |
| 法规名称 | 模糊搜索法规名称                           |
| 法规类型 | 法律/行政法规/部门规章/司法解释/地方性法规 |
| 效力状态 | 现行有效/已修订/已废止                     |

### 法规列表字段

| 列名     | 说明                                           |
| -------- | ---------------------------------------------- |
| 法规名称 | 法规名称                                       |
| 法规类型 | 法律/行政法规/部门规章/司法解释/地方性法规     |
| 分类     | 法规分类                                       |
| 发布机关 | 发布法规的机关                                 |
| 发布日期 | 发布日期                                       |
| 实施日期 | 生效日期                                       |
| 效力状态 | 现行有效（绿色）/已修订（橙色）/已废止（红色） |

### 场景十一：新增法规

1. 点击「新增法规」按钮
2. 填写法规信息：

| 字段     | 必填 | 说明                                       |
| -------- | :--: | ------------------------------------------ |
| 法规名称 |  ✅  | 法规名称                                   |
| 法规类型 |  ✅  | 法律/行政法规/部门规章/司法解释/地方性法规 |
| 发布机关 |  -   | 发布法规的机关                             |
| 发布日期 |  -   | 发布日期                                   |
| 实施日期 |  -   | 生效日期                                   |
| 法规内容 |  -   | 法规全文内容                               |

3. 点击「确定」保存

::: tip 智能识别法规新增法规时支持 OCR 识别功能，上传法规截图可自动识别法规名称、发布机关、内容等信息。:::

### 场景十二：收藏法规

- 点击「收藏」将法规加入收藏夹
- 点击「取消收藏」移出收藏夹

### 场景十三：标注失效

当法规已废止时：

1. 点击「标注失效」按钮
2. 确认后法规状态变为「已废止」

### 场景十四：编辑和删除法规

- **编辑**：点击「编辑」修改法规信息
- **删除**：点击「删除」删除法规

## 常见问题

**Q: 可以下载案例文档吗？**

有权限的用户可以下载案例相关文档。

**Q: 法规库多久更新一次？**

法规库由管理员手动维护，可以随时添加新法规或标注失效法规。

**Q: 如何贡献知识文章？**

有写入权限的用户可以在知识文章模块创建新文章，保存为草稿后发布。

**Q: OCR 识别支持哪些格式？**

支持常见图片格式（JPG、PNG 等），上传裁判文书或法规截图即可自动识别。

**Q: 收藏的内容在哪里查看？**

收藏的文章、案例、法规可以在各自模块中通过筛选查看。
