# 知识库

知识库模块用于管理律所的知识文章、案例库、法规库等知识资源。

## 谁能使用？

所有用户都可以查看知识库内容。

| 功能 | 管理员 | 主任 | 团队负责人 | 财务 | 律师 | 行政 | 实习 |
|------|:-----:|:----:|:--------:|:----:|:----:|:----:|:----:|
| 知识文章 | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ |
| 案例库 | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ |
| 法规库 | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ |

---

## 知识文章

**菜单位置**：知识库 → 知识文章

管理和查阅律所内部的知识文章。

### 文章分类

- 法律实务
- 案例分析
- 行业动态
- 内部培训
- 其他

### 查看文章

1. 在文章列表中搜索或筛选
2. 点击文章标题查看详情

### 创建文章

1. 点击「新增文章」
2. 填写文章信息：

| 字段 | 必填 | 说明 |
|------|:----:|------|
| 标题 | ✅ | 文章标题 |
| 分类 | ✅ | 文章分类 |
| 内容 | ✅ | 文章正文 |
| 标签 | - | 关键词标签 |
| 是否公开 | ✅ | 是否对所有人可见 |

3. 保存或发布

---

## 案例库

**菜单位置**：知识库 → 案例库

查阅参考案例。

### 案例列表

| 列名 | 说明 |
|------|------|
| 案例名称 | 案例标题 |
| 案件类型 | 民事/刑事/行政等 |
| 案由 | 案件案由 |
| 判决结果 | 胜诉/败诉/和解等 |
| 承办律师 | 办理该案的律师 |

### 查看案例

1. 在案例列表中搜索
2. 点击案例名称查看详情
3. 查看案件概况、办理经过、裁判要点等

### 添加案例

有权限的用户可以添加新案例：

1. 点击「新增案例」
2. 填写案例信息
3. 上传相关文档
4. 保存

---

## 法规库

**菜单位置**：知识库 → 法规库

查询法律法规。

### 法规分类

- 宪法
- 法律
- 行政法规
- 司法解释
- 部门规章
- 地方性法规

### 搜索法规

1. 输入关键词搜索
2. 按分类筛选
3. 按时效性筛选（现行有效/已废止）

### 查看法规

点击法规名称查看全文内容。

## 常见问题

**Q: 可以下载案例文档吗？**

有权限的用户可以下载案例相关文档。

**Q: 法规库多久更新一次？**

法规库会定期更新，具体频率由管理员设定。

**Q: 如何贡献知识文章？**

有写入权限的用户可以在知识文章模块创建新文章。
