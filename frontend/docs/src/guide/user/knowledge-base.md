# 知识库

知识库是律所的经验宝库，收录了典型案例、知识文章和法律法规，帮助律师学习和参考。

## 谁能使用知识库？

| 功能 | ADMIN | DIRECTOR | PARTNER | 财务 | 主办律师 | 协办律师 | 行政 | 实习 |
|------|:-----:|:--------:|:-------:|:----:|:-------:|:-------:|:----:|:----:|
| 案例库 | ✅ | ✅ | ❌ | ✅ | ✅ | ✅ | ✅ | ✅ |
| 知识文章 | ✅ | ✅ | ❌ | ✅ | ✅ | ✅ | ✅ | ✅ |
| 法规库 | ✅ | ✅ | ❌ | ✅ | ✅ | ✅ | ✅ | ✅ |

::: tip 说明
知识库对大多数角色开放，包括实习律师。合伙人角色暂未配置知识库权限，如需访问请联系管理员。
:::

---

## 进入知识库

点击左侧菜单「知识库」，可以看到三个子菜单：
- 案例库：典型案例收录
- 知识文章：实务经验、文书模板等
- 法规库：法律法规查询

---

## 案例库

进入「知识库」→「案例库」

页面标题为「案例库」，描述为「管理典型案例」。

### 场景一：我想找类似的案例参考

办案时想看看有没有类似的案例可以参考：

1. 进入「案例库」
2. 使用筛选条件缩小范围：

| 筛选条件 | 说明 |
|---------|------|
| 案由类型 | 民事/刑事/行政/知识产权/劳动争议/商事 |
| 案例分类 | 从分类树中选择（支持多级分类） |
| 搜索 | 输入关键词搜索案例名称 |

3. 点击「查询」按钮搜索，或点击「重置」清空筛选条件
4. 找到感兴趣的案例，点击「查看」
5. 可以看到案例详情：
   - 案例名称和案由类型
   - 审理法院和判决日期
   - 案件结果（胜诉/部分胜诉/败诉/调解/撤诉）
   - 经办律师
   - 参考价值
   - 案例摘要（案情要点、裁判要旨）

**收藏案例：**
- 点击案例列表中的「收藏」链接可以收藏案例
- 已收藏的案例显示「取消收藏」链接
- 收藏的案例方便后续查找

::: tip 参考价值
案例会标注参考价值（高/中/低），用不同颜色标签显示：
- 🔴 高：红色标签，重点参考
- 🟠 中：橙色标签，一般参考
- 🔵 低：蓝色标签，了解即可
:::

---

### 场景二：我办了一个典型案例，想分享给大家

1. 进入「案例库」
2. 点击「添加案例」按钮
3. 填写案例信息：

| 字段 | 说明 |
|------|------|
| 案例名称 | 案例的名称（必填） |
| 案由类型 | 民事/刑事/行政/知识产权/劳动争议/商事（必填） |
| 案例分类 | 从分类树中选择 |
| 审理法院 | 审理案件的法院 |
| 判决日期 | 案件判决的日期 |
| 案件结果 | 胜诉/部分胜诉/败诉/调解/撤诉 |
| 经办律师 | 办理案件的律师 |
| 参考价值 | 高/中/低 |
| 案例摘要 | 案情要点、裁判要旨、办案心得 |

4. 点击「确定」保存

---

### 场景三：编辑或删除案例

**编辑案例：**
1. 在案例列表中找到要编辑的案例
2. 点击「编辑」链接
3. 修改案例信息
4. 点击「确定」保存

**删除案例：**
1. 点击「删除」链接
2. 在确认弹窗中点击「确定」

## 知识文章

进入「知识库」→「知识文章」

页面标题为「知识文章」，描述为「管理知识库文章」。

知识文章用于分享实务经验、文书模板、行业动态等内容。

### 场景四：我想查找学习资料

1. 进入「知识文章」页面
2. 使用筛选条件：

| 筛选条件 | 说明 |
|---------|------|
| 文章标题 | 输入关键词搜索 |
| 文章分类 | 法律法规/实务经验/文书模板/行业动态 |
| 状态 | 全部/草稿/已发布/已归档 |

3. 点击「查询」按钮搜索
4. 找到感兴趣的文章，点击「编辑」查看详情
5. 可以对文章进行点赞、收藏

---

### 场景五：我想发布一篇知识文章

1. 进入「知识文章」页面
2. 点击「发布文章」按钮
3. 填写文章信息：

| 字段 | 说明 |
|------|------|
| 文章标题 | 文章的标题（必填） |
| 文章分类 | 法律法规/实务经验/文书模板/行业动态（必填） |
| 文章内容 | 文章正文 |

4. 点击「确定」保存

::: info 文章状态
- **草稿**：保存后默认为草稿状态（橙色标签）
- **已发布**：点击「发布」后，文章对所有人可见（绿色标签）
- **已归档**：过时的文章可以归档（灰色标签）
:::

---

### 场景六：我想管理我的文章

**编辑文章：** 点击「编辑」修改文章内容

**发布文章：** 草稿状态的文章，点击「发布」让大家看到

**归档文章：** 已发布的文章，点击「归档」将其归档

**点赞文章：** 已发布的文章，点击「点赞」表示认可

**收藏/取消收藏：** 点击「收藏」或「取消收藏」管理你的收藏

**删除文章：** 点击「删除」删除文章（需确认）

---

## 法规库

进入「知识库」→「法规库」

页面标题为「法规库」，描述为「查询和管理法律法规」。

法规库收录了各类法律法规，方便律师查阅。

### 场景七：我想查找某条法规

页面左侧是法规分类树，右侧是法规列表。

**方法一：按分类查找**
1. 在左侧分类树中点击分类
2. 右侧会显示该分类下的法规

**方法二：搜索查找**
1. 在搜索框输入法规名称
2. 选择法规类型（法律/行政法规/部门规章/司法解释/地方性法规）
3. 选择效力状态（现行有效/已修订/已废止）
4. 点击「查询」

---

### 场景八：我想添加一条法规

1. 进入「法规库」页面
2. 点击「新增法规」按钮
3. 填写法规信息：

| 字段 | 说明 |
|------|------|
| 法规名称 | 法规的全称（必填） |
| 法规类型 | 法律/行政法规/部门规章/司法解释/地方性法规（必填） |
| 分类 | 从分类树中选择 |
| 发布机关 | 发布该法规的机关 |
| 发布日期 | 法规发布的日期 |
| 实施日期 | 法规开始实施的日期 |
| 法规内容 | 法规的正文内容 |

4. 点击「确定」保存

---

### 场景九：某条法规已经失效了，要标注

1. 找到该法规
2. 点击「标注失效」
3. 确认后法规状态变为「已废止」

---

### 场景十：收藏法规

1. 找到要收藏的法规
2. 点击「收藏」链接
3. 已收藏的法规显示「取消收藏」链接

---

## 法规类型说明

| 类型 | 说明 |
|------|------|
| 法律 | 全国人大及其常委会制定的法律 |
| 行政法规 | 国务院制定的行政法规 |
| 部门规章 | 国务院各部门制定的规章 |
| 司法解释 | 最高法、最高检发布的司法解释 |
| 地方性法规 | 地方人大制定的法规 |

| 已废止 | 法规已失效 | 红色 |

## 常见问题

**Q: 案例库里的案例是真实的吗？**
A: 是的，都是律所实际办理过的案件。但为保护当事人隐私，部分信息可能做了脱敏处理。

**Q: 我可以下载案例的判决书吗？**
A: 如果案例关联了文档（如判决书），可以在案例详情中查看和下载。

**Q: 为什么我添加不了案例/文章？**
A: 检查一下是否有相应权限。如果没有，请联系管理员。

**Q: 文章发布后还能修改吗？**
A: 可以。点击「编辑」修改后保存即可。

**Q: 怎么找到我收藏的文章/法规？**
A: 收藏的内容会显示「取消收藏」链接，可以通过收藏状态筛选。

**Q: 法规内容是最新的吗？**
A: 法规库需要人工维护，如发现法规已更新或失效，请联系管理员更新或使用「标注失效」功能。

**Q: 案由类型有哪些选项？**
A: 民事、刑事、行政、知识产权、劳动争议、商事。

**Q: 文章分类有哪些选项？**
A: 法律法规、实务经验、文书模板、行业动态。
