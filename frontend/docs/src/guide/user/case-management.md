# 项目管理

在系统中，「项目」就是你办理的案件。无论是诉讼案件还是非诉业务，都统称为「项目」。

## 谁能使用项目管理？

| 功能 | ADMIN | DIRECTOR | PARTNER | 财务 | 主办律师 | 协办律师 | 行政 | 实习 |
|------|:-----:|:--------:|:-------:|:----:|:-------:|:-------:|:----:|:----:|
| 项目列表 | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ❌ |
| 我的项目 | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ |
| 合同管理 | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ❌ |
| 工时管理 | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ |
| 任务管理 | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ |

::: warning 实习律师限制
实习律师只能访问「我的项目」，不能查看项目列表和合同管理。如需查看其他项目，请联系主办律师。
:::

::: tip 数据范围
- 主办律师、协办律师、实习律师：只能看到自己参与的项目
- 财务、行政：只能看到本部门的项目
- 合伙人：可以看到本部门及下级部门的项目
- 主任、管理员：可以看到全所项目
:::

## 项目类型说明

| 类型 | 说明 |
|------|------|
| 诉讼案件 | 民事、刑事、行政等诉讼业务 |
| 非诉项目 | 尽职调查、法律意见书等非诉讼业务 |
| 法律顾问 | 常年法律顾问服务 |
| 专项服务 | 其他专项法律服务 |

## 重要：先有合同，才能建项目

::: warning 请注意
系统要求你必须先创建合同，合同审批通过后，才能创建项目。这是为了确保每个案件都有正式的委托手续。
:::

**正确的接案流程**：

```
录入客户 → 创建合同 → 提交合同审批 → 审批通过 → 创建项目 → 开始办案
```

## 进入项目管理

点击左侧菜单「项目管理」→「项目列表」

## 场景一：我要接一个新案子

### 第一步：确保客户已录入

如果客户还没录入系统，先去 [录入客户](/guide/user/client-management)。

### 第二步：创建合同

1. 在项目列表页面，点击「创建合同」按钮
2. 填写合同信息（详见 [合同管理](/guide/user/contract-management)）
3. 提交合同审批

### 第三步：等待合同审批

合同创建后需要审批。你可以在「合同管理」中查看审批状态。

### 第四步：创建项目

合同审批通过后：

1. 点击「新增项目」按钮
2. **选择合同**（必选）：从下拉框选择已审批的合同
3. 系统会自动填充客户、收费方式等信息
4. 补充项目信息：
   - **项目名称**：如「张三诉李四合同纠纷案」
   - **项目类型**：诉讼案件/非诉项目/法律顾问/专项服务
   - **业务类型**：具体业务分类
   - **案件类型**（诉讼类）：民事/刑事/行政/劳动仲裁/商事仲裁/执行案件
   - **案由**（民事/行政案件）：系统提供级联选择，如「合同纠纷」→「买卖合同纠纷」
   - **审理阶段**：根据案件类型显示不同选项，支持多选
   - **对方当事人**：案件的对方
   - **主办律师**：负责这个案子的律师
   - **案源人**：介绍这个案子的人
5. 点击「确定」保存

::: tip 一个合同可以创建多个项目
比如一个法律顾问合同，可能会产生多个具体的服务项目。
:::

## 场景二：案件有多个委托人

系统支持添加多个委托人：

1. 在创建项目时，找到「代理客户」区域
2. 第一个客户默认是「主要」客户
3. 点击「添加共同委托人」添加更多客户
4. 为每个客户选择角色（原告/被告/第三人等）
5. 可以点击「设为主要」更换主要客户

## 场景四：查看项目详情

1. 在项目列表中找到项目
2. 点击「详情」按钮
3. 详情页顶部显示项目基本信息卡片，包含：
   - 项目编号、项目名称、项目类型、业务类型
   - 客户、状态、案源人、主办律师、部门
   - 收费类型、预估费用、实际费用
   - 立案日期、预计结案日期、实际结案日期
   - 标的金额、项目描述、备注等

4. 基本信息下方有多个标签页：
   - **卷宗文件**：项目文档管理，支持目录结构、上传、预览、下载（默认显示）
   - **证据整理**：项目证据管理，支持表格式和清单式两种编辑模式
   - **出函记录**：项目相关的出函申请，显示申请编号、函件类型、接收单位等
   - **期限提醒**：项目重要期限和提醒
   - **日程安排**：项目相关的日程安排，可以添加、取消、删除日程
   - **时间线**：项目重要事件记录，按时间顺序展示
   - **团队成员**：项目参与人员和角色，显示提成比例、是否案源等信息

5. 如果项目有对方当事人信息，会在标签页下方显示「对方当事人信息」卡片，包含：
   - 对方当事人名称
   - 对方律师姓名、执业证号、律师事务所
   - 对方律师联系电话、邮箱

6. 页面右上角有操作按钮：
   - **编辑**：编辑项目信息（已归档的项目不可编辑）
   - **申请结案**：为进行中的项目申请结案
   - **创建档案**：为已结案的项目创建档案

## 场景五：申请出函（介绍信）

在项目详情页的「出函记录」标签中，可以申请出函：

### 申请出函

1. 点击「申请出函」按钮
2. 填写出函信息：

| 字段 | 说明 |
|------|------|
| 函件模板 | 选择函件类型（必填） |
| 接收单位 | 函件发送的目标单位（必填） |
| 联系人 | 对方联系人 |
| 联系电话 | 对方电话 |
| 联系地址 | 对方地址 |
| 出函事由 | 出函的原因和目的（必填） |
| 出函律师 | 系统自动填充项目团队律师，可修改 |
| 审批人 | 选择审批人（必填） |
| 份数 | 需要打印的份数 |
| 期望日期 | 期望拿到函件的日期 |
| 备注 | 其他说明 |

3. 点击「确定」提交申请

### 出函状态说明

| 状态 | 颜色 | 含义 |
|------|------|------|
| 待审批 | 橙色 | 等待审批人审批 |
| 已审批 | 蓝色 | 审批通过，等待打印 |
| 已退回 | 金色 | 被退回修改 |
| 已拒绝 | 红色 | 审批被拒绝 |
| 已打印 | 青色 | 函件已打印，等待领取 |
| 已领取 | 绿色 | 律师已领取函件 |
| 已取消 | 灰色 | 申请已取消 |

## 场景六：管理项目期限提醒

在项目详情页，切换到「期限提醒」标签页，可以管理项目的重要期限。

### 添加期限提醒

1. 点击「添加期限」按钮
2. 填写期限信息：

| 字段 | 说明 |
|------|------|
| 期限类型 | 举证期限/上诉期限/答辩期限/开庭日期/其他 |
| 期限名称 | 如「一审举证期限」（必填） |
| 截止日期 | 期限的截止日期（必填） |
| 提前提醒 | 提前几天提醒（1/3/5/7/14天） |
| 说明 | 备注信息 |

3. 点击「确定」保存

### 自动创建常用期限

1. 点击「自动创建常用期限」按钮
2. 系统会根据项目类型自动创建常见的期限提醒
3. 你可以根据实际情况修改或删除

### 完成期限

当某个期限任务完成后：
1. 在期限列表中找到对应期限
2. 点击「完成」按钮
3. 期限状态变为「已完成」

### 删除期限

1. 点击「删除」按钮
2. 确认后删除期限

### 期限状态说明

| 状态 | 颜色 | 含义 |
|------|------|------|
| 剩余X天 | 蓝色 | 期限还有X天到期 |
| 今天到期 | 橙色 | 期限今天到期 |
| 已过期X天 | 红色 | 期限已过期X天 |
| 已完成 | 绿色 | 期限任务已完成 |

::: warning 期限提醒
系统会在期限到期前按设定的天数发送通知提醒，请及时处理重要期限，避免错过诉讼时效。
:::

## 场景七：记录工作时间

### 在工时管理页面记录

1. 进入「项目管理」→「工时管理」
2. 页面有两个标签页：「全部工时」和「我的工时」
3. 点击「新增工时」
4. 填写工时信息：

| 字段 | 说明 |
|------|------|
| 所属项目 | 选择项目（必填） |
| 工作日期 | 哪天做的工作（必填） |
| 工时(小时) | 工作了多长时间，支持0.1小时精度（必填） |
| 工作类型 | 法律研究/文书起草/客户沟通/出庭/会议/其他（必填） |
| 可计费 | 这个工时是否可以向客户收费 |
| 工作描述 | 具体做了什么（必填） |

5. 保存后状态为「草稿」
6. 点击「提交」进入审核流程

### 工时筛选

- 按项目筛选
- 按律师筛选（全部工时标签页）
- 按工作类型筛选
- 按状态筛选
- 按日期范围筛选

::: info 工时审核
工时提交后需要审核：
- **草稿**：可以编辑、删除、提交
- **待审核**：等待审核，可以通过或拒绝
- **已审核**：审核通过
- **已拒绝**：需要修改后重新提交
:::

## 场景八：管理项目任务

进入「项目管理」→「任务管理」

页面有多个标签页：全部任务、待处理、进行中、待验收、已完成

### 创建任务

1. 点击「新增任务」
2. 填写任务信息：

| 字段 | 说明 |
|------|------|
| 所属项目 | 选择项目（必填） |
| 任务标题 | 任务的名称（必填） |
| 负责人 | 谁来做这个任务（按部门筛选选择） |
| 优先级 | 低/中/高/紧急 |
| 截止日期 | 什么时候要完成 |
| 任务描述 | 详细说明 |

3. 点击「确定」保存

### 任务状态流转

| 状态 | 颜色 | 含义 | 下一步 |
|------|------|------|--------|
| 待处理 | 橙色 | 任务刚创建 | 开始处理 |
| 进行中 | 蓝色 | 正在处理 | 完成任务 |
| 待验收 | 紫色 | 任务完成，等待创建人验收 | 验收通过/退回 |
| 已完成 | 绿色 | 任务完成 | - |
| 已取消 | 灰色 | 任务取消 | - |

### 完成和验收任务

**完成任务：** 点击「完成」，任务状态变为「待验收」

**验收任务：** 任务创建人可以验收（只有创建人能看到验收按钮）：
- **通过**：任务状态变为「已完成」
- **退回**：需要填写退回意见，任务回到「进行中」

## 场景九：案件办完了，要结案

### 结案前检查

- [ ] 所有费用已收齐
- [ ] 所有文档已上传
- [ ] 工时记录已完成

### 结案步骤

1. 进入项目详情
2. 点击右上角「申请结案」按钮（仅进行中的项目显示）
3. 确认结案申请
4. 系统会提交结案申请给主任审批

结案审批通过后，项目状态变为「已结案」，后续可以创建档案。

## 场景十二：管理项目证据

在项目详情页的「证据整理」标签中，可以管理项目相关的证据：

::: warning 编辑权限
已结案或已归档的项目不能编辑证据，只能查看。
:::

### 证据编辑模式

系统支持两种编辑模式：

- **表格式**：表格形式编辑，适合快速录入
- **清单式**：分组清单形式，适合整理和导出

### 添加证据

1. 点击「添加证据」按钮
2. 填写证据信息（详见 [证据管理](/guide/user/evidence-management)）
3. 选择文件来源（从卷宗选择或直接上传）
4. 保存

### 证据分组

- 可以创建证据分组，如「合同类证据」「财务凭证」
- 将证据拖拽到分组中，方便管理

### 生成证据清单

1. 证据整理完成后，点击「生成清单」或「导出」
2. 选择清单格式（表格式/清单式）
3. 导出为Word或PDF

更多详细操作请参考 [证据管理](/guide/user/evidence-management) 章节。

## 常见问题

**Q: 为什么不能直接创建项目？**
A: 系统要求先有合同才能创建项目，请先创建并审批合同。

**Q: 合同审批要多久？**
A: 取决于审批流程，可在合同管理页面查看进度。

**Q: 项目创建后能修改吗？**
A: 结案前可以修改，结案后不能修改。

**Q: 项目编号怎么生成的？**
A: 系统自动生成，格式为 `M + 年份 + 序号`。

**Q: 期限提醒是怎么通知的？**
A: 系统会在期限到期前按设定的天数（如提前3天）发送系统通知，你可以在右上角的通知铃铛中看到提醒。

**Q: 期限过期了还能标记完成吗？**
A: 可以，即使期限已过期，你仍然可以点击「完成」标记为已完成。

**Q: 自动创建的期限可以删除吗？**
A: 可以，自动创建的期限和手动添加的期限一样，都可以编辑或删除。

**Q: 出函申请被退回怎么办？**
A: 查看退回原因，修改后重新提交。出函申请可以在项目详情的「出函记录」标签中查看和管理。

**Q: 任务验收按钮在哪里？**
A: 只有任务创建人才能看到验收按钮，且任务状态必须是「待验收」。
