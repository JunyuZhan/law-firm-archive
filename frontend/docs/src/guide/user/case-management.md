# 项目管理

在系统中，「项目」就是你办理的案件。无论是诉讼案件还是非诉业务，都统称为「项目」。

## 谁能使用项目管理？

| 功能 | 管理员 | 主任 | 团队负责人 | 财务 | 律师 | 行政 | 实习 |
|------|:-----:|:----:|:--------:|:----:|:----:|:----:|:----:|
| 项目列表 | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ❌ |
| 我的项目 | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ |
| 合同管理 | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ❌ |
| 工时管理 | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ |
| 任务管理 | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ |

::: warning 实习律师限制
实习律师只能访问「我的项目」，不能查看项目列表和合同管理。如需查看其他项目，请联系律师。
:::

::: tip 数据范围
- 律师、实习律师：只能看到自己参与的项目
- 财务、行政：只能看到本部门的项目
- 团队负责人：可以看到本团队及下级团队的项目
- 主任、管理员：可以看到全所项目
:::

## 项目类型说明

| 类型代码 | 类型名称 | 说明 |
|----------|----------|------|
| LITIGATION | 诉讼案件 | 民事、刑事、行政等诉讼业务 |
| NON_LITIGATION | 非诉项目 | 尽职调查、法律意见书等非诉讼业务 |
| LEGAL_COUNSEL | 法律顾问 | 常年法律顾问服务 |
| SPECIAL_SERVICE | 专项服务 | 其他专项法律服务 |

## 重要：先有合同，才能建项目

::: warning 请注意
系统要求你必须先创建合同，合同审批通过后，才能创建项目。这是为了确保每个案件都有正式的委托手续。
:::

**正确的接案流程**：

```
录入客户 → 创建合同 → 提交合同审批 → 审批通过 → 创建项目 → 开始办案
```

## 场景一：我要接一个新案子

### 第一步：确保客户已录入

如果客户还没录入系统，先去 [录入客户](/guide/user/client-management)。

### 第二步：创建合同

1. 在项目列表页面，点击「创建合同」按钮
2. 选择合同模板（必选）
3. 填写合同信息：
   - **客户**：选择委托人（必填）
   - **合同金额**：合同总金额（必填）
   - **签约日期**：默认今天
   - **收费方式**：固定收费/计时收费/风险代理/混合收费
   - **案件类型**（诉讼类）：民事/刑事/行政/劳动仲裁/商事仲裁/执行案件
   - **案由**（民事/行政案件）：级联选择
   - **审理阶段**：支持多选（一审/二审/再审/执行等）
   - **对方当事人**：案件的对方
   - **标的金额**：案件标的
   - **管辖法院**：受理法院
   - **案情摘要**：案件简要描述
   - **提成方案**：选择提成分配方案
   - **合同参与人**：添加承办律师、协办律师、案源人等
4. 点击「确定」保存合同
5. 在合同列表中找到合同，点击「提交审批」
6. 选择审批人，确认提交

### 第三步：等待合同审批

合同创建后需要审批。你可以在「合同管理」中查看审批状态。

### 第四步：创建项目

合同审批通过后：

1. 点击「新增项目」按钮
2. **选择合同**（必选）：从下拉框选择已审批的合同
3. 系统会自动填充客户、收费方式、案件类型、案由、对方当事人、标的金额、主办律师、案源人等信息
4. 补充项目信息：
   - **项目名称**：如「张三诉李四合同纠纷案」
   - **项目大类**：诉讼案件/非诉项目/法律顾问/专项服务
   - **案件类型**：具体案件分类
   - **案由**（民事/行政案件）：系统提供级联选择
   - **代理客户**：支持多个委托人，可设置主要客户和客户角色
   - **对方当事人**：案件的对方（多个用逗号分隔）
   - **主办律师**：负责这个案子的律师（按部门筛选选择）
   - **案源人**：介绍这个案子的人
5. 点击「确定」保存

::: tip 一个合同可以创建多个项目
比如一个法律顾问合同，可能会产生多个具体的服务项目。
:::

## 场景二：案件有多个委托人

系统支持添加多个委托人：

1. 在创建项目时，找到「代理客户」区域
2. 第一个客户默认是「主要」客户
3. 点击「添加共同委托人」添加更多客户
4. 为每个客户选择角色：
   - 原告（PLAINTIFF）
   - 被告（DEFENDANT）
   - 第三人（THIRD_PARTY）
   - 申请人（APPLICANT）
   - 被申请人（RESPONDENT）
5. 可以点击「设为主要」更换主要客户
6. 点击「创建新客户」可以快速跳转到客户创建页面

## 场景四：查看项目详情

1. 在项目列表中找到项目
2. 点击「详情」按钮
3. 详情页顶部显示项目基本信息卡片，包含：
   - 项目编号、项目名称、项目类型、业务类型
   - 客户、状态、案源人、主办律师、部门
   - 收费类型、预估费用、实际费用
   - 立案日期、预计结案日期、实际结案日期
   - 标的金额、创建时间、项目描述、备注等

4. 基本信息下方有多个标签页：
   - **卷宗文件**：项目文档管理，支持目录结构、上传、预览、下载（默认显示）
   - **证据整理**：项目证据管理，支持表格式和清单式两种编辑模式
   - **出函记录**：项目相关的出函申请，显示申请编号、函件类型、接收单位等
   - **期限提醒**：项目重要期限和提醒
   - **日程安排**：项目相关的日程安排，可以添加、取消、删除日程
   - **时间线**：项目重要事件记录，按时间顺序展示
   - **团队成员**：项目参与人员和角色，显示提成比例、是否案源等信息

5. 如果项目有对方当事人信息，会在标签页下方显示「对方当事人信息」卡片，包含：
   - 对方当事人名称
   - 对方律师姓名、执业证号、律师事务所
   - 对方律师联系电话、邮箱

6. 页面右上角有操作按钮：
   - **编辑**：编辑项目信息（待审批结案/已结案/已归档的项目不可编辑）
   - **申请结案**：为进行中（ACTIVE）的项目申请结案
   - **创建档案**：为已结案（CLOSED）的项目创建档案

## 场景六：管理项目期限提醒

在项目详情页，切换到「期限提醒」标签页，可以管理项目的重要期限。

::: warning 操作限制
待审批结案（PENDING_CLOSE）、已结案（CLOSED）、已归档（ARCHIVED）状态的项目不能新增期限。
:::

### 添加期限提醒

1. 点击「添加期限」按钮
2. 填写期限信息：

| 字段 | 说明 |
|------|------|
| 期限类型 | 举证期限/上诉期限/答辩期限/开庭日期/其他 |
| 期限名称 | 如「一审举证期限」（必填） |
| 截止日期 | 期限的截止日期（必填） |
| 提前提醒 | 提前几天提醒（默认3天） |
| 说明 | 备注信息 |

3. 点击「确定」保存

### 自动创建常用期限

1. 点击「自动创建常用期限」按钮
2. 系统会根据项目类型自动创建常见的期限提醒
3. 你可以根据实际情况修改或删除

### 完成期限

当某个期限任务完成后：
1. 在期限列表中找到对应期限
2. 点击「完成」按钮（仅待处理PENDING状态显示）
3. 期限状态变为「已完成」

### 删除期限

1. 点击「删除」按钮
2. 确认后删除期限

### 期限状态说明

| 状态 | 颜色 | 含义 |
|------|------|------|
| 剩余X天 | 蓝色 | 期限还有X天到期 |
| 今天到期 | 橙色 | 期限今天到期 |
| 已过期X天 | 红色 | 期限已过期X天 |
| 已完成 | 绿色 | 期限任务已完成 |

::: warning 期限提醒
系统会在期限到期前按设定的天数发送通知提醒，请及时处理重要期限，避免错过诉讼时效。
:::

## 场景七：记录工作时间

### 在工时管理页面记录

1. 进入「项目管理」→「工时管理」
2. 页面有两个标签页：「全部工时」和「我的工时」
3. 点击「新增工时」
4. 填写工时信息：

| 字段 | 说明 |
|------|------|
| 所属项目 | 选择项目（必填，支持搜索） |
| 工作日期 | 哪天做的工作（必填） |
| 工时(小时) | 工作了多长时间，支持0.1小时精度，最大24小时（必填） |
| 工作类型 | 法律研究/文书起草/客户沟通/出庭/会议/其他（必填） |
| 可计费 | 这个工时是否可以向客户收费（开关） |
| 工作描述 | 具体做了什么（必填） |

5. 保存后状态为「草稿」
6. 点击「提交」进入审核流程

### 工时筛选

- 按项目筛选（支持搜索）
- 按律师筛选（全部工时标签页）
- 按工作类型筛选
- 按状态筛选
- 按开始日期筛选
- 按结束日期筛选

### 工时状态说明

| 状态代码 | 状态名称 | 颜色 | 说明 |
|----------|----------|------|------|
| DRAFT | 草稿 | 灰色 | 可以编辑、删除、提交 |
| PENDING | 待审核 | 橙色 | 等待审核，可以通过或拒绝 |
| APPROVED | 已审核 | 绿色 | 审核通过 |
| REJECTED | 已拒绝 | 红色 | 需要修改后重新提交 |

### 工时操作

- **编辑**：修改工时记录
- **提交**：草稿状态可提交审核
- **通过**：待审核状态可通过
- **拒绝**：待审核状态可拒绝（需填写拒绝原因）
- **删除**：删除工时记录

## 场景八：管理项目任务

进入「项目管理」→「任务管理」

页面有多个标签页：全部任务、待处理、进行中、待验收、已完成

### 创建任务

1. 点击「新增任务」
2. 填写任务信息：

| 字段 | 说明 |
|------|------|
| 所属项目 | 选择项目（必填，支持搜索） |
| 任务标题 | 任务的名称（必填） |
| 负责人 | 谁来做这个任务（按部门树选择） |
| 优先级 | 低(LOW)/中(MEDIUM)/高(HIGH)/紧急(URGENT) |
| 截止日期 | 什么时候要完成 |
| 任务描述 | 详细说明 |

3. 点击「确定」保存

### 任务筛选

- 按项目筛选（支持搜索）
- 按任务状态筛选
- 按优先级筛选

### 任务状态流转

| 状态代码 | 状态名称 | 颜色 | 含义 | 下一步 |
|----------|----------|------|------|--------|
| TODO | 待处理 | 橙色 | 任务刚创建 | 完成任务 |
| IN_PROGRESS | 进行中 | 蓝色 | 正在处理 | 完成任务 |
| PENDING_REVIEW | 待验收 | 紫色 | 任务完成，等待创建人验收 | 验收通过/退回 |
| COMPLETED | 已完成 | 绿色 | 任务完成 | - |
| CANCELLED | 已取消 | 灰色 | 任务取消 | - |

### 完成和验收任务

**完成任务：** 点击「完成」，任务状态变为「待验收」

**验收任务：** 任务创建人可以验收（只有创建人能看到验收按钮）：
- **通过**：任务状态变为「已完成」
- **退回**：需要填写退回意见，任务回到「进行中」

### 任务操作

- **编辑**：修改任务信息
- **完成**：将任务标记为完成（待处理/进行中状态）
- **验收**：验收任务（待验收状态，仅创建人可见）
- **删除**：删除任务

## 场景九：案件办完了，要结案

### 结案前检查

- [ ] 所有费用已收齐
- [ ] 所有文档已上传
- [ ] 工时记录已完成

### 结案步骤

1. 进入项目详情
2. 点击右上角「申请结案」按钮（仅进行中ACTIVE的项目显示）
3. 填写结案信息：
   - **结案日期**：选择结案日期（必填）
   - **结案原因**：胜诉/败诉/和解/撤诉/项目完成/其他（必填）
   - **结案结果**：案件结果描述
   - **结案总结**：案件总结
   - **审批人**：选择审批人（必填，系统会推荐团队负责人）
4. 点击「确定」提交结案申请

结案审批通过后，项目状态变为「已结案」（CLOSED），后续可以创建档案。

### 结案原因选项

| 代码 | 名称 | 说明 |
|------|------|------|
| WIN | 胜诉 | 诉讼案件胜诉 |
| LOSE | 败诉 | 诉讼案件败诉 |
| SETTLEMENT | 和解 | 双方和解 |
| WITHDRAWAL | 撤诉 | 撤回起诉 |
| COMPLETED | 项目完成 | 非诉项目完成 |
| OTHER | 其他 | 其他原因 |

## 场景十一：暂停或恢复项目

**恢复**：编辑项目，将状态改回「进行中」（ACTIVE）

## 场景十二：管理项目证据

在项目详情页的「证据整理」标签中，可以管理项目相关的证据：

::: warning 编辑权限
已归档（ARCHIVED）的项目不能编辑证据，只能查看。已结案（CLOSED）的项目仍可编辑，因为可能需要补充文档以符合入库条件。
:::

### 证据编辑模式

系统支持两种编辑模式：

- **表格式**：表格形式编辑，适合快速录入
- **清单式**：分组清单形式，适合整理和导出

### 添加证据

1. 点击「添加证据」按钮
2. 填写证据信息（详见 [证据管理](/guide/user/evidence-management)）
3. 选择文件来源（从卷宗选择或直接上传）
4. 保存

### 证据分组

- 可以创建证据分组，如「合同类证据」「财务凭证」
- 将证据拖拽到分组中，方便管理

### 生成证据清单

1. 证据整理完成后，点击「生成清单」或「导出」
2. 选择清单格式（表格式/清单式）
3. 导出为Word或PDF

更多详细操作请参考 [证据管理](/guide/user/evidence-management) 章节。

## 场景十三：管理项目日程

在项目详情页的「日程安排」标签中，可以管理项目相关的日程：

::: warning 操作限制
待审批结案（PENDING_CLOSE）、已结案（CLOSED）、已归档（ARCHIVED）状态的项目不能新增日程。
:::

### 添加日程

1. 点击「添加日程」按钮
2. 填写日程信息：

| 字段 | 说明 |
|------|------|
| 日程标题 | 如「XX案开庭」（必填） |
| 日程类型 | 开庭(COURT)/会议(MEETING)/期限(DEADLINE)/约见(APPOINTMENT)/其他(OTHER)（必填） |
| 日期 | 日程日期（必填，默认今天） |

## 常见问题

**Q: 为什么不能直接创建项目？**
A: 系统要求先有合同才能创建项目，请先创建并审批合同。

**Q: 合同审批要多久？**
A: 取决于审批流程，可在合同管理页面查看进度。

**Q: 项目创建后能修改吗？**
A: 待审批结案、已结案、已归档状态不能修改，其他状态可以修改。

**Q: 项目编号怎么生成的？**
A: 系统自动生成，格式为 `M + 年份 + 序号`。

**Q: 期限提醒是怎么通知的？**
A: 系统会在期限到期前按设定的天数（如提前3天）发送系统通知，你可以在右上角的通知铃铛中看到提醒。

**Q: 期限过期了还能标记完成吗？**
A: 可以，即使期限已过期，你仍然可以点击「完成」标记为已完成。

**Q: 自动创建的期限可以删除吗？**
A: 可以，自动创建的期限和手动添加的期限一样，都可以编辑或删除。

**Q: 出函申请被退回怎么办？**
A: 查看退回原因，修改后重新提交。出函申请可以在项目详情的「出函记录」标签中查看和管理。

**Q: 任务验收按钮在哪里？**
A: 只有任务创建人才能看到验收按钮，且任务状态必须是「待验收」。

**Q: 已结案的项目还能编辑文档吗？**
A: 可以。已结案（CLOSED）的项目仍可编辑卷宗文件和证据，因为可能需要补充文档以符合入库条件。只有已归档（ARCHIVED）的项目才完全禁止编辑。

## 合同管理详解

合同管理是项目管理的前置环节，所有项目都必须基于已审批的合同创建。

### 进入合同管理

点击左侧菜单「项目管理」→「合同管理」

### 合同列表筛选

| 筛选项 | 说明 |
|--------|------|
| 创建年份 | 默认当前年份，可选最近5年或全部年份 |
| 合同编号 | 按合同编号搜索 |
| 合同名称 | 按合同名称搜索 |
| 客户 | 选择客户筛选 |
| 合同类型 | 服务合同/常年法顾/诉讼代理/非诉项目 |
| 收费方式 | 固定收费/计时收费/风险代理/混合收费 |
| 状态 | 草稿/待审批/生效中/已拒绝/已终止/已完成 |
| 签约人 | 选择签约人筛选 |

### 创建合同

1. 点击「创建合同」按钮
2. **选择合同模板**（必选）：系统提供多种合同模板
3. 填写合同信息：

**基本信息：**
| 字段 | 说明 |
|------|------|
| 客户 | 选择委托人（必填） |
| 合同金额 | 合同总金额（必填），系统自动转换大写 |
| 签约日期 | 默认今天 |
| 生效日期 | 合同生效日期 |
| 到期日期 | 合同到期日期 |
| 签约人 | 选择签约律师 |
| 所属部门 | 选择部门 |
| 付款条款 | 付款方式说明 |
| 备注 | 其他备注 |

**诉讼类合同扩展字段：**
| 字段 | 说明 |
|------|------|
| 案件类型 | 民事/刑事/行政/劳动仲裁/商事仲裁/执行案件 |
| 案由 | 级联选择（民事/行政案件） |
| 审理阶段 | 支持多选（一审/二审/再审/执行等） |
| 对方当事人 | 案件对方 |
| 标的金额 | 案件标的，系统自动转换大写 |
| 管辖法院 | 受理法院 |
| 案情摘要 | 案件简要描述 |
| 利冲审查状态 | 待审查/已通过/未通过/无需审查 |
| 预收差旅费 | 预收的差旅费用 |
| 风险代理比例 | 风险代理的提成比例 |

**提成分配：**
| 字段 | 说明 |
|------|------|
| 提成方案 | 选择预设的提成分配方案 |
| 律所比例 | 律所提成比例 |
| 主办律师比例 | 主办律师提成比例 |
| 协办律师比例 | 协办律师提成比例 |
| 辅助人员比例 | 辅助人员提成比例 |

- 点击「添加参与人」添加合同参与人
- 选择人员、角色（承办律师/协办律师/案源人/律师助理）

|----------|----------|------|

| LITIGATION | 诉讼代理 | 诉讼案件代理合同 |

| 状态代码 | 状态名称 | 颜色 | 说明 |
|----------|----------|------|------|

| 提交审批 | 提交合同审批 | 草稿 |

| 变更申请 | 申请合同变更 | 生效中 |

### 合同详情

合同详情页包含多个标签页：

1. **基本信息**：合同基本信息展示
2. **付款计划**：管理合同付款计划
   - 添加付款阶段（阶段名称、金额、比例、计划日期）
   - 查看付款状态
3. **参与人**：管理合同参与人
   - 添加/编辑/删除参与人
   - 设置提成比例

### 打印合同

合同审批通过后，可以打印：

1. 在合同详情页点击「打印」按钮
2. 选择打印内容：
   - 打印合同：打印正式合同文本
   - 打印收案审批表：打印收案审批表
3. 点击「确定打印」
4. 系统打开打印预览窗口

### 编辑收案审批表

1. 在合同详情页点击「编辑审批表」按钮
2. 编辑案情摘要内容
3. 点击「保存」

### 预览收案审批表

1. 在合同详情页点击「预览审批表」按钮
2. 系统打开预览窗口
3. 可以直接打印
