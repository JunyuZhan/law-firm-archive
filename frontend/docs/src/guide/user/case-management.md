# 项目管理

项目管理模块用于管理律所的业务项目（案件），包括项目创建、合同管理、任务分配、工时记录等。

## 谁能使用？

| 功能 | 管理员 | 主任 | 团队负责人 | 财务 | 律师 | 行政 | 实习 |
|------|:-----:|:----:|:--------:|:----:|:----:|:----:|:----:|
| 项目列表 | ✅ | ✅ | ✅ | ✅(查看) | ✅ | ❌ | ❌ |
| 我的项目 | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ |
| 合同管理 | ✅ | ✅ | ✅ | ✅ | ✅ | ❌ | ❌ |
| 任务管理 | ✅ | ✅ | ✅ | ❌ | ✅ | ❌ | ✅ |
| 工时管理 | ✅ | ✅ | ✅ | ✅ | ✅ | ❌ | ✅ |

## 项目列表

**菜单位置**：项目管理 → 项目列表

查看所有有权限访问的项目。

### 查询筛选

| 筛选项 | 说明 |
|--------|------|
| 年份 | 按创建年份筛选 |
| 项目名称 | 模糊搜索 |
| 项目类型 | 诉讼/非诉/顾问 |
| 状态 | 进行中/已完成/已归档/已取消 |
| 主办律师 | 筛选主办律师 |

### 项目列表字段

| 列名 | 说明 |
|------|------|
| 项目编号 | 系统自动生成 |
| 项目名称 | 项目/案件名称 |
| 项目类型 | 诉讼/非诉/顾问 |
| 案件类型 | 民事/刑事/行政等 |
| 客户名称 | 委托客户 |
| 主办律师 | 负责律师 |
| 状态 | 当前状态 |
| 收费方式 | 计时/固定/风险/混合 |
| 创建时间 | 项目创建时间 |

### 新增项目

#### 方式一：直接创建

1. 点击「新增项目」
2. 填写项目信息：

**基本信息**：

| 字段 | 必填 | 说明 |
|------|:----:|------|
| 项目名称 | ✅ | 如「XX公司诉XX公司合同纠纷案」 |
| 项目类型 | ✅ | 诉讼/非诉/顾问 |
| 案件类型 | ✅ | 根据项目类型自动筛选选项 |
| 案由 | - | 诉讼案件需选择案由 |

**当事人信息**：

| 字段 | 必填 | 说明 |
|------|:----:|------|
| 委托客户 | ✅ | 选择已有客户或新建 |
| 客户角色 | ✅ | 原告/被告/第三人/申请人/被申请人 |
| 对方当事人 | - | 对方当事人名称 |
| 对方律师 | - | 对方代理律师信息 |

**办理信息**：

| 字段 | 必填 | 说明 |
|------|:----:|------|
| 案源人 | - | 案源引入人 |
| 主办律师 | ✅ | 负责该项目的律师 |
| 所属部门 | - | 归属部门 |
| 收费方式 | ✅ | 计时/固定/风险/混合 |
| 预估费用 | - | 预估律师费 |
| 诉讼金额 | - | 诉讼标的金额 |

3. 点击「确定」保存

#### 方式二：从合同创建

1. 点击「从合同创建」
2. 选择已审批通过的合同
3. 系统自动填充相关信息
4. 补充其他信息后保存

### 查看项目详情

点击项目名称或「详情」按钮，进入项目详情页，包含：

- **基本信息**：项目概况
- **团队成员**：参与项目的律师和助理
- **任务**：项目任务列表
- **工时**：工时记录
- **文档**：项目文档
- **收款**：收款记录
- **出函**：律师函申请

### 项目状态

| 状态 | 说明 |
|------|------|
| ACTIVE | 进行中 |
| COMPLETED | 已完成 |
| ARCHIVED | 已归档 |
| CANCELLED | 已取消 |

## 合同管理

**菜单位置**：项目管理 → 合同管理

管理业务合同（委托代理合同等）。

### 合同列表字段

| 列名 | 说明 |
|------|------|
| 合同编号 | 系统自动生成 |
| 合同名称 | 合同标题 |
| 客户名称 | 签约客户 |
| 合同类型 | 服务合同/顾问合同等 |
| 合同金额 | 合同总金额 |
| 收费方式 | 固定/计时/风险/混合 |
| 签约日期 | 合同签订日期 |
| 状态 | 草稿/待审批/已审批/已拒绝/已作废 |

### 创建合同

1. 点击「新增合同」
2. 填写合同信息：

| 字段 | 必填 | 说明 |
|------|:----:|------|
| 合同名称 | ✅ | 合同标题 |
| 客户 | ✅ | 选择签约客户 |
| 合同类型 | ✅ | 服务/顾问/其他 |
| 收费方式 | ✅ | 固定/计时/风险/混合 |
| 合同金额 | ✅ | 合同总金额 |
| 签约日期 | ✅ | 签订日期 |
| 生效日期 | - | 合同生效日期 |
| 到期日期 | - | 合同到期日期 |
| 签约人 | - | 负责签约的律师 |
| 备注 | - | 其他说明 |

3. 点击「确定」保存
4. 保存后合同进入「待审批」状态

### 合同审批

合同需要由主任或团队负责人审批：
1. 审批人在「审批中心」查看待审批合同
2. 点击「通过」或「拒绝」

### 合同状态

| 状态 | 说明 |
|------|------|
| DRAFT | 草稿 |
| PENDING | 待审批 |
| APPROVED | 已审批 |
| REJECTED | 已拒绝 |
| VOIDED | 已作废 |

## 任务管理

| 列名 | 说明 |
|------|------|
| 任务标题 | 任务名称 |
| 所属项目 | 关联的项目 |

4. 保存

|------|------|

## 工时管理

**菜单位置**：项目管理 → 工时管理

记录项目的工作时间。

### 工时列表字段

| 列名 | 说明 |
|------|------|
| 日期 | 工作日期 |
| 项目 | 关联项目 |
| 工作内容 | 工作描述 |
| 工时 | 工作时长（小时） |
| 状态 | 待审批/已审批/已拒绝 |

### 记录工时

1. 点击「新增工时」
2. 选择项目
3. 填写日期、工作内容、工时
4. 保存

### 工时审批

工时可以由项目负责人或主任审批。

## 常见问题

**Q: 项目编号是怎么生成的？**

项目编号由系统自动生成，格式根据系统配置，通常包含年份和序号。

**Q: 一个项目可以有多个客户吗？**

可以，支持添加多个委托人，每个委托人可以设置不同的角色（原告、被告等）。

**Q: 合同审批被拒绝后怎么办？**

可以修改合同信息后重新提交审批。

**Q: 工时记录错误可以修改吗？**

未审批的工时可以修改或删除，已审批的工时需要联系管理员处理。
