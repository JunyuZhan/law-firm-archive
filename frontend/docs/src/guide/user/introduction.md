# 欢迎使用智慧律所管理系统

这是一套专为律师事务所设计的业务管理系统。无论你是律师、财务还是行政人员，这份手册都会帮助你快速上手。

---

## 我是谁，我能做什么？

系统会根据你的角色显示不同的菜单。系统共有9个角色：

| 角色 | 代码 | 数据范围 | 定位 |
|------|------|----------|------|
| 超级管理员 | ADMIN | 全所 | 系统技术管理 |
| 律所主任 | DIRECTOR | 全所 | 业务最高管理 |
| 合伙人 | PARTNER | 部门及下级 | 部门业务管理 |
| 财务主管 | FINANCE_MANAGER | 全所 | 财务管理负责人 |
| 财务人员 | FINANCE | 本部门 | 日常财务操作 |
| 主办律师 | LEAD_LAWYER | 自己 | 案件主办人 |
| 协办律师 | ASSOCIATE | 自己 | 案件协办人 |
| 行政人员 | ADMIN_STAFF | 本部门 | 行政后勤管理 |
| 实习律师 | TRAINEE | 自己 | 实习人员 |

### 如果你是主办律师 / 协办律师

你主要会用到：
- **客户管理**：录入和管理客户信息、利冲检索
- **项目管理**：管理自己负责的案件、工时记录、任务管理
- **合同管理**：创建和查看合同
- **文档管理**：上传和管理案件文档、用印申请
- **档案管理**：查看档案、申请借阅
- **我的收款**：查看自己案件的收款情况
- **我的提成**：查看自己的提成明细
- **我的工资**：查看和确认每月工资明细
- **费用报销**：提交报销申请
- **知识库**：查阅案例库、知识文章、法规库

::: warning 实习律师限制
实习律师不能访问：项目列表（只能看我的项目）、合同管理、案源管理、用印申请、档案管理
:::

### 如果你是财务人员 / 财务主管

你会用到：
- **客户管理**：查看客户信息
- **项目管理**：查看项目信息
- **合同管理**：查看合同信息
- **收款管理**：登记客户付款、查看收款明细
- **提成管理**：计算提成、审批提成、发放提成
- **发票管理**：管理发票开具
- **财务报表**：查看财务统计
- **薪资管理**：管理全所员工工资表、编辑扣减项、发放工资
- **文档管理**：查看文档、用印申请
- **档案管理**：查看档案、申请借阅
- **知识库**：查阅案例库

::: tip 财务主管特权
财务主管可以查看全所财务数据，普通财务人员只能查看本部门数据。
:::

### 如果你是行政人员

你会用到：
- **客户管理**：查看客户信息
- **项目管理**：查看项目信息
- **合同管理**：查看合同信息
- **行政管理**：考勤、请假、会议室、资产、采购、供应商、加班、外出、会议记录、出函管理
- **文档管理**：查看文档、用印申请
- **档案管理**：查看档案、申请借阅
- **我的收款/提成**：查看自己的收款和提成
- **费用报销**：提交报销申请
- **知识库**：查阅案例库

### 如果你是律所主任 / 合伙人

你可以：
- **审批中心**：审批合同、收款变更等
- **查看全所/部门业务数据**：项目、收入、提成等
- **财务管理**：查看财务报表、收款管理、提成管理
- **人力资源**：晋升管理、发展计划、转正管理、离职管理
- **出函管理**：管理律师函等出函
- **数据交接**：管理离职交接、项目移交、客户移交

::: tip 主任特权
律所主任可以管理提成规则、查看操作日志、管理出函模板和合同模板。
:::

### 如果你是超级管理员

你可以：
- **用户管理**：创建、修改、禁用账号
- **角色管理**：配置角色权限
- **部门管理**：设置部门和层级
- **菜单管理**：配置系统菜单
- **系统配置**：配置系统参数
- **操作日志**：查看系统操作记录
- **模板管理**：出函模板、合同模板
- **数据交接**：管理离职交接、项目移交、客户移交

::: warning 管理员定位
超级管理员是系统技术管理角色，不参与业务审批流程。业务审批由律所主任和合伙人负责。
:::

## 系统的基本操作流程

### 接案流程

```
1. 录入客户 → 2. 利冲检索 → 3. 创建合同 → 4. 合同审批 → 5. 创建项目 → 6. 开始办案
```

### 办案流程

```
1. 记录工时 → 2. 上传文档 → 3. 管理任务 → 4. 申请结案 → 5. 结案审批 → 6. 档案归档
```

### 收款流程

```
1. 客户付款 → 2. 财务登记收款 → 3. 系统计算提成 → 4. 提成审批 → 5. 提成发放
```

### 工资发放流程

```
1. 财务编辑扣减项 → 2. 提交工资表 → 3. 员工确认 → 4. 财务确认 → 5. 主任审批 → 6. 发放工资
```

## 界面介绍

登录后，你会看到系统的主界面：

### 顶部导航栏

- **左侧**：系统 Logo 和名称
- **右侧**：消息通知、个人头像、退出登录

### 左侧菜单栏

显示你有权限访问的功能模块。点击菜单项可以进入对应功能。

### 主内容区

显示当前选中菜单的具体内容。

### 常用操作

- **搜索**：大多数列表页面都有搜索框，可以快速查找
- **筛选**：使用筛选条件缩小显示范围
- **新增**：点击「新增」或「创建」按钮添加新记录
- **编辑**：点击「编辑」按钮修改记录
- **删除**：点击「删除」按钮删除记录（谨慎操作）
- **详情**：点击「详情」或记录名称查看详细信息
