# 欢迎使用智慧律所管理系统

这是一套专为律师事务所设计的业务管理系统。无论你是律师、财务还是行政人员，这份手册都会帮助你快速上手。

---

## 系统角色

系统根据你的角色显示不同的功能菜单。共有 7 个角色：

| 角色 | 数据范围 | 定位 |
|------|----------|------|
| 管理员 | 全所 | 系统配置、用户管理 |
| 主任 | 全所 | 业务最高管理、审批 |
| 团队负责人 | 部门及下级 | 部门业务管理、审批 |
| 财务 | 全所 | 收款、提成、薪资管理 |
| 律师 | 自己相关 | 案件办理 |
| 行政 | 全所 | 行政后勤管理 |
| 实习律师 | 自己相关 | 实习人员，有限权限 |

---

## 功能模块概览

### 工作台
- **审批中心**：处理各类业务审批（主任/团队负责人/管理员）
- **日程管理**：管理开庭、会议、约见等日程
- **报表中心**：查看业务数据统计

### 客户管理
- **客户列表**：录入和管理客户信息
- **利冲审查**：利益冲突检索
- **案源管理**：管理案源线索

### 项目管理
- **项目列表**：所有项目列表（有权限查看）
- **我的项目**：自己参与的项目
- **合同管理**：管理业务合同
- **我的合同**：自己相关的合同
- **任务管理**：项目任务分配和跟踪
- **工时管理**：工时记录和统计

### 财务管理
- **收款管理**：登记和管理收款
- **我的收款**：查看自己的收款
- **提成管理**：提成计算和发放
- **我的提成**：查看自己的提成
- **发票管理**：管理发票开具
- **费用报销**：提交和审批报销
- **财务报表**：查看财务统计

### 卷宗管理
- **文档列表**：项目文档管理
- **我的文档**：自己的文档
- **用印管理**：印章使用管理
- **用印申请**：申请用印
- **文档模板**：管理文档模板
- **AI 撰写**：AI 辅助文档撰写

### 档案管理
- **档案列表**：查看档案
- **档案借阅**：申请借阅档案
- **档案迁移**：档案销毁/迁移管理

### 行政管理
- **考勤管理**：考勤记录
- **请假管理**：请假申请和审批
- **会议室**：会议室预约
- **资产管理**：固定资产管理
- **采购管理**：采购申请
- **供应商**：供应商管理
- **加班管理**：加班记录
- **外出管理**：外出申请
- **会议记录**：会议纪要
- **出函管理**：律师函管理

### 人力资源
- **员工管理**：员工信息管理
- **薪资管理**：工资表管理
- **我的工资**：查看自己的工资
- **发展计划**：员工发展规划
- **晋升管理**：晋升申请和审批
- **转正管理**：转正申请
- **离职管理**：离职流程

### 知识库
- **知识文章**：内部知识文章
- **案例库**：参考案例
- **法规库**：法律法规查询

### 系统管理（管理员）
- **用户管理**：管理用户账号
- **角色管理**：配置角色权限
- **部门管理**：设置部门结构
- **菜单管理**：配置系统菜单
- **系统配置**：系统参数配置
- **数据字典**：管理数据字典
- **操作日志**：查看操作记录
- **出函模板**：管理出函模板
- **合同模板**：管理合同模板

## 基本操作流程

### 接案流程
```
录入客户 → 利冲检索 → 创建合同 → 合同审批 → 创建项目 → 开始办案
```

### 办案流程
```
记录工时 → 上传文档 → 管理任务 → 申请结案 → 结案审批 → 档案归档
```

### 收款流程
```
客户付款 → 财务登记收款 → 系统计算提成 → 提成审批 → 提成发放
```

## 界面介绍

### 顶部导航栏
- **左侧**：系统 Logo 和名称
- **右侧**：消息通知（铃铛图标）、个人头像

### 左侧菜单栏
显示你有权限访问的功能模块。点击菜单项进入对应功能。

### 常用操作
- **搜索**：大多数列表页面都有搜索框
- **新增**：点击「新增」按钮添加新记录
- **编辑**：点击「编辑」按钮修改记录
- **删除**：点击「删除」按钮删除记录
- **详情**：点击记录名称或「详情」查看详细信息
