# 合同管理

合同是律所与客户之间的正式委托协议。在系统中，合同是创建项目的前提条件。

## 谁能使用合同管理？

| 功能 | 管理员 | 主任 | 团队负责人 | 财务 | 律师 | 行政 | 实习 |
|------|:-----:|:----:|:--------:|:----:|:----:|:----:|:----:|
| 合同管理 | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ❌ |
| 合同审批 | ✅ | ✅ | ✅ | ❌ | ❌ | ❌ | ❌ |

::: warning 实习律师限制
实习律师不能访问合同管理。如需创建合同，请联系律师。
:::

::: tip 审批权限
合同审批由管理员、主任和团队负责人负责。
:::

---

## 进入合同管理

点击左侧菜单「项目管理」→「合同管理」

页面顶部显示统计卡片（合同总数、生效中、待审批、本月新增），下方是合同列表。

支持按年份筛选（默认显示当前年份的合同）。

---

## 场景一：创建新合同

### 第一步：点击创建

在合同列表页面，点击「创建合同」按钮。

### 第二步：选择合同模板（推荐）

系统提供了多种合同模板：
- 服务合同
- 常年法律顾问合同
- 诉讼代理合同（民事/刑事）
- 非诉项目合同

选择模板后，系统会自动填充合同类型和收费方式，并根据模板类型显示对应的字段。

### 第三步：填写合同基本信息

| 字段 | 说明 | 示例 |
|------|------|------|
| 合同名称 | 合同的名称 | 「张三诉李四合同纠纷案委托代理合同」 |
| 客户 | 委托人（必填） | 从下拉框选择已录入的客户 |
| 合同类型 | 服务合同/常年法顾/诉讼代理/非诉项目 | 根据业务类型选择 |
| 收费方式 | 固定收费/计时收费/风险代理/混合收费 | 根据约定选择 |
| 合同金额 | 约定的总金额（自动显示大写） | 如 50000 |
| 签约日期 | 合同签署日期（默认今天） | |
| 生效日期 | 合同开始生效的日期 | 通常与签约日期相同 |
| 到期日期 | 合同结束日期 | 根据约定填写 |

### 第四步：填写案件信息（诉讼类合同）

如果是诉讼代理合同，还需要填写：

| 字段 | 说明 |
|------|------|
| 案件类型 | 民事/刑事/行政/劳动仲裁/商事仲裁/执行案件 |
| 案由 | 级联选择具体案由（民事/行政案件需要选择） |
| 审理阶段 | 根据案件类型显示不同选项，支持多选 |
| 对方当事人 | 案件的对方 |
| 标的金额 | 案件涉及的金额（自动显示大写） |
| 管辖法院 | 案件管辖的法院 |
| 案情摘要 | 案件基本情况描述（用于收案审批表） |

**审理阶段选项说明**：

| 案件类型 | 可选阶段 |
|---------|---------|
| 民事案件 | 一审/二审/再审/执行 |
| 刑事案件 | 侦查阶段/审查起诉/一审/二审/死刑复核/再审 |
| 行政案件 | 行政复议/一审/二审/再审 |
| 劳动仲裁 | 仲裁阶段/一审/二审/再审/执行 |
| 商事仲裁 | 仲裁阶段/执行 |
| 执行案件 | 执行阶段/执行异议/执行复议 |

### 第五步：设置提成方案和参与人

#### 选择提成方案

1. 在「提成方案」下拉框中选择一个方案
2. 系统会自动显示该方案的比例设置：
   - 律所留存比例
   - 主办律师比例
   - 协办律师比例
   - 辅助人员比例
   - 案源人比例
3. 如果方案允许修改，你可以调整具体比例

#### 添加参与人

系统会根据提成方案自动添加参与人角色，你需要为每个角色选择具体的人：

| 角色 | 说明 |
|------|------|
| 承办律师（主办） | 负责案件的主要律师 |
| 协办律师 | 协助办案的律师 |
| 案源人 | 介绍这个案子的人 |
| 律师助理 | 协助工作的助理 |

点击「添加参与人」可以添加更多人员。每个参与人的提成比例会根据选择的提成方案自动填充。

### 第六步：填写付款条款

在「付款条款」文本框中，描述付款方式和时间，例如：
- 「合同签订后 3 日内支付全款」
- 「分两期支付：签约时支付 50%，结案时支付 50%」

### 第七步：保存或提交

- **保存草稿**：点击「保存」，合同状态为「草稿」，可以继续编辑
- **提交审批**：点击「提交审批」，合同进入审批流程

---

## 场景二：提交合同审批

### 草稿状态的合同

1. 在合同列表中找到草稿状态的合同
2. 点击「提交审批」按钮
3. 确认提交

### 审批流程

合同提交后，会按照律所设定的审批流程进行审批：

1. **小金额合同**（如 5 万以下）：部门负责人审批
2. **大金额合同**（如 5 万以上）：需要主任审批
3. **特殊合同**（如风险代理）：可能需要多级审批

### 查看审批进度

在合同列表中，可以看到合同的当前状态：
- **草稿**：还没提交审批
- **待审批**：已提交，等待审批
- **生效中**：审批通过
- **已拒绝**：审批被拒绝

点击合同可以查看详细的审批记录。

---

## 场景三：合同被拒绝了

如果合同审批被拒绝：

1. 查看拒绝原因（在合同详情或审批记录中）
2. 根据拒绝原因修改合同
3. 重新提交审批

---

## 场景四：查看合同详情

1. 在合同列表中找到合同
2. 点击合同编号或「详情」按钮
3. 详情页包含多个标签：

| 标签 | 内容 |
|------|------|
| 基本信息 | 合同的基本信息 |
| 付款计划 | 分期付款的计划 |
| 参与人 | 合同的参与律师和提成比例 |
| 审批记录 | 审批的历史记录 |

---

## 场景五：设置付款计划

如果合同是分期付款，可以设置付款计划：

1. 进入合同详情
2. 切换到「付款计划」标签
3. 点击「添加付款计划」
4. 填写：
   - 阶段名称（如「首付款」「尾款」）
   - 金额
   - 比例（占总金额的百分比）
   - 计划日期
5. 保存

付款计划会与财务模块关联，方便跟踪收款进度。

---

## 场景六：打印合同和收案审批表

合同审批通过后，可以打印正式合同和收案审批表：

### 打印操作

1. 进入合同详情
2. 点击「打印」按钮
3. 选择打印内容：
   - ☑ 打印合同
   - ☑ 打印收案审批表
4. 点击「确定」
5. 在弹出的预览窗口中点击「打印」

### 编辑收案审批表

如果需要修改收案审批表中的案情摘要：

1. 在合同详情页点击「编辑审批表」
2. 修改案情摘要内容
3. 点击「保存」

### 单独预览审批表

1. 点击「预览审批表」按钮
2. 在新窗口中查看审批表
3. 可以直接打印或关闭

---

## 场景七：合同变更

如果已生效的合同需要变更（如金额调整、期限延长）：

1. 在合同列表中找到合同
2. 点击「申请变更」
3. 填写变更信息：
   - 变更原因
   - 变更内容描述
   - 修改需要变更的字段
4. 提交变更审批

变更审批通过后，合同信息会更新。

## 合同状态说明

| 状态 | 含义 | 可以做什么 |
|------|------|-----------|
| 草稿 | 合同还在编辑中 | 编辑、提交审批、删除 |
| 待审批 | 已提交，等待审批 | 查看、撤回 |
| 生效中 | 审批通过，合同生效 | 查看、打印、创建项目、申请变更 |
| 已拒绝 | 审批被拒绝 | 编辑、重新提交 |
| 已终止 | 合同提前终止 | 只能查看 |
| 已完成 | 合同已履行完毕 | 只能查看 |

| 类型 | 说明 | 适用场景 |
|------|------|---------|
| 服务合同 | 一般法律服务合同 | 法律咨询、合同审查等 |

| 诉讼代理 | 诉讼案件代理合同 | 民事、刑事、行政诉讼 |

## 收费方式说明

| 方式 | 说明 | 计算方法 |
|------|------|---------|
| 固定收费 | 约定一个固定金额 | 按约定金额收取 |
| 计时收费 | 按工作时间收费 | 小时费率 × 工作时长 |
| 风险代理 | 根据案件结果收费 | 基础费 + 胜诉后按比例收取 |
| 混合收费 | 固定费用 + 风险费用 | 基础费 + 风险费 |

## 常见问题

**Q: 合同编号是怎么生成的？**
A: 系统自动生成，格式为 `HT + 年份 + 序号`，如 `HT202601001`。

**Q: 合同审批要多久？**
A: 取决于审批人的处理速度。你可以在合同详情中查看当前审批人，必要时可以联系催促。

**Q: 合同可以删除吗？**
A: 只有草稿状态的合同可以删除。已提交审批或已生效的合同不能删除。

**Q: 一个合同可以创建多个项目吗？**
A: 可以。比如法律顾问合同可能产生多个具体服务项目。

**Q: 合同金额可以修改吗？**
A: 草稿状态可以直接修改。已生效的合同需要通过「申请变更」流程修改。

**Q: 提成比例是怎么设置的？**
A: 提成比例由提成方案决定。选择提成方案后，系统会自动设置各角色的比例。如果方案允许修改，你可以调整具体比例。
