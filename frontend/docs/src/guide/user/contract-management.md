# 合同管理

合同是律所与客户之间的正式委托协议。在系统中，合同是创建项目的前提条件。

## 谁能使用合同管理？

| 功能     | 管理员 | 主任 | 团队负责人 | 财务 | 律师 | 行政 | 实习 |
| -------- | :----: | :--: | :--------: | :--: | :--: | :--: | :--: |
| 合同管理 |   ✅   |  ✅  |     ✅     |  ✅  |  ✅  |  ✅  |  ❌  |
| 合同审批 |   ✅   |  ✅  |     ✅     |  ❌  |  ❌  |  ❌  |  ❌  |

::: warning 实习律师限制实习律师不能访问合同管理。如需创建合同，请联系律师。:::

::: tip 审批权限合同审批由管理员、主任和团队负责人负责。:::

---

## 角色与操作权限概览

在具备「合同管理」或「合同审批」权限的前提下，不同操作还会受合同本身的创建人、签约人以及状态影响。下表为常见操作的权限说明：

| 操作 | 谁可以执行 | 前置条件 |
| --- | --- | --- |
| 创建合同 | 具有「合同管理」权限的用户（管理员/主任/团队负责人/财务/律师/行政） | 无 |
| 编辑草稿 | 合同创建人或签约人 | 合同状态为「草稿」 |
| 提交审批 | 合同创建人或签约人 | 合同状态为「草稿」或「已拒绝」 |
| 审批通过/审批拒绝 | 具有「合同审批」权限的审批人（管理员/主任/团队负责人），在审批中心中操作 | 合同状态为「待审批」，且为当前审批人 |
| 查看合同详情 | 具有「合同管理」权限的用户 | 合同已存在 |
| 打印合同和审批表 | 具有「合同管理」权限的用户 | 合同状态为「生效中」 |
| 编辑收案审批表 | 合同创建人或签约人 | 合同状态为「生效中」 |
| 创建项目 | 合同创建人或签约人 | 合同状态为「生效中」 |
| 申请合同变更 | 合同创建人或签约人 | 合同状态为「生效中」 |

其中「合同创建人或签约人」的判断逻辑与系统中按钮展示保持一致：只有当前登录用户是合同的创建人或签约人时，列表中才会出现「编辑」「提交审批」「重新提交」「变更」「创建项目」等操作。

## 进入合同管理

点击左侧菜单「项目管理」→「合同管理」

页面顶部显示统计卡片（合同总数、生效中、合同总金额、待收金额），下方是合同列表：

- 统计卡片展示的是当前系统中你有权限查看的合同总体情况，不会随着下方列表筛选条件改变而变化，适合作为全局概览。
- 合同总数：当前系统中所有状态的合同数量。
- 生效中：当前状态为「生效中」的合同数量。
- 合同总金额：所有合同的「合同金额」字段之和（单位：元），不区分币种时按人民币统计。
- 待收金额：合同总金额减去所有已确认收款金额之和，仅统计已在财务模块中确认的回款记录。
- 合同列表支持按创建年份和多种条件组合筛选，方便快速定位目标合同。

### 列表筛选说明

1. **创建年份**（左侧下拉框）
   - 默认选中当前年份，列表仅显示当年创建的合同；
   - 选择「全部年份」后，系统会清除时间范围限制，显示所有年份的合同；
   - 切换年份时，页码会自动重置为第 1 页。
2. **合同编号**
   - 支持输入完整编号或前缀进行模糊匹配；
   - 按回车或点击「查询」生效。
3. **合同名称**
   - 支持关键字模糊搜索，如输入「合同纠纷」可匹配包含该词的合同名称。
4. **收费方式**
   - 下拉选择固定收费 / 计时收费 / 风险代理 / 混合收费；
   - 可与其他条件组合使用。
5. **合同状态**
   - 下拉选择草稿 / 待审批 / 生效中 / 已拒绝 / 已终止 / 已完成；
   - 便于快速查看某一状态下的合同，例如仅查看「待审批」合同。
6. **签约人**
   - 使用人员树选择签约律师；
   - 支持按部门展开查找，便于大型律所按部门定位。

点击「查询」会根据当前筛选条件重新加载列表；点击「重置」会恢复默认筛选条件（**当前年份 + 清空其他条件**）。

常见组合示例：

- 按「创建年份 + 合同状态 = 待审批 + 签约人 = 自己」筛选：快速找到当前需要自己处理审批的合同；
- 按「创建年份 = 全部年份 + 收费方式 = 风险代理」筛选：统计近几年所有风险代理合同；
- 按「合同编号/名称 + 状态 = 生效中」筛选：精确定位某一份正在履行的合同。

### 列表字段说明

合同列表每一列的含义如下：

| 列名 | 含义 | 示例 |
| --- | --- | --- |
| 合同编号 | 系统自动生成的唯一编号 | CT-2026-0001 |
| 合同名称 | 创建合同时填写的合同标题 | 张三诉李四合同纠纷案委托代理合同 |
| 客户 | 关联的客户名称 | 北京某科技有限公司 |
| 合同类型 | 服务合同/常年法顾/诉讼代理/非诉项目 | 诉讼代理 |
| 收费方式 | 固定收费/计时收费/风险代理/混合收费 | 固定收费 |
| 合同金额 | 合同约定的总金额（列表中以金额格式展示） | ¥50,000.00 |
| 状态 | 当前合同所处状态 | 生效中 |
| 签约日期 | 合同签订日期 | 2026-01-15 |
| 操作 | 对合同进行的常见操作入口 | 查看、编辑、提交审批、重新提交、变更、创建项目等 |

## 场景一：创建新合同

### 第一步：点击创建

在合同列表页面，点击「创建合同」按钮。

### 第二步：选择合同模板（必选）

系统提供了多种合同模板：

- 服务合同
- 常年法律顾问合同
- 诉讼代理合同（民事/刑事）
- 非诉项目合同

创建新合同时，必须先选择一个合同模板，否则无法保存。

选择模板后，系统会自动填充合同类型和收费方式，并根据模板类型显示对应的字段。例如：

- 诉讼代理模板：会出现案件类型、案由、审理阶段、对方当事人、标的金额、管辖法院、案情摘要等案件信息字段
- 刑事类模板：在诉讼案件信息基础上，还会出现「被告人姓名」「涉嫌罪名」「辩护阶段」等字段
- 非诉项目模板：会出现「计时收费标准」（合伙人/资深律师/律师助理的小时费率）
- 常年法律顾问模板：会出现「服务小时数」等字段，用于约定每月服务时长

### 第三步：填写合同基本信息

| 字段 | 说明 | 示例 |
| --- | --- | --- |
| 合同名称 | 合同的名称，建议包含当事人姓名和案件类型，便于检索 | 「张三诉李四合同纠纷案委托代理合同」 |
| 客户 | 委托人（必填） | 从下拉框选择已录入的客户，如未找到可点击右侧「新建」快速创建客户；支持按客户编号或名称搜索 |
| 合同类型 | 服务合同/常年法顾/诉讼代理/非诉项目 | 根据业务类型选择 |
| 收费方式 | 固定收费/计时收费/风险代理/混合收费 | 根据约定选择 |
| 合同金额 | 约定的总金额（自动显示大写），用于统计和付款计划对齐 | 如 50000 |
| 签约日期 | 合同签署日期（默认今天） | 如 2026-01-15 |
| 生效日期 | 合同开始生效的日期 | 通常与签约日期相同，如有约定可以滞后生效 |
| 到期日期 | 合同结束日期 | 根据约定填写，如为长期顾问可约定一年或多年 |

### 第四步：填写案件信息（诉讼类合同）

如果是诉讼代理合同，还需要填写：

| 字段 | 说明 |
| --- | --- |
| 案件类型 | 民事案件/刑事案件/行政案件/破产案件/知识产权案件/劳动仲裁/商事仲裁/执行案件 |
| 案由 | 级联选择具体案由（仅对需要案由的案件类型显示，如民事案件/刑事案件/行政案件/劳动仲裁等，系统会从后台自动加载可选列表） |
| 审理阶段 | 根据案件类型从系统字典动态加载选项，支持多选 |
| 对方当事人 | 案件的对方 |
| 标的金额 | 案件涉及的金额（自动显示大写） |
| 管辖法院 | 案件管辖的法院 |
| 案情摘要 | 案件基本情况描述（用于收案审批表） |

**审理阶段选项说明**：

| 案件类型 | 可选阶段（具体选项由系统字典配置，仅作示例） |
| -------- | -------------------------------------------- |
| 民事案件 | 一审/二审/再审/执行等                        |
| 刑事案件 | 侦查阶段/审查起诉/一审/二审/死刑复核/再审等  |
| 行政案件 | 行政复议/一审/二审/再审等                    |
| 劳动仲裁 | 仲裁阶段/一审/二审/再审/执行等               |
| 商事仲裁 | 仲裁阶段/执行等                              |
| 执行案件 | 执行阶段/执行异议/执行复议等                 |

### 第五步：设置提成方案和参与人

#### 选择提成方案

1. 在「提成方案」下拉框中选择一个方案
2. 系统会自动显示该方案的比例设置：
   - 律所留存比例
   - 主办律师比例
   - 协办律师比例
   - 辅助人员比例
   - 案源人比例
3. 如果方案允许修改，你可以调整具体比例。系统会校验参与人提成比例总和，不允许超过 100%，避免重复分配；在合同详情页「参与人」标签上方，会实时显示「提成比例总和」，总和超过 100% 时会以红色标签提示「超过100%」。

#### 添加参与人

系统会根据提成方案自动添加参与人角色，你需要为每个角色选择具体的人：

| 角色             | 说明               |
| ---------------- | ------------------ |
| 承办律师（主办） | 负责案件的主要律师 |
| 协办律师         | 协助办案的律师     |
| 案源人           | 介绍这个案子的人   |
| 律师助理         | 协助工作的助理     |

点击「添加参与人」可以添加更多人员。每个参与人的提成比例会根据选择的提成方案自动填充。在合同详情页的「参与人」标签中，你还可以看到每位参与人的实际提成比例与方案中约定比例的对比，便于核对是否与律所政策一致。

如果你没有手动添加任何参与人，系统在保存合同时会自动将签约人或当前登录用户作为参与人，避免遗漏提成分配。

在「参与人」标签页中，你可以：

- 点击「添加参与人」：选择人员、角色，并设置提成比例；
- 点击「编辑」：修改已添加参与人的角色、提成比例或备注（人员本身不可更改，如需更换人员，可删除后重新添加）；
- 点击「删除」：删除不再参与该合同提成分配的人员。

### 第六步：填写付款条款

在「付款条款」文本框中，描述付款方式和时间，例如：

- 「合同签订后 3 日内支付全款」
- 「分两期支付：签约时支付 50%，结案时支付 50%」

在「其他信息」区域，根据收费方式和模板不同，还可以补充：

- 利冲审查状态：待审查/已通过/未通过/无需审查（默认无需审查）
- 当收费方式为风险代理时：填写「风险代理比例」
- 其他收费相关字段：如预支差旅费、争议解决方式、仲裁委员会、特别约定、备注等

### 第七步：保存或提交

- **保存草稿**：点击「保存」，合同状态为「草稿」，可以继续编辑
- **提交审批**：点击「提交审批」，合同进入审批流程

## 场景二：提交合同审批

### 草稿状态的合同

1. 在合同列表中找到草稿状态的合同
2. 点击「提交审批」按钮
3. 确认提交

### 审批流程

合同提交后，会按照律所设定的审批流程进行审批：

1. **小金额合同**（如 5 万以下）：部门负责人审批
2. **大金额合同**（如 5 万以上）：需要主任审批
3. **特殊合同**（如风险代理）：可能需要多级审批

具有「合同审批」权限的审批人可以有两种处理方式：

- 在「审批中心 → 待我审批」中集中处理所有业务的审批；
- 也可以在「合同管理」列表中，对状态为「待审批」的合同直接点击「通过」「拒绝」按钮进行处理（系统会弹出确认对话框，拒绝时需要填写拒绝事由，且不能为空或仅包含空格）。

### 查看审批进度

在合同列表中，可以看到合同的当前状态：

- **草稿**：还没提交审批
- **待审批**：已提交，等待审批
- **生效中**：审批通过，可打印合同、创建项目、申请变更
- **已拒绝**：审批被拒绝

点击合同可以查看详细的审批记录。

---

## 场景三：合同被拒绝了

如果合同审批被拒绝：

1. 查看拒绝原因（在合同详情或审批记录中）
2. 根据拒绝原因修改合同
3. 重新提交审批

提示：

- 合同被拒绝后状态为「已拒绝」，在合同列表的操作列会显示「编辑」「重新提交」；
- 重新提交后会重新进入审批流程，审批记录中会保留历史拒绝意见和新的审批结果，便于追踪每一次修改的原因。

## 场景四：查看合同详情

1. 在合同列表中找到合同
2. 点击合同编号或「详情」按钮
3. 详情页包含多个标签：

| 标签 | 内容 |
| --- | --- |
| 基本信息 | 合同的核心信息，包括合同编号、合同名称、客户、合同类型、收费方式、合同金额、已收金额、待收金额、审理阶段、标的金额、管辖法院、对方当事人、利冲审查状态、风险代理比例、签约/生效/到期日期等 |
| 付款计划 | 分期付款的计划 |
| 参与人 | 合同的参与律师和提成比例。你可以在这里查看、添加、编辑或删除参与人，系统会实时计算提成比例总和并提示是否超过 100%。 |
| 审批记录 | 审批的历史记录 |

## 场景五：设置付款计划

如果合同是分期付款，可以设置付款计划：

1. 进入合同详情
2. 切换到「付款计划」标签
3. 点击「添加付款计划」
4. 填写：
   - 阶段名称（如「首付款」「尾款」）
   - 金额（本阶段计划应收金额，支持两位小数）
   - 比例（占总金额的百分比，0–100，保留一位小数，可选）
   - 计划日期（预计收款日期）
   - 备注（可选，用于说明该阶段收款约定）
5. 保存

付款计划会与财务模块关联，方便跟踪收款进度。在「付款计划」标签页顶部，可以看到：

- **付款计划总额**：所有阶段金额之和
- **合同金额**：当前合同的总金额
- 当两者相等时，会显示「已匹配」标签；不相等时，会提示「差额：XXX 元」，便于校对。

在付款计划列表中，你可以：

- 点击「编辑」修改已有的付款计划，包括阶段名称、金额、比例、计划日期和备注；
- 点击「删除」移除不需要的付款计划记录。

列表中每一条付款计划会显示：

- 阶段名称：如「首付款」「二期」「尾款」；
- 金额：以金额格式展示本阶段计划金额；
- 比例(%)：该阶段对应的比例（如 30%、40%），方便与合同总金额对照；
- 计划日期/实际日期：计划收款时间和已记录的实际收款时间；
- 状态：如「待收款」「部分收款」「已收款」等，来源于财务模块的收款记录。

## 场景六：合同变更

合同生效后，如果因业务调整需要修改关键信息（如金额、期限、案件信息等），必须走变更流程，而不能直接编辑。

### 变更操作步骤

1. 在「合同管理」列表中，找到状态为「生效中」的合同。
2. 点击操作列的「变更」按钮（或在详情页右上角点击「变更」）。
3. 在弹出的「申请合同变更」窗口中填写：
   - **变更原因**：必填，说明为什么需要变更。
   - **变更内容说明**：详细描述修改了哪些条款。
   - **修改字段**：直接在表单中修改需要变更的信息（如合同金额、签约日期、案件信息等）。
4. 点击「提交审批」。

提交后，合同会生成一条变更记录，并可能需要重新经过审批流程（视律所配置而定）。

### 查看变更记录

在合同详情页的「变更记录」标签中，你可以查看到该合同的所有历史变更，包括：

- 变更编号与时间
- 变更原因
- 变更前后的关键字段对比（如金额从 10,000 变为 12,000）
- 审批结果

> **注意**：如果变更涉及金额调整，审批通过后系统会自动通知财务模块，财务人员会收到「合同变更待处理」提醒，以确保财务数据与合同一致。

1. 确认合同状态为「生效中」
2. 在合同列表中找到目标合同
3. 在操作列点击「创建项目」

## 业务全流程示例

下面用一个典型的诉讼代理合同为例，串起合同管理相关的主要操作流程。

1. **客户与需求**
   - 客户提出法律服务需求（如合同纠纷、劳动争议等）
   - 如系统中还没有该客户，可在创建合同时通过「客户」字段右侧的「新建」按钮快速创建客户
2. **创建合同草稿（本页：合同管理）**
   - 在「合同管理」列表中点击「创建合同」
   - 选择合适的合同模板（服务合同 / 常年法顾 / 诉讼代理 / 非诉项目）
   - 填写合同基本信息（合同名称、客户、合同类型、收费方式、金额、签约/生效/到期日期等）
   - 若为诉讼类合同，补充案件信息（案件类型、案由、审理阶段、对方当事人、标的金额、管辖法院、案情摘要等）
   - 选择提成方案并配置参与人（如不手动添加，系统会自动将签约人或当前用户作为参与人）
   - 设置付款条款、利冲审查状态、风险代理比例（如适用）等其他信息
   - 点击「保存」，生成状态为「草稿」的合同
3. **提交合同审批（本页：合同管理 + 审批中心）**
   - 在合同列表中找到草稿合同，点击「提交审批」
   - 合同状态变为「待审批」，出现在相关审批人的「审批中心 → 待我审批」列表中
   - 审批人可以在审批中心直接「通过」或「拒绝」：
     - 通过：合同状态变为「生效中」
     - 拒绝：合同状态变为「已拒绝」，拒绝时必须填写拒绝原因
   - 承办人可在合同详情中查看审批记录和拒绝原因，修改后重新提交审批
4. **合同生效后的操作（本页：合同管理 + 财务模块 + 项目管理）**
   - 合同状态为「生效中」时，可以：
     - 在合同详情中打印合同文本和收案审批表，供双方签字及律所存档
     - 在「付款计划」标签页中设置分期付款计划，查看「付款计划总额」是否与「合同金额」匹配
     - 基于合同创建项目（仅合同创建人或签约律师可见「创建项目」按钮），自动带入合同编号、名称、客户、金额等信息
     - 随着财务模块记录回款，合同详情中的「已收金额」「待收金额」会实时更新
5. **合同变更与终止（本页：合同管理）**
   - 如需调整金额、期限等，可在合同列表中对生效中的合同点击「变更」，填写变更原因和变更内容后提交审批
   - 变更审批通过后，合同信息更新，并保留变更记录
   - 如因履行完毕或提前解除等原因结束合同，可在系统中将合同状态处理为「已完成」或「已终止」，仅保留查看权限

## 数据权限与可见性

为了保障数据安全和业务规范，系统对不同角色的操作权限进行了如下限制：

| 操作 | 角色要求 | 状态限制 | 说明 |
| --- | --- | --- | --- |
| **查看合同** | 承办律师、审批人、财务人员、管理员 | 无 | 通常只能查看自己参与或负责的合同；管理员可查看所有。 |
| **编辑合同** | 创建人、签约律师 | 草稿、已拒绝 | 合同生效后不可直接编辑，必须走变更流程。 |
| **删除合同** | 管理员 | 草稿 | 已生效的合同通常不允许删除，仅可终止或完成。 |
| **提交审批** | 创建人、签约律师 | 草稿、已拒绝 |  |
| **审批/拒绝** | 具有「合同审批」权限的人员 | 待审批 | 审批人由系统配置决定（如团队长、主任等）。 |
| **申请变更** | 创建人、签约律师 | 生效中 |  |
| **创建项目** | 创建人、签约律师 | 生效中 | 基于合同快速创建项目。 |
| **编辑审批表** | 创建人、签约律师 | 生效中 | 仅限修改审批表中的备注或摘要信息。 |
| **确认收款** | 财务人员 | 生效中 | 在财务模块操作，合同模块仅展示状态。 |

## 常见问题

**Q: 合同编号是怎么生成的？** A: 系统自动生成，格式为 `HT + 年份 + 序号`，如 `HT202601001`。

**Q: 合同审批要多久？** A: 取决于审批人的处理速度。你可以在合同详情中查看当前审批人，必要时可以联系催促。

**Q: 合同可以删除吗？** A: 当前合同列表页面不提供删除按钮。出于合规和审计需要，一般不建议删除合同。如确有特殊情况，请联系管理员处理。

**Q: 一个合同可以创建多个项目吗？** A: 可以。比如法律顾问合同可能产生多个具体服务项目。

**Q: 合同金额可以修改吗？** A: 草稿状态可以直接修改。已生效的合同需要通过「申请变更」流程修改。

**Q: 提成比例是怎么设置的？** A: 提成比例由提成方案决定。选择提成方案后，系统会自动设置各角色的比例。如果方案允许修改，你可以调整具体比例。
