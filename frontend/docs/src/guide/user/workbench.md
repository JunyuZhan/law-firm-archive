# 工作台

工作台是您的日常工作中心，包含审批中心、日程管理等功能，帮助您高效处理各项事务。

## 谁能使用审批中心？

| 功能 | ADMIN | DIRECTOR | PARTNER | 财务 | 律师 | 行政 | 实习 |
|------|:-----:|:--------:|:-------:|:----:|:----:|:----:|:----:|
| 审批中心 | ❌ | ✅ | ✅ | ❌ | ❌ | ❌ | ❌ |

::: tip 说明
审批中心只有律所主任和合伙人可以访问。超级管理员是系统管理角色，不参与业务审批流程。
:::

---

## 审批中心

### 进入审批中心

点击左侧菜单「工作台」→「审批中心」

页面标题为「审批中心」，描述为「集中管理所有业务审批，快速处理待审批事项」。

页面分为两个标签页：
- **待我审批**：需要你处理的审批事项（显示待审批数量徽章）
- **我发起的**：你提交的审批申请（显示数量徽章）

---

### 场景一：处理待审批事项

1. 进入「审批中心」，默认显示「待我审批」
2. 页面显示所有待你审批的事项，包括：
   - 合同审批
   - 用印申请
   - 利冲检查
   - 费用报销
   - 收款变更
   - 项目结案
   - 转正申请
   - 离职申请
   - 工资表审批

3. 表格显示以下信息：

| 列名 | 说明 |
|------|------|
| 审批编号 | 审批的唯一编号 |
| 业务类型 | 审批的业务类型 |
| 业务标题 | 审批事项的标题 |
| 申请人 | 提交申请的人 |
| 优先级 | 高（红色）/中（橙色）/低（灰色） |
| 紧急程度 | 紧急（红色标签）/普通 |
| 申请时间 | 申请提交的时间 |

---

### 场景二：审批通过

**方式一：快速审批**
1. 在列表中找到要审批的事项
2. 直接点击「通过」按钮（绿色）
3. 在确认弹窗中点击「通过」

**方式二：查看详情后审批**
1. 点击「详情」查看完整信息
2. 在弹窗中查看审批详情
3. 点击「通过」按钮

---

### 场景三：审批拒绝

1. 点击「拒绝」按钮（红色）
2. 在弹窗中**必须填写拒绝原因**
3. 点击「拒绝」完成操作

::: warning 拒绝原因
拒绝审批时必须填写原因，让申请人知道为什么被拒绝，以便修改后重新申请。
:::

---

### 场景四：批量审批

当有多个审批事项需要处理时：

1. 勾选要审批的记录（左侧复选框）
2. 点击工具栏的「批量通过」或「批量拒绝」按钮
3. 批量拒绝时需要填写统一的拒绝原因
4. 确认后完成批量操作

::: tip 批量操作
批量操作可以大大提高审批效率，适合处理同类型的审批事项。
:::

---

### 场景五：查看我发起的审批

1. 切换到「我发起的」标签页
2. 查看你提交的所有审批申请
3. 表格显示以下信息：

| 列名 | 说明 |
|------|------|
| 审批编号 | 审批的唯一编号 |
| 业务类型 | 审批的业务类型 |
| 业务标题 | 审批事项的标题 |
| 审批人 | 处理审批的人 |
| 状态 | 待审批/已通过/已拒绝/已取消 |
| 审批意见 | 审批人填写的意见 |
| 申请时间 | 申请提交的时间 |
| 审批时间 | 审批处理的时间 |

4. 点击「查看」可以看到审批详情

---

### 场景六：查看审批详情

1. 点击「详情」或「查看」按钮
2. 弹窗显示审批的完整信息：
   - 审批编号
   - 业务类型和标题
   - 申请人和申请时间
   - 审批状态
   - 审批意见（如果已审批）
   - 业务详情

3. 在待审批详情中，可以直接进行审批操作

---

## 审批状态说明

| 状态 | 含义 | 颜色 |
|------|------|------|
| 待审批 | 等待审批人处理 | 橙色 |
| 已通过 | 审批通过 | 绿色 |
| 已拒绝 | 审批被拒绝 | 红色 |
| 已取消 | 申请人主动取消 | 灰色 |

---

### 场景七：跳转到业务详情

在审批详情弹窗中，点击「查看业务详情」可以跳转到对应的业务页面，查看完整的业务信息。

## 工作台首页

### 进入工作台首页

点击左侧菜单「首页」，或直接点击系统 Logo

工作台首页是你的日常工作中心，汇总展示各类待办事项和快捷入口。

---

### 场景八：查看欢迎区域

登录后，页面顶部显示欢迎区域（紫色渐变背景）：

1. **问候语**：根据当前时间显示不同的问候语
   - 早上好（6:00-9:00）
   - 上午好（9:00-12:00）
   - 中午好（12:00-14:00）
   - 下午好（14:00-18:00）
   - 晚上好（18:00-22:00）
   - 夜深了（22:00-6:00）

2. **当前日期**：显示完整的日期信息，如「2026年1月8日 星期四」

3. **审批中心按钮**：右侧显示「审批中心」按钮，带有待审批数量徽章

---

### 场景九：查看统计卡片

欢迎区域下方显示四个统计卡片，点击可跳转到对应页面：

| 卡片 | 图标 | 数据 | 点击跳转 |
|------|------|------|----------|
| 我的项目 | 📁 | 进行中的项目数量 | 我的项目页面 |
| 我的客户 | 👥 | 负责的客户数量 | 客户管理页面 |
| 本月工时 | ⏱️ | 本月记录的工时小时数 | 工时管理页面 |
| 待办任务 | ✅ | 待处理的任务数量 | 任务管理页面 |

---

### 场景十：查看待审批提醒

如果你有待处理的审批事项，统计卡片下方会显示醒目的提醒条（橙色背景）：

1. 显示 🔔 图标
2. 显示「您有 X 个待审批事项」
3. 显示「点击前往审批中心处理」
4. 点击整个提醒条可以直接跳转到审批中心

::: tip 及时处理审批
待审批事项会影响其他同事的工作进度，建议及时处理。
:::

---

### 场景十一：查看待办任务

页面左侧显示「待办任务」卡片：

1. 显示你的待办任务列表（最多 5 条）
2. 每条任务显示：
   - 优先级颜色条（高-红色/中-橙色/低-绿色）
   - 任务标题
   - 关联项目名称
   - 截止日期标签（已过期显示红色）
3. 点击「查看全部 →」跳转到任务管理页面
4. 如果没有待办任务，显示「暂无待办任务，休息一下吧 ☕」

---

### 场景十二：查看最近项目

页面右侧显示「最近项目」卡片：

1. 显示你参与的最近项目（最多 5 个）
2. 每个项目显示：
   - 项目首字母头像（颜色根据状态变化）
   - 项目名称
   - 客户名称
   - 项目状态标签
3. 点击「查看全部 →」跳转到项目列表页面
4. 点击任意项目可跳转到项目详情页
5. 如果没有项目，显示「暂无项目，开始创建吧 🚀」

---

### 场景十三：使用快捷操作

页面底部显示「快捷操作」卡片，提供常用功能的快捷入口：

| 操作 | 图标 | 跳转页面 |
|------|------|----------|
| 新建项目 | 📝 | 项目列表 |
| 新建客户 | 👤 | 客户管理 |
| 记录工时 | ⏰ | 工时管理 |
| 创建任务 | ✏️ | 任务管理 |
| 财务合同 | 📄 | 财务合同 |
| 审批中心 | 📋 | 审批中心（带待审批数量徽章） |

点击对应图标即可快速跳转。

## 日程管理

### 进入日程管理

点击左侧菜单「工作台」→「日程管理」

页面提供两种视图：
- **日历视图**：以月历形式展示日程
- **列表视图**：以表格形式展示日程

---

### 场景十四：在日历上创建日程

1. 切换到「日历视图」
2. 点击日历上的某一天
3. 弹出新建日程窗口
4. 填写日程信息：

| 字段 | 说明 |
|------|------|
| 日程标题 | 如「XX案开庭」（必填） |
| 日程类型 | 开庭/会议/期限/约见/其他（必填） |
| 日期 | 日程日期（必填） |
| 全天 | 勾选表示全天日程 |
| 时间 | 开始时间和结束时间（非全天时必填） |
| 地点 | 如「XX法院第3法庭」 |
| 关联项目 | 可选择关联的项目（可选） |
| 提前提醒 | 不提醒/提前5分钟/提前15分钟/提前30分钟/提前1小时/提前2小时/提前1天/提前2天 |
| 备注 | 其他备注信息 |

5. 点击「确定」保存

---

### 场景十五：在列表中管理日程

1. 切换到「列表视图」
2. 可以按日程类型筛选
3. 表格显示所有日程，包括标题、类型、时间、地点、关联项目、状态
4. 操作：
   - **编辑**：修改日程信息
   - **取消**：取消进行中的日程
   - **删除**：删除日程

---

### 场景十六：编辑日程

**在日历视图**：
1. 点击日历上的日程条目
2. 弹出编辑窗口
3. 修改信息后点击「确定」

**在列表视图**：
1. 点击「编辑」按钮
2. 修改信息后点击「确定」

---

### 场景十七：切换月份

在日历视图中：
- 点击「上月」查看上个月的日程
- 点击「下月」查看下个月的日程
- 点击「今天」快速回到当前月份

## 常见问题

**Q: 审批可以撤回吗？**
A: 已提交的审批不能撤回，但可以联系审批人说明情况。

**Q: 日程提醒是怎么提醒的？**
A: 系统会在设定的时间前通过系统通知提醒你。

**Q: 可以给日程添加参与人吗？**
A: 目前日程是个人日程，如需多人参与，建议在备注中说明参与人员。

**Q: 日程可以重复吗？**
A: 目前不支持重复日程，需要手动创建每次的日程。

## 报表中心

### 进入报表中心

点击左侧菜单「工作台」→「报表中心」

报表中心提供业务数据的可视化统计，帮助管理层了解律所运营情况。

---

### 场景十八：查看数据概览

进入报表中心，默认显示「数据概览」标签页：

**收入统计卡片**：
- 总收入：历史累计收入
- 本月收入：当月收入
- 本年收入：当年累计收入
- 待收金额：应收未收金额
- 收入趋势图：按月显示收入变化

**项目统计卡片**：
- 总项目数
- 进行中项目数
- 已完成项目数
- 项目状态分布饼图
- 项目类型分布柱状图

**客户统计卡片**：
- 总客户数
- 正式客户数
- 潜在客户数
- 本月新增客户数
- 客户类型分布图

**律师业绩排行**：
- 显示收入排名前10的律师
- 包含收入、案件数、工时等指标

---

### 场景十九：导出报表

1. 切换到「导出报表」标签页
2. 选择时间范围（开始日期和结束日期）
3. 选择要导出的报表类型：
   - **收入报表**：收入明细和汇总
   - **项目报表**：项目列表和统计
   - **客户报表**：客户列表和分析
   - **律师业绩报表**：律师业绩排名
   - **应收报表**：应收账款明细

4. 点击对应按钮，系统生成报表
5. 报表生成后可以下载 Excel 文件
