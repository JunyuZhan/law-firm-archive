## 审批中心

**菜单位置**：工作台 → 审批中心

审批中心用于集中管理所有业务审批，快速处理待审批事项。页面分为两个标签页：

- **待我审批**：需要你处理的审批（显示待审批数量）
- **我发起的**：你提交的审批申请

### 待我审批

表格显示以下信息：

| 列名     | 说明                                     |
| -------- | ---------------------------------------- |
| 审批编号 | 唯一编号                                 |
| 业务类型 | 合同审批、用印申请、利冲检查、费用报销等 |
| 业务标题 | 审批事项标题                             |
| 申请人   | 提交申请的人                             |
| 优先级   | 高（红）/ 中（橙）/ 低（灰）             |
| 紧急程度 | 紧急（红色标签）/ 普通                   |
| 申请时间 | 申请提交时间                             |

**操作**：

- **详情**：查看审批详细信息，可在详情页进行审批操作
- **通过**：直接审批通过
- **拒绝**：审批拒绝（需填写拒绝原因，必填）

### 场景一：审批通过

1. 在待我审批列表中找到要审批的记录
2. 点击「通过」按钮
3. 在确认弹窗中点击「通过」
4. 审批完成

### 场景二：审批拒绝

1. 在待我审批列表中找到要拒绝的记录
2. 点击「拒绝」按钮
3. 在弹窗中填写拒绝原因（必填，最多500字）
4. 点击「拒绝」确认

### 场景三：查看审批详情

1. 点击「详情」按钮
2. 查看审批详细信息：
   - 审批编号、业务类型、业务编号
   - 业务标题、申请人、申请时间
   - 审批人、状态、优先级、紧急程度
   - 审批时间、审批意见（如有）
3. 点击「查看业务详情」可跳转到对应业务页面
4. 待审批状态可在详情页直接进行审批操作

### 场景四：批量审批

支持同时处理多个审批申请，提高审批效率。

1. 在待我审批列表中，使用表格左侧的复选框勾选要处理的记录（可多选）
2. 点击「批量通过」或「批量拒绝」按钮
3. 批量通过：确认后直接通过选中的所有审批
4. 批量拒绝：必须填写统一的拒绝原因（必填，最多500字）
5. 确认后批量处理选中的审批

::: tip 批量操作限制

- 只有待审批状态（PENDING）的记录才能批量处理
- 批量拒绝时必须填写拒绝原因
- 批量操作后，系统会立即更新审批状态，相关业务状态也会同步更新 :::

### 我发起的

查看你提交的所有审批申请，包含审批状态和审批意见。

| 列名     | 说明                        |
| -------- | --------------------------- |
| 审批编号 | 唯一编号                    |
| 业务类型 | 业务类型                    |
| 业务标题 | 审批事项标题                |
| 审批人   | 审批人                      |
| 状态     | 待审批/已通过/已拒绝/已取消 |
| 审批意见 | 审批人的意见                |
| 申请时间 | 申请提交时间                |
| 审批时间 | 审批处理时间                |

| 状态      | 含义   | 颜色 |
| --------- | ------ | ---- |
| PENDING   | 待审批 | 橙色 |
| APPROVED  | 已通过 | 绿色 |
| REJECTED  | 已拒绝 | 红色 |
| CANCELLED | 已取消 | 灰色 |

## 日程管理

**菜单位置**：工作台 → 日程管理

管理开庭、会议、约见等日程安排。

### 视图切换

- **📅 日历视图**：月历形式展示日程
- **📋 列表视图**：表格形式展示日程

### 今日日程

页面顶部显示今天的所有日程，包括：

- 日程类型标签（带颜色）
- 开始时间
- 标题
- 地点

### 创建日程

1. 在日历视图点击某一天，或点击「新建日程」
2. 填写日程信息：

| 字段     | 说明                                           |
| -------- | ---------------------------------------------- |
| 日程标题 | 如「XX案开庭」（必填）                         |
| 日程类型 | 开庭/会议/期限/约见/其他（必填）               |
| 日期     | 日程日期（必填）                               |
| 全天     | 是否全天日程                                   |
| 时间     | 开始和结束时间（非全天时填写）                 |
| 地点     | 如「XX法院第3法庭」                            |
| 关联项目 | 可选择关联的项目                               |
| 提前提醒 | 不提醒/5分钟/15分钟/30分钟/1小时/2小时/1天/2天 |
| 备注     | 其他说明                                       |

3. 点击「确定」保存

### 日程类型

| 类型 | 颜色 |
| ---- | ---- |
| 开庭 | 红色 |
| 会议 | 蓝色 |
| 期限 | 橙色 |
| 约见 | 绿色 |
| 其他 | 紫色 |

### 日程状态

| 状态      | 含义   |
| --------- | ------ |
| ACTIVE    | 进行中 |
| COMPLETED | 已完成 |
| CANCELLED | 已取消 |

### 操作

- **编辑**：修改日程信息
- **取消**：取消进行中的日程
- **删除**：删除日程

### 切换月份

在日历视图中：

- 「上月」查看上个月
- 「今天」快速回到当前月
- 「下月」查看下个月

## 报表中心

**菜单位置**：工作台 → 报表中心

查看业务数据统计图表。

### 数据概览

**收入统计**：

- 总收入、本月收入、本年收入、待收金额
- 收入趋势图（按月折线图）

**项目统计**：

- 总项目数、进行中、已完成
- 项目状态分布饼图
- 项目类型分布柱状图

**客户统计**：

- 总客户数、正式客户、潜在客户、本月新增
- 客户类型分布环形图

**律师业绩排行**：

- 收入排名前10的律师（横向柱状图）
- 悬停显示：收入、案件数、工时

### 导出报表

1. 切换到「导出报表」标签页
2. 选择时间范围
3. 点击对应按钮导出：
   - 收入报表
   - 项目报表
   - 客户报表
   - 律师业绩报表
   - 应收报表

4. 下载 Excel 文件

## 常见问题

**Q: 审批可以撤回吗？**

已提交的审批不能撤回，需要联系审批人说明情况。

**Q: 拒绝审批必须填写原因吗？**

是的，拒绝审批时必须填写拒绝原因，最多500字。

**Q: 日程提醒是怎么提醒的？**

系统会在设定时间前通过系统通知提醒。

**Q: 日程可以设置重复吗？**

目前不支持重复日程，需要手动创建每次的日程。

**Q: 报表数据多久更新？**

报表数据是实时查询的，每次进入页面获取最新数据。

**Q: 可以在审批详情页直接审批吗？**

可以。点击「详情」进入审批详情页后，待审批状态的记录可以直接在详情页进行通过或拒绝操作。
