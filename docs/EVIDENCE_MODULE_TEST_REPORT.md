## 发现的问题及修复状态

### 问题 1: evidence_list 表结构不完整 ✅ 已修复

**问题描述**: 
`evidence_list` 数据库表缺少 `updated_by`、`version` 字段，且 `evidence_ids` 列类型不兼容。

**修复方案**:
已更新初始化脚本 `scripts/init-db/06-evidence-schema.sql`：
- 添加 `updated_by bigint` 字段
- 添加 `version integer DEFAULT 1` 字段
- 将 `evidence_ids` 类型从 `jsonb` 改为 `text`

**修复状态**: ✅ 已修复

### 问题 2: 未校验案件存在性 🟢 低优先级

**问题描述**: 
创建证据时未校验 `matterId` 对应的案件是否存在。

**影响范围**:
- 数据完整性

**修复建议**:
在 `EvidenceAppService.createEvidence` 中添加案件存在性校验。
