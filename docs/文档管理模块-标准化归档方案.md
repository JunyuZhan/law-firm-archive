# 文档管理模块 - 标准化归档方案

## 概述

根据律所实际业务需求，文档管理模块已重新设计为**以项目为核心的标准化归档系统**。系统根据不同项目类型（刑事、民事、法律顾问、非诉讼）自动生成对应的标准化文件夹结构，确保文档按照规范顺序存储，便于后续归档管理。

## 核心设计理念

### 1. 项目为中心
- 文档管理完全围绕项目展开
- 每个项目拥有独立的文档目录
- 离开项目的文档没有业务意义

### 2. 标准化归档
- 严格按照律所归档标准设置文件夹结构
- 不同项目类型对应不同的归档清单
- 文档按编号顺序排列，便于查找和归档

### 3. 类型化管理
- 根据项目类型自动生成对应的文件夹树
- 支持刑事、民事/行政、法律顾问、非诉讼四大类型
- 民事案件和行政案件使用相同的归档结构
- 每种类型都有专门的归档标准

## 标准化文件夹结构

### 一、刑事案件卷宗结构

```
刑事卷宗/
├── 01-收案审批表/
├── 02-授权委托书/
├── 03-委托合同/
├── 04-收费发票/
├── 05-办案机关卷宗材料/阅卷笔录/
├── 06-会见笔录/

├── 11-工作日志/
├── 12-结案（归档）审批表/
└── 13-其他材料/
```

> **说明**：民事案件和行政案件使用相同的归档结构，因为两者在诉讼程序和文档类型上基本一致。


## 功能特性

### 1. 智能文件夹生成
- 系统根据项目类型自动生成对应的文件夹结构
- 无需手动创建，确保结构标准化
- 支持动态切换不同项目类型的文件夹视图

### 2. 项目筛选功能
- **按年度筛选**：快速定位特定年份的项目
- **按项目类型筛选**：民事、刑事、商事、法律顾问等
- **按项目状态筛选**：进行中、已结案、已暂停等
- **关键词搜索**：项目名称、客户名称模糊搜索

### 3. 完整的文档操作
- ✅ **在线预览**：支持Word、PDF、图片等格式
- ✅ **文档编辑**：修改文档名称和描述信息
- ✅ **批量上传**：支持多文件同时上传
- ✅ **安全下载**：带权限控制的文档下载
- ✅ **文档分享**：生成临时分享链接（7天有效）
- ✅ **版本管理**：完整的文档版本历史记录
- ✅ **操作日志**：详细的文档访问和操作记录

### 4. 用户体验优化
- **卡片式项目展示**：直观显示项目信息和文档统计
- **树形文件夹导航**：清晰的层级结构展示
- **面包屑导航**：实时显示当前位置路径
- **响应式布局**：适配桌面端和移动端
- **拖拽上传**：支持拖拽文件到指定文件夹

## 业务价值

## 技术实现

### 1. 前端架构
- **Vue 3 + TypeScript**：现代化的前端技术栈
- **Ant Design Vue**：企业级UI组件库
- **响应式设计**：支持多种屏幕尺寸
- **组件化开发**：高度可复用的组件设计

### 2. 核心组件
- **ProjectDocumentManager**：主要的文档管理组件
- **FolderTree**：动态文件夹树组件
- **DocumentTable**：文档列表展示组件
- **UploadModal**：文档上传弹窗组件
