## 🔧 测试覆盖的 API 接口

### 客户管理 (`/client`)
- `GET /client/list` - 分页查询客户列表
- `GET /client/{id}` - 获取客户详情
- `POST /client` - 创建客户
- `PUT /client` - 更新客户
- `DELETE /client/{id}` - 删除客户
- `PUT /client/{id}/status` - 修改客户状态
- `GET /client/select-options` - 客户选择列表

### 案源管理 (`/client/lead`)
- `GET /client/lead` - 查询案源列表
- `GET /client/lead/{id}` - 获取案源详情
- `POST /client/lead` - 创建案源
- `GET /client/lead/{id}/follow-ups` - 查询跟进记录
- `POST /client/lead/{id}/follow-up` - 创建跟进记录
- `GET /client/lead/statistics` - 案源统计

### 利冲审查 (`/client/conflict-check`)
- `GET /client/conflict-check/list` - 利冲审查列表
- `GET /client/conflict-check/{id}` - 利冲审查详情
- `POST /client/conflict-check/quick` - 快速利冲检索
- `POST /client/conflict-check/apply` - 申请利冲审查
- `POST /client/conflict-check/{id}/approve` - 审批通过

### 联系人管理 (`/client/contact`)
- `GET /client/contact/client/{clientId}` - 客户联系人列表
- `GET /client/contact/{id}` - 联系人详情
- `POST /client/contact` - 创建联系人
- `PUT /client/contact/{id}` - 更新联系人
- `DELETE /client/contact/{id}` - 删除联系人
