### ⚠️ 发现的问题

#### 问题1：parent_code字段未填充

**问题描述**：
- SQL脚本中的INSERT语句没有包含`parent_code`字段
- 对于二级案由（如`8.1`, `8.2`），应该设置`parent_code='8'`
- 当前INSERT语句格式：
  ```sql
  INSERT INTO sys_cause_of_action (code, name, cause_type, category_code, category_name, level, sort_order) VALUES
  ```
- 缺少`parent_code`字段

**影响**：
- 无法通过`parent_code`字段建立父子关系
- 前端级联选择可能无法正常工作
- 数据关联性缺失

**修复方案**：
1. 修改INSERT语句，添加`parent_code`字段
2. 对于二级案由，自动提取父代码
3. 一级案由的`parent_code`为NULL
