## 一、功能概览

### 1.1 系统定位

面向中小型律所（20-50人）的一体化业务管理平台，覆盖律所日常运营的核心业务流程。

### 1.2 用户角色

| 角色 | 说明 | 主要功能 |
|------|------|----------|
| 超级管理员 | 系统最高权限 | 系统配置、用户管理、全部数据 |
| 合伙人 | 律所合伙人 | 审批决策、业绩分析、全局数据 |
| 财务主管 | 财务管理负责人 | 财务审批、报表统计、全部财务数据 |
| 财务人员 | 财务操作人员 | 收款核销、发票管理、提成计算 |
| 主办律师 | 案件主办人 | 项目管理、文书处理、团队协调 |
| 协办律师 | 案件协办人 | 任务执行、文书协作、工时记录 |
| 行政人员 | 行政后勤人员 | 考勤管理、资产管理、档案管理 |
| 实习律师 | 实习人员 | 辅助工作、学习成长、有限权限 |
