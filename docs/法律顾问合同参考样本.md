# 法律顾问合同参考样本

## 常年法律顾问合同

```html
<h2 style="text-align: center; letter-spacing: 0.5em;">常年法律顾问合同</h2>
<p style="text-align: center;">合同编号：${contractNo}</p>

<table style="width: 100%; border-collapse: collapse; margin: 20px 0;">
  <tr>
    <td style="width: 15%; padding: 8px; border: 1px solid #000;"><strong>甲方</strong></td>
    <td style="padding: 8px; border: 1px solid #000;">${clientName}</td>
  </tr>
  <tr>
    <td style="padding: 8px; border: 1px solid #000;"><strong>地址</strong></td>
    <td style="padding: 8px; border: 1px solid #000;">${clientAddress}</td>
  </tr>
  <tr>
    <td style="padding: 8px; border: 1px solid #000;"><strong>联系人</strong></td>
    <td style="padding: 8px; border: 1px solid #000;">${clientPhone}</td>
  </tr>
  <tr>
    <td style="padding: 8px; border: 1px solid #000;"><strong>乙方</strong></td>
    <td style="padding: 8px; border: 1px solid #000;">${firmName}</td>
  </tr>
  <tr>
    <td style="padding: 8px; border: 1px solid #000;"><strong>地址</strong></td>
    <td style="padding: 8px; border: 1px solid #000;">${firmAddress}</td>
  </tr>
  <tr>
    <td style="padding: 8px; border: 1px solid #000;"><strong>电话</strong></td>
    <td style="padding: 8px; border: 1px solid #000;">${firmPhone}</td>
  </tr>
</table>

<p>甲方聘请乙方担任常年法律顾问，为甲方提供法律服务。经双方友好协商，根据《中华人民共和国律师法》及有关法律法规的规定，达成如下协议：</p>

<h3>第一条 服务内容</h3>
<p>乙方为甲方提供以下法律服务：</p>
<p>（一）为甲方日常经营活动提供法律咨询；</p>
<p>（二）协助甲方草拟、审核、修改各类合同、协议及其他法律文书；</p>
<p>（三）为甲方的重大经营决策提供法律意见；</p>
<p>（四）应甲方要求，参与重要商务谈判，提供法律支持；</p>
<p>（五）协助甲方建立和完善内部管理规章制度；</p>
<p>（六）为甲方员工进行法律知识培训；</p>
<p>（七）代理甲方处理非诉讼法律事务；</p>
<p>（八）定期向甲方提供与其业务相关的法律法规信息；</p>
<p>（九）双方约定的其他法律服务事项。</p>

<h3>第二条 服务方式</h3>
<p>（一）电话咨询：工作时间随时接受甲方电话咨询；</p>
<p>（二）书面意见：根据甲方需要出具书面法律意见书；</p>
<p>（三）现场服务：根据甲方需要，安排律师到甲方办公场所提供现场法律服务；</p>
<p>（四）远程服务：通过电子邮件、视频会议等方式提供法律服务。</p>

<h3>第三条 服务期限</h3>
<p>本合同有效期为<strong>壹年</strong>，自<strong>${effectiveDate}</strong>起至<strong>${expiryDate}</strong>止。</p>
<p>合同期满前三十日，任何一方未书面提出终止合同的，本合同自动续展一年，续展次数不限。</p>

<h3>第四条 顾问费用及支付</h3>
<p>（一）年度顾问费为人民币<strong>${totalAmount}</strong>元（大写：${totalAmountChinese}）；</p>
<p>（二）付款方式：${paymentTerms}；</p>
<p>（三）乙方收到顾问费后，向甲方开具正式发票；</p>
<p>（四）本合同约定的服务内容范围内的法律服务，不另行收取费用。</p>

<h3>第五条 特别事项收费</h3>
<p>以下事项不包含在常年法律顾问服务范围内，如甲方委托乙方办理，双方另行协商收费：</p>
<p>（一）代理诉讼、仲裁案件；</p>
<p>（二）代理非诉讼重大专项法律事务；</p>
<p>（三）参与重大项目的尽职调查；</p>
<p>（四）其他双方约定的特别事项。</p>

<h3>第六条 甲方权利和义务</h3>
<p>（一）有权要求乙方按本合同约定提供法律服务；</p>
<p>（二）有权了解法律服务工作进展情况；</p>
<p>（三）应按时足额支付顾问费用；</p>
<p>（四）应如实向乙方提供与法律事务相关的资料和信息；</p>
<p>（五）应配合乙方开展法律服务工作。</p>

<h3>第七条 乙方权利和义务</h3>
<p>（一）有权按本合同约定收取法律顾问费；</p>
<p>（二）应指派具有专业能力的律师为甲方提供服务；</p>
<p>（三）应勤勉尽责，维护甲方合法权益；</p>
<p>（四）应对甲方的商业秘密和个人隐私严格保密；</p>
<p>（五）应及时回复甲方的法律咨询，一般问题24小时内答复，复杂问题不超过3个工作日。</p>

<h3>第八条 保密条款</h3>
<p>（一）乙方对在服务过程中知悉的甲方商业秘密、经营信息等负有保密义务；</p>
<p>（二）保密义务不因本合同的终止而解除；</p>
<p>（三）乙方应要求相关律师及工作人员遵守保密义务。</p>

<h3>第九条 合同变更与解除</h3>
<p>（一）经双方协商一致，可以变更或解除本合同；</p>
<p>（二）任何一方解除合同应提前三十日书面通知对方；</p>
<p>（三）合同解除后，双方应就已提供的服务和已支付的费用进行结算。</p>

<h3>第十条 违约责任</h3>
<p>（一）甲方逾期支付顾问费的，每逾期一日应按应付款项的万分之五支付违约金；</p>
<p>（二）乙方未按约定提供服务或服务质量不符合要求的，甲方有权要求乙方改正，情节严重的，甲方有权解除合同并要求退还相应费用。</p>

<h3>第十一条 争议解决</h3>
<p>因本合同产生的争议，双方应友好协商解决；协商不成的，任何一方均可向乙方所在地人民法院提起诉讼。</p>

<h3>第十二条 其他约定</h3>
<p>（一）本合同未尽事宜，由双方协商后签订补充协议；</p>
<p>（二）本合同一式两份，甲乙双方各执一份，具有同等法律效力；</p>
<p>（三）本合同自双方签字盖章之日起生效。</p>

<p style="margin-top: 3em;">&nbsp;</p>

<table style="width: 100%; border: none;">
  <tr>
    <td style="width: 50%; vertical-align: top; border: none;">
      <p><strong>甲方（盖章）：</strong></p>
      <p>&nbsp;</p>
      <p>&nbsp;</p>
      <p><strong>法定代表人/授权代表：</strong></p>
      <p>&nbsp;</p>
      <p><strong>日期：</strong></p>
    </td>
    <td style="width: 50%; vertical-align: top; border: none;">
      <p><strong>乙方（盖章）：</strong>${firmName}</p>
      <p>&nbsp;</p>
      <p>&nbsp;</p>
      <p><strong>负责人：</strong>${firmLegalPerson}</p>
      <p><strong>承办律师：</strong>${lawyerName}</p>
      <p><strong>日期：</strong>${signDate}</p>
    </td>
  </tr>
</table>
```

## 可用变量说明

| 变量名 | 说明 | 示例值 |
|--------|------|--------|
| `${contractNo}` | 合同编号（系统自动生成） | 2026顾字第0001号 |
| `${clientName}` | 委托人/客户名称 | 贵州XX科技有限公司 |
| `${clientAddress}` | 客户地址 | 贵州省贵阳市XX路XX号 |
| `${clientPhone}` | 客户联系电话 | 13800138000 |
| `${firmName}` | 律师事务所名称 | 贵州威迪律师事务所 |
| `${firmAddress}` | 律所地址 | 贵州省毕节市七星关区开行路联邦金座10楼 |
| `${firmPhone}` | 律所电话 | 0857-8228444 |
| `${firmLegalPerson}` | 律所负责人 | 孟天明 |
| `${lawyerName}` | 承办律师 | 李律师 |
| `${lawyerLicenseNo}` | 律师执业证号 | 15200201912345678 |
| `${totalAmount}` | 合同金额 | 50,000.00 |
| `${totalAmountChinese}` | 大写金额 | 伍万元整 |
| `${effectiveDate}` | 生效日期 | 2026年1月1日 |
| `${expiryDate}` | 到期日期 | 2026年12月31日 |
| `${signDate}` | 签订日期 | 2026年1月5日 |
| `${paymentTerms}` | 付款方式 | 按年支付，于每年1月15日前支付 |

---

## 使用说明

1. 将上述HTML模板复制到合同模板编辑器中
2. 系统会自动识别 `${}` 格式的变量
3. 生成实际合同时，变量会自动替换为真实数据
4. 合同编号 `${contractNo}` 会使用系统管理中配置的编号格式
