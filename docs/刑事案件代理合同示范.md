# 刑事案件代理合同示范

本文档包含刑事案件委托代理合同的两个版本：标准版和简版。这些模板已集成到系统的合同模板管理中。

## 合同模板统一命名

系统中的合同模板采用以下统一命名规范：

| 模板名称 | 适用场景 | 合同类型 |
|---------|---------|---------|
| 民事委托代理合同（标准版） | 民事、行政案件诉讼代理 | 诉讼代理 |
| 刑事委托代理合同（标准版） | 刑事案件辩护，完整条款 | 诉讼代理 |
| 刑事委托代理合同（简版） | 刑事案件辩护，简化版本 | 诉讼代理 |
| 常年法律顾问合同 | 企业常年法律顾问服务 | 常年法顾 |

## 使用方法

1. 进入 **系统管理 → 合同模板管理**
2. 点击 **新增模板** 按钮
3. 在弹出的编辑窗口中，点击 **选择模板** 下拉菜单
4. 选择需要的模板类型
5. 系统会自动加载模板内容和名称
6. 可根据需要修改模板内容
7. 点击 **保存模板** 完成创建

## 支持的变量

模板中可使用以下变量，在生成实际合同时会自动替换为真实数据：

### 委托人信息
- `${clientName}` - 委托人姓名
- `${clientIdNumber}` - 身份证号码
- `${clientAddress}` - 联系地址
- `${clientPhone}` - 联系电话
- `${clientPostcode}` - 邮政编码

### 律所信息
- `${firmName}` - 律师事务所名称
- `${firmAddress}` - 律所地址
- `${firmPhone}` - 律所电话
- `${firmPostcode}` - 邮政编码
- `${firmLegalPerson}` - 律所负责人
- `${firmBankName}` - 开户银行
- `${firmBankAccount}` - 银行账号

### 承办律师
- `${lawyerName}` - 承办律师姓名
- `${lawyerLicenseNo}` - 律师执业证号

### 案件信息
- `${matterName}` - 案件名称
- `${causeOfAction}` - 案由/罪名
- `${procedureStage}` - 代理程序/阶段
- `${authorityScope}` - 代理权限
- `${defendant}` - 被告人/犯罪嫌疑人
- `${opposingParty}` - 对方当事人
- `${relatedParty}` - 关联当事人

### 费用信息
- `${totalAmount}` - 合同金额
- `${totalAmountChinese}` - 大写金额
- `${paymentTerms}` - 付款方式

### 日期信息
- `${contractNo}` - 合同编号（系统自动生成）
- `${signDate}` - 签订日期
- `${effectiveDate}` - 生效日期
- `${expiryDate}` - 到期日期
- `${currentYear}` - 当前年份
- `${currentDate}` - 当前日期

## 注意事项

1. 合同编号 `${contractNo}` 会根据系统管理中配置的编号格式自动生成
2. 模板支持富文本格式，包括粗体、表格、字号等
3. 预览时变量会显示为蓝色示例值，实际合同中为黑色正文
4. 可通过打印功能导出为PDF文件
