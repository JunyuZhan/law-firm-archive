# Grafana 中文界面设置指南

## 📋 概述

Grafana 支持多种语言，包括中文（简体）和中文（繁体）。本指南介绍如何手动设置 Grafana 界面为中文。

---

## 🌐 方法一：通过用户偏好设置（推荐）

### 步骤详解

#### 1. 登录 Grafana

访问 Grafana 地址：
- 生产环境：`http://your-server-ip:3000` 或 `http://grafana.albertzhan.top`
- 本地开发：`http://localhost:3000`

**登录信息**：
- **用户名**：`admin`
- **密码**：查看 `.env` 文件中的 `GRAFANA_PASSWORD`，如果没有设置则默认为 `admin`

#### 2. 打开用户偏好设置

有两种方式可以打开偏好设置：

**方式 A：通过左下角用户图标**
1. 在 Grafana 界面左下角找到用户图标（圆形头像）
2. 点击用户图标
3. 在下拉菜单中选择 **"Preferences"**（偏好设置）

**方式 B：通过右上角头像**
1. 在 Grafana 界面右上角找到用户头像
2. 点击头像
3. 在下拉菜单中选择 **"Preferences"**（偏好设置）

#### 3. 选择中文语言

在偏好设置页面中：

1. 找到 **"Language"**（语言）选项
2. 点击语言下拉菜单
3. 选择以下选项之一：
   - **"中文（简体）"** - 简体中文（推荐）
   - **"中文（繁体）"** - 繁体中文
   - **"中文（简体，中国）"** - 简体中文（中国）
   - **"中文（繁体，台湾）"** - 繁体中文（台湾）

#### 4. 保存设置

1. 点击页面底部的 **"Save"**（保存）按钮
2. 界面会立即刷新并显示为中文

## 🎯 详细操作路径

### 完整操作路径

```
登录 Grafana
  ↓
点击左下角用户图标（或右上角头像）
  ↓
选择 "Preferences"（偏好设置）
  ↓

  ↓
选择 "中文（简体）"
  ↓
点击 "Save"（保存）
  ↓
界面自动切换为中文 ✅
```

## ⚙️ 方法二：通过配置文件设置（全局默认）

如果你想为所有用户设置默认中文，可以通过环境变量配置：

### Docker Compose 配置

在 `docker/docker-compose.prod.yml` 中，Grafana 服务已配置默认中文：

```yaml
grafana:
  environment:
    - GF_DEFAULT_LANGUAGE=zh-Hans  # 设置默认语言为简体中文
```

### 应用配置

```bash
# 重启 Grafana 容器使配置生效
cd docker
docker compose --env-file ../.env -f docker-compose.prod.yml restart grafana
```

**注意**：此方法设置的是**新用户的默认语言**，已登录用户的个人设置不受影响。

---

## 🔄 切换回英文

如果需要切换回英文：

1. 按照上述步骤打开 **Preferences**（偏好设置）
2. 在 **Language**（语言）下拉菜单中选择 **"English"**
3. 点击 **"Save"**（保存）

## ❓ 常见问题

### Q1: 找不到 Preferences 选项？

**A**: 确保你已经登录。如果没有登录，先登录后再操作。

### Q2: 保存后界面没有变化？

**A**: 
- 尝试刷新页面（F5 或 Ctrl+R）
- 清除浏览器缓存后重新访问
- 检查浏览器是否阻止了页面刷新

### Q3: 语言选项中没有中文？

**A**: 
- 确保 Grafana 版本支持中文（Grafana 7.0+ 都支持）
- 尝试更新 Grafana 镜像：`docker pull grafana/grafana:latest`

### Q4: 设置后部分内容还是英文？

**A**: 
- 这是正常的，某些插件或自定义面板可能没有中文翻译
- 核心界面和系统面板应该已经显示为中文

### Q5: 如何为所有用户设置默认中文？

**A**: 
- 使用配置文件方法（方法二）
- 或者通过 Grafana 管理界面设置组织级别的默认语言
