## 1. 背景与结论

### 1.1 法律依据

根据《中华人民共和国国家赔偿法》，国家赔偿分为 **行政赔偿** 和 **刑事赔偿**，两者在以下方面存在本质差异：

| 维度 | 行政赔偿 | 刑事赔偿 |
|------|----------|----------|
| **前置条件** | 无 | 必须 **刑事诉讼终结** |
| **复议性质** | 行政复议（《行政复议法》） | 刑事赔偿复议（《国家赔偿法》） |
| **最终救济** | **行政诉讼** | **法院赔偿委员会** |
| **审理程序** | 普通行政审判 | 赔偿委员会决定（非诉讼） |
| **可否调解** | 可以 | 不适用 |

**结论**：行政赔偿和刑事赔偿程序完全不同，不能使用同一案件类型。

## 2. 目标与范围

### 2.1 目标

- 在案件创建/编辑中可选择"行政国家赔偿"或"刑事国家赔偿"案件类型
- 支持国家赔偿案件的关键业务字段记录（赔偿义务机关、致损行为、请求金额、程序节点日期等）
- 支持刑事赔偿特有的"刑事诉讼终结"确认和原刑事案件关联
- 对存量数据保持兼容，不影响既有案件类型与模板变量功能

### 2.2 非目标（本阶段不做）

- 不定义完整的"国家赔偿事项库"权威版本与更新机制
- 不扩展完整的办案流程引擎（沿用现有任务/期限/工时模块）

## 3. 方案选择

### 3.1 方案 A：单一类型 + 扩展表区分（❌ 不推荐）

```
caseType = "STATE_COMPENSATION"
扩展表 compensation_track = "ADMINISTRATIVE" | "CRIMINAL"
```

**问题**：
- 用户无法直观区分行政/刑事赔偿
- 代理阶段字典需要根据 track 动态加载，复杂度高
- 统计时需要 JOIN 扩展表才能区分

### 3.2 方案 B：两个独立案件类型（✅ 推荐）

```
caseType = "STATE_COMP_ADMIN"     // 行政国家赔偿
caseType = "STATE_COMP_CRIMINAL"  // 刑事国家赔偿
```

**优点**：
- 用户界面清晰，可直接选择
- 各自有独立的代理阶段字典
- 统计、权限、合同编号都清晰
- 符合国家赔偿法的法律架构

**本方案后续说明以方案 B 为主。**

## 4. 数据模型设计

### 4.1 主表

继续使用现有 `matter` 表：

| 字段 | 值 | 说明 |
|------|-----|------|
| `matterType` | `LITIGATION` | 归入诉讼/争议类项目 |
| `caseType` | `STATE_COMP_ADMIN` / `STATE_COMP_CRIMINAL` | 新增两个类型 |
| `litigationStage` | 动态字典 | 根据案件类型加载对应阶段字典 |
| `claimAmount` | 请求赔偿总额 | 用于"请求赔偿总额" |

### 4.2 扩展表：国家赔偿案件业务信息

> **[v2.1修正]** 删除冗余的 `compensation_track` 字段，该信息可从 `matter.case_type` 推断

```sql
CREATE TABLE matter_state_compensation (
    matter_id BIGINT PRIMARY KEY REFERENCES matter(id) ON DELETE CASCADE,

    -- 赔偿义务机关
    obligor_org_name VARCHAR(255),            -- 赔偿义务机关名称
    obligor_org_type VARCHAR(50),             -- 字典：PUBLIC_SECURITY/PROCURATORATE/COURT/PRISON/ADMIN_ORGAN

    -- 致损行为
    case_source VARCHAR(50),                  -- 字典：违法拘留/违法强制措施/错误判决/违法查封等
    damage_description TEXT,                  -- 损害情况描述

    -- 刑事赔偿特有字段
    criminal_case_terminated BOOLEAN,         -- 刑事诉讼是否终结（刑事赔偿必填）
    criminal_case_no VARCHAR(100),            -- 原刑事案件编号
    compensation_committee VARCHAR(255),      -- 受理的赔偿委员会

    -- 程序日期
    application_date DATE,                    -- 赔偿申请日（2年时效）
    acceptance_date DATE,                     -- 受理日
    decision_date DATE,                       -- 赔偿义务机关决定日（2个月期限）
    reconsideration_date DATE,                -- 复议/复核申请日（30日期限）
    reconsideration_decision_date DATE,       -- 复议决定日
    committee_app_date DATE,                  -- 赔偿委员会申请日
    committee_decision_date DATE,             -- 赔偿委员会决定日
    admin_litigation_filing_date DATE,        -- 行政赔偿诉讼立案日
    admin_litigation_court_name VARCHAR(255), -- 行政诉讼法院

    -- 赔偿请求
    claim_amount DECIMAL(18,2),               -- 请求赔偿总额
    compensation_items JSONB,                 -- 赔偿项目明细
    -- JSONB 结构示例：
    -- [
    --   {"type": "人身自由赔偿", "days": 100, "daily_amount": 436.89, "amount": 43689},
    --   {"type": "精神损害抚慰金", "amount": 50000},
    --   {"type": "财产损害赔偿", "amount": 100000}
    -- ]

    -- 决定结果
    decision_result VARCHAR(50),              -- 字典：GRANTED/DENIED/PARTIAL_GRANTED
    approved_amount DECIMAL(18,2),            -- 决定赔偿金额
    payment_status VARCHAR(50),               -- 字典：UNPAID/PAID/PARTIAL_PAID
    payment_date DATE,

    remark TEXT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 索引
CREATE INDEX idx_msc_obligor_org ON matter_state_compensation(obligor_org_type);
CREATE INDEX idx_msc_decision_date ON matter_state_compensation(decision_date);
```

**字段说明**：

| 字段 | 适用范围 | 必填 | 说明 |
|------|----------|------|------|
| `criminal_case_terminated` | 刑事赔偿 | ✅ | 刑事赔偿的前置条件 |
| `criminal_case_no` | 刑事赔偿 | 推荐 | 关联原刑事案件 |
| `admin_litigation_filing_date` | 行政赔偿 | - | 进入行政诉讼时填写 |
| `compensation_committee` | 刑事赔偿 | - | 赔偿委员会名称 |

## 11. 附录：法律程序参考

### 11.1 行政赔偿程序

```
┌─────────────┐
│  赔偿申请   │ → 向赔偿义务机关提出（2年时效）
└─────────────┘
       │
       ▼
┌─────────────┐
│  机关决定   │ → 赔偿义务机关2个月内作出决定
└─────────────┘
       │
       ▼ 逾期或不服
┌─────────────┐
│  行政复议   │ → 收到决定后30日内申请
└─────────────┘
       │
       ▼ 仍不服
┌─────────────┐
│  行政诉讼   │ → 向法院提起行政赔偿诉讼
└─────────────┘
```

### 11.2 刑事赔偿程序

```
┌─────────────────┐
│ 刑事诉讼终结    │ → 前置条件（必填）
└─────────────────┘
       │
       ▼
┌─────────────┐
│  赔偿申请   │ → 向赔偿义务机关提出
└─────────────┘
       │
       ▼
┌─────────────┐
│  机关决定   │ → 赔偿义务机关2个月内决定
└─────────────┘
       │
       ▼ 逾期或不服
┌─────────────┐
│   复议申请  │ → 向上一级机关申请复议
└─────────────┘
       │
       ▼ 仍不服
┌─────────────┐
│ 赔偿委员会  │ → 向同级法院赔偿委员会申请
└─────────────┘
```

### 11.3 关键时间节点

| 事项 | 时限 |
|------|------|
| 申请时效 | 知道侵权之日起 **2年** |
| 机关决定期限 | 收到申请后 **2个月** |
| 复议申请期限 | 收到决定后 **30日** |
| 材料补正告知 | 收到申请后 **5日** 内一次性告知 |
| 赔偿委员会立案 | 符合条件 **7日** 内立案 |
| 人身自由赔偿金标准 | **436.89元/日**（2024年标准） |
