# 文档6：财务管理模块开发指南

## 概述

财务管理模块是智慧律所管理系统的核心业务模块，涵盖收费合同管理、收款核销、提成计算、发票管理等核心业务流程。本指南详细阐述财务管理模块的业务逻辑、技术实现和开发规范，确保系统符合律师事务所的真实业务需求。

## 一、业务架构设计

### 1.1 核心业务目标

1. **收入管理**：规范合同收费、跟踪收款进度、确保资金安全
2. **成本控制**：管理办案成本、控制运营费用、提高利润率
3. **提成核算**：公平计算律师提成、激励团队积极性
4. **税务合规**：准确计算税费、规范发票管理、确保税务合规
5. **决策支持**：提供财务数据报表、支持管理层决策

## 二、详细业务逻辑设计

### 2.1 收费合同管理

#### 2.1.1 合同收费方式

律师事务所常见的收费方式：

1. **固定收费**：一次性收取固定金额
   - 适用场景：常年法律顾问、标准合同审查
   - 业务规则：一次性支付或分期支付

2. **计时收费**：按律师工作时间计费
   - 适用场景：诉讼案件、复杂非诉项目
   - 业务规则：小时费率 × 有效工时，按月结算

3. **风险代理**：按案件结果比例收费
   - 适用场景：诉讼案件，特别是经济纠纷
   - 业务规则：基础费用 + 风险比例 × 回款金额
   - 风险比例阶梯：10万以下30%，10-50万35%，50-100万40%，100万以上45%

4. **混合收费**：固定费用 + 风险比例
   - 适用场景：重大复杂案件
   - 业务规则：预付基础费用，案件成功后按比例提成

#### 2.1.2 合同状态流转

```
草稿(DRAFT)
  ↓ 提交审批
待审批(PENDING_APPROVAL)
  ↓ 审批通过
生效(ACTIVE) → 履行中(IN_PROGRESS) → 已完成(COMPLETED)
  ↓ 审批驳回              ↓ 提前终止
已驳回(REJECTED)         已终止(TERMINATED)
  ↓ 合同变更
变更中(MODIFYING) → 已变更(MODIFIED)
  ↓ 到期
已到期(EXPIRED)
```

#### 2.2.2 OCR智能识别规则

1. **识别字段优先级**：
   - 必识别字段：金额、付款方名称、收款日期
   - 可选识别字段：备注、银行账号、交易流水号

2. **金额识别规则**：
   - 支持中文大写数字识别：壹、贰、叁...
   - 支持货币符号：¥、$、€
   - 支持多种格式：100,000.00、100000、10万元

3. **付款方匹配规则**：
   - 精确匹配：客户全称完全一致
   - 模糊匹配：包含关键字段（去除"有限公司"等后缀）
   - 关联匹配：通过联系人、电话等信息关联

#### 2.2.3 智能匹配算法

```java
// 匹配分数计算模型
public class MatchScoreCalculator {
    
    // 金额匹配权重：40%
    private static final BigDecimal AMOUNT_WEIGHT = new BigDecimal("0.4");
    // 付款方匹配权重：35%
    private static final BigDecimal PAYER_WEIGHT = new BigDecimal("0.35");
    // 时间匹配权重：15%
    private static final BigDecimal TIME_WEIGHT = new BigDecimal("0.15");
    // 备注匹配权重：10%
    private static final BigDecimal REMARK_WEIGHT = new BigDecimal("0.1");
    
    public BigDecimal calculateScore(PaymentRecord payment, Matter matter) {
        BigDecimal score = BigDecimal.ZERO;
        
        // 1. 金额匹配
        BigDecimal amountScore = calculateAmountScore(payment.getAmount(), matter.getPendingAmount());
        score = score.add(amountScore.multiply(AMOUNT_WEIGHT));
        
        // 2. 付款方匹配
        BigDecimal payerScore = calculatePayerScore(payment.getPayer(), matter.getClientName());
        score = score.add(payerScore.multiply(PAYER_WEIGHT));
        
        // 3. 时间匹配（收款日期在合同期限内）
        BigDecimal timeScore = calculateTimeScore(payment.getPaymentDate(), matter.getContractPeriod());
        score = score.add(timeScore.multiply(TIME_WEIGHT));
        
        // 4. 备注匹配
        BigDecimal remarkScore = calculateRemarkScore(payment.getRemark(), matter.getMatterNo());
        score = score.add(remarkScore.multiply(REMARK_WEIGHT));
        
        return score.setScale(4, RoundingMode.HALF_UP);
    }
    
    private BigDecimal calculateAmountScore(BigDecimal paymentAmount, BigDecimal pendingAmount) {
        if (paymentAmount == null || pendingAmount == null) {
            return BigDecimal.ZERO;
        }
        
        BigDecimal diff = paymentAmount.subtract(pendingAmount).abs();
        BigDecimal ratio = diff.divide(pendingAmount, 4, RoundingMode.HALF_UP);
        
        if (ratio.compareTo(new BigDecimal("0.01")) <= 0) {
            return BigDecimal.ONE;  // 误差1%以内，满分
        } else if (ratio.compareTo(new BigDecimal("0.05")) <= 0) {
            return new BigDecimal("0.8");  // 误差5%以内，80%
        } else if (ratio.compareTo(new BigDecimal("0.1")) <= 0) {
            return new BigDecimal("0.5");  // 误差10%以内，50%
        } else {
            return BigDecimal.ZERO;
        }
    }
    
    private BigDecimal calculatePayerScore(String payer, String clientName) {
        if (payer == null || clientName == null) {
            return BigDecimal.ZERO;
        }
        
        // 标准化处理
        String normalizedPayer = normalizeName(payer);
        String normalizedClient = normalizeName(clientName);
        
        if (normalizedPayer.equals(normalizedClient)) {
            return BigDecimal.ONE;  // 完全匹配
        } else if (normalizedPayer.contains(normalizedClient) || normalizedClient.contains(normalizedPayer)) {
            return new BigDecimal("0.8");  // 包含关系
        } else {
            // 计算相似度
            double similarity = calculateStringSimilarity(normalizedPayer, normalizedClient);
            return BigDecimal.valueOf(similarity);
        }
    }
}
```

```java
    PARTIAL("部分核销", "金额部分核销");
    
    private final String name;
    private final String description;
    
    ReconciliationStatus(String name, String description) {
        this.name = name;
        this.description = description;
    }
    
    // 状态流转验证
    public boolean canTransferTo(ReconciliationStatus targetStatus) {
        switch (this) {
            case PENDING:
                return targetStatus == AUTO_MATCHED || 
                       targetStatus == MANUAL_MATCHED || 
                       targetStatus == REJECTED;
            case AUTO_MATCHED:
            case MANUAL_MATCHED:
                return targetStatus == CONFIRMED || 
                       targetStatus == REJECTED ||
                       targetStatus == PARTIAL;
            case PARTIAL:
                return targetStatus == CONFIRMED || 
                       targetStatus == REJECTED;
            default:
                return false;
        }
    }
}
```

### 2.3 提成计算管理

#### 2.3.1 提成计算业务规则

**核心公式**：
```
净收入 = 到账金额 - 税费 - 管理费 - 办案成本
个人净收入 = 净收入 × 个人分配比例
提成金额 = 个人净收入 × 阶梯提成比例
```

**详细计算步骤**：

1. **计算税费**：
   - 增值税：6%（小规模纳税人）
   - 附加税：增值税的12%
   - 合计税率：6.72%

2. **计算管理费**：
   - 律所管理费：净收入的10-15%
   - 团队管理费：净收入的5-10%
   - 合计管理费率：15-25%

3. **计算办案成本**：
   - 诉讼费、保全费、鉴定费、差旅费、打印费等
   - 按实际发生额计算

4. **分配比例确定**：
   - 主办律师：40-60%
   - 协办律师：20-40%
   - 案源律师：10-20%
   - 助理律师：0-10%

5. **阶梯提成比例**：
   ```
   个人净收入 ≤ 10万：30%
   10万 < 个人净收入 ≤ 50万：35%
   50万 < 个人净收入 ≤ 100万：40%
   个人净收入 > 100万：45%
   ```

#### 2.3.3 提成计算服务

> **重要**：提成计算必须考虑律师的**薪酬模式（compensationType）**。
> - 提成制律师（COMMISSION）：正常计算提成
> - 授薪制律师（SALARIED）：不参与提成，分配比例视为0
> - 混合制律师（HYBRID）：正常计算提成

```java
// 提成计算服务
@Service
@Slf4j
@RequiredArgsConstructor
public class CommissionCalculationService {
    
    private final CommissionRuleRepository ruleRepository;
    private final CommissionRepository commissionRepository;
    private final UserRepository userRepository;
    private final PaymentRepository paymentRepository;
    private final MatterRepository matterRepository;
    private final MatterParticipantRepository matterParticipantRepository;
    
    @Transactional
    public CommissionCalculationResult calculateCommission(Long paymentId) {
        // 1. 获取收款信息
        Payment payment = paymentRepository.findById(paymentId)
                .orElseThrow(() -> new BusinessException("收款记录不存在"));
        
        // 2. 获取项目信息
        Matter matter = matterRepository.findById(payment.getMatterId())
                .orElseThrow(() -> new BusinessException("项目不存在"));
        
        // 3. 获取项目参与人（含薪酬模式信息）
        List<MatterParticipant> participants = matterParticipantRepository
                .findByMatterId(matter.getId());
        
        // 4. 过滤出提成制律师（排除授薪制）
        List<MatterParticipant> commissionEligible = filterCommissionEligible(participants);
        
        // 5. 重新计算分配比例（排除授薪制后重新归一化）
        normalizeDistributionRatio(commissionEligible);
        
        // 6. 获取适用的提成规则
        CommissionRule rule = findApplicableRule(matter, participants);
        
        // 7. 计算办案成本
        BigDecimal costAmount = calculateCostAmount(matter.getId());
        
        // 8. 计算每个提成制参与人的提成
        List<Commission> commissions = new ArrayList<>();
        
        for (MatterParticipant participant : commissionEligible) {
            Commission commission = calculateIndividualCommission(
                    payment, matter, participant, rule, costAmount);
            commissions.add(commission);
        }
        
        // 9. 为授薪律师生成0提成记录（用于查询完整性）
        for (MatterParticipant participant : participants) {
            if (!isCommissionEligible(participant.getUserId())) {
                commissions.add(createZeroCommission(payment, matter, participant, rule));
            }
        }
        
        // 10. 保存提成记录
        commissionRepository.saveBatch(commissions);
        
        // 11. 返回计算结果
        return new CommissionCalculationResult(paymentId, commissions);
    }
    
    /**
     * 过滤出有提成资格的参与人（提成制或混合制）
     */
    private List<MatterParticipant> filterCommissionEligible(List<MatterParticipant> participants) {
        return participants.stream()
                .filter(p -> isCommissionEligible(p.getUserId()))
                .collect(Collectors.toList());
    }
    
    /**
     * 判断律师是否有提成资格
     */
    private boolean isCommissionEligible(Long userId) {
        User user = userRepository.findById(userId).orElse(null);
        if (user == null) {
            return false;
        }
        // 提成制或混合制才有提成资格
        String compType = user.getCompensationType();
        return "COMMISSION".equals(compType) || "HYBRID".equals(compType);
    }
    
    /**
     * 归一化分配比例（排除授薪制后，重新计算提成制律师的分配比例）
     * 
     * 例如：原分配 主办60%（提成）+ 协办A 30%（授薪）+ 协办B 10%（提成）
     * 排除授薪后：主办 60/(60+10)=85.7% + 协办B 10/(60+10)=14.3%
     */
    private void normalizeDistributionRatio(List<MatterParticipant> eligibleParticipants) {
        BigDecimal totalRatio = eligibleParticipants.stream()
                .map(MatterParticipant::getDistributionRatio)
                .reduce(BigDecimal.ZERO, BigDecimal::add);
        
        if (totalRatio.compareTo(BigDecimal.ZERO) > 0) {
            for (MatterParticipant p : eligibleParticipants) {
                BigDecimal normalizedRatio = p.getDistributionRatio()
                        .divide(totalRatio, 4, RoundingMode.HALF_UP);
                p.setNormalizedRatio(normalizedRatio); // 临时字段，用于本次计算
            }
        }
    }
    
    private Commission calculateIndividualCommission(Payment payment, Matter matter,
                                                     MatterParticipant participant,
                                                     CommissionRule rule, BigDecimal costAmount) {
        // 使用归一化后的比例
        BigDecimal ratio = participant.getNormalizedRatio() != null 
                ? participant.getNormalizedRatio() 
                : participant.getDistributionRatio();
        
        // 计算净收入
        BigDecimal grossAmount = payment.getAmount();
        BigDecimal taxAmount = grossAmount.multiply(rule.getTaxRate());
        BigDecimal managementFee = grossAmount.multiply(rule.getManagementFeeRate());
        BigDecimal netAmount = grossAmount.subtract(taxAmount)
                .subtract(managementFee)
                .subtract(costAmount);
        
        // 个人净收入
        BigDecimal individualNetAmount = netAmount.multiply(ratio);
        
        // 根据阶梯计算提成比例
        BigDecimal commissionRate = calculateTieredRate(individualNetAmount, rule.getRateTiers());
        
        // 提成金额
        BigDecimal commissionAmount = individualNetAmount.multiply(commissionRate);
        
        return Commission.builder()
                .paymentId(payment.getId())
                .matterId(matter.getId())
                .userId(participant.getUserId())
                .ruleId(rule.getId())
                .grossAmount(grossAmount.multiply(ratio))
                .taxAmount(taxAmount.multiply(ratio))
                .managementFee(managementFee.multiply(ratio))
                .costAmount(costAmount.multiply(ratio))
                .netAmount(individualNetAmount)
                .distributionRatio(ratio)
                .commissionRate(commissionRate)
                .commissionAmount(commissionAmount)
                .compensationType(getUserCompensationType(participant.getUserId()))
                .status("CALCULATED")
                .build();
    }
    
    /**
     * 为授薪律师生成0提成记录
     */
    private Commission createZeroCommission(Payment payment, Matter matter,
                                             MatterParticipant participant,
                                             CommissionRule rule) {
        return Commission.builder()
                .paymentId(payment.getId())
                .matterId(matter.getId())
                .userId(participant.getUserId())
                .ruleId(rule.getId())
                .grossAmount(BigDecimal.ZERO)
                .taxAmount(BigDecimal.ZERO)
                .managementFee(BigDecimal.ZERO)
                .costAmount(BigDecimal.ZERO)
                .netAmount(BigDecimal.ZERO)
                .distributionRatio(BigDecimal.ZERO)
                .commissionRate(BigDecimal.ZERO)
                .commissionAmount(BigDecimal.ZERO)
                .compensationType("SALARIED")
                .status("SALARIED_EXEMPT") // 授薪免提成
                .remark("授薪律师不参与项目提成分配")
                .build();
    }
    
    private String getUserCompensationType(Long userId) {
        return userRepository.findById(userId)
                .map(User::getCompensationType)
                .orElse("COMMISSION");
    }
    
    private BigDecimal calculateTieredRate(BigDecimal amount, List<RateTier> tiers) {
        for (RateTier tier : tiers) {
            boolean aboveMin = amount.compareTo(tier.getMinAmount()) >= 0;
            boolean belowMax = tier.getMaxAmount() == null || 
                               amount.compareTo(tier.getMaxAmount()) < 0;
            
            if (aboveMin && belowMax) {
                return tier.getRate();
            }
        }
        // 默认返回最高档
        return tiers.get(tiers.size() - 1).getRate();
    }
}
```
