# 测试执行台账模板

用于记录每一轮测试活动的执行情况，便于后续追踪版本、环境、范围、结果和复测闭环。

## 字段说明

| 字段 | 说明 |
| --- | --- |
| 台账编号 | 每轮测试唯一编号 |
| 测试日期 | 执行日期 |
| 客户名称 | 当前客户名称 |
| 环境 | 测试环境说明 |
| 部署版本 | 面向部署的版本号 |
| 镜像标签 | 本次部署镜像标签 |
| 提交号 | 对应 Git 提交短哈希 |
| 测试类型 | 冒烟 / 回归 / 验收 / 联调 |
| 测试范围 | 本轮覆盖的业务范围 |
| 执行人 | 执行测试的人 |
| 结果 | 通过 / 部分通过 / 失败 |
| 关联缺陷 | 缺陷台账编号 |
| 是否回滚 | 是否触发回滚 |
| 备注 | 关键留痕 |
