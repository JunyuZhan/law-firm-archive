# 发布前验收清单

本文档用于镜像发布、客户首次部署或升级前的统一核查。

## 1. 版本与镜像

- [ ] 已确认本次发布版本号，例如 `v0.1.7`
- [ ] 已确认 Git 提交号
- [ ] 后端镜像已按版本号打标签
- [ ] 前端镜像已按版本号打标签
- [ ] 如有 ES 定制镜像，已按版本号打标签
- [ ] 已记录 `APP_VERSION`
- [ ] 已记录 `APP_COMMIT_SHA`
- [ ] 已记录 `APP_BUILD_TIME`
- [ ] 已确认私有仓库镜像可拉取

## 2. 初始化与配置

- [ ] 已确认初始化脚本入口正确
- [ ] 已确认默认角色、菜单和基础配置完整
- [ ] 已确认部署目录与源码目录分离
- [ ] 已确认 `.env` 或环境变量配置完整
- [ ] 已确认客户域名、CORS、对象存储代理前缀配置正确

## 3. 备份与回滚

- [ ] 已完成升级前备份
- [ ] 已确认回滚所需上一稳定版本镜像可拉取
- [ ] 已确认回滚步骤和负责人

## 4. 测试与验收

- [ ] 已执行最小回归集
- [ ] 已执行部署后冒烟测试
- [ ] 已更新测试台账
- [ ] 已更新缺陷台账
- [ ] 已确认本次遗留问题和风险范围

## 5. 发布结论

- [ ] 允许发布
- [ ] 有条件发布
- [ ] 暂缓发布
