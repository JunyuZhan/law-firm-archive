## 📝 建议配置

### 1. 启用语言切换按钮

如果需要显示语言切换按钮，可以在 `preferences.ts` 中配置：

```typescript
export const overridesPreferences = defineOverridesPreferences({
  app: {
    name: import.meta.env.VITE_APP_TITLE,
    accessMode: 'backend',
    locale: 'zh-CN', // 可选：显式设置默认语言
  },
  widget: {
    languageToggle: true, // 启用语言切换按钮
  },
});
```

### 2. 设置默认语言

如果需要修改默认语言为英文：

```typescript
export const overridesPreferences = defineOverridesPreferences({
  app: {
    name: import.meta.env.VITE_APP_TITLE,
    accessMode: 'backend',
    locale: 'en-US', // 设置为英文
  },
});
```

## 📝 建议改进

### 1. 将硬编码文本国际化

**工作台页面示例** (`frontend/apps/web-antd/src/views/dashboard/index/index.vue`):

当前（硬编码）:
```vue
<Page title="工作台" description="欢迎使用智慧律所管理系统">
  <h2>欢迎使用智慧律所管理系统</h2>
  <Statistic title="我的项目" :value="stats.matterCount" suffix="个">
```

建议（国际化）:
```vue
<Page :title="$t('page.dashboard.title')" :description="$t('page.dashboard.description')">
  <h2>{{ $t('page.dashboard.welcome') }}</h2>
  <Statistic :title="$t('page.dashboard.myMatters')" :value="stats.matterCount" :suffix="$t('page.dashboard.unit')">
```

需要在语言包中添加对应翻译：
- `page.dashboard.title`: "工作台" / "Dashboard"
- `page.dashboard.description`: "欢迎使用智慧律所管理系统" / "Welcome to Law Firm Management System"
- `page.dashboard.welcome`: "欢迎使用智慧律所管理系统" / "Welcome to Law Firm Management System"
- `page.dashboard.myMatters`: "我的项目" / "My Matters"
- `page.dashboard.unit`: "个" / ""

### 2. 后端国际化（可选）

如果需要后端错误消息国际化，需要配置 Spring MessageSource：
- 创建 `messages_zh_CN.properties` 和 `messages_en_US.properties`
- 配置 `LocaleResolver` 和 `MessageSource`
- 在 Controller 中使用 `MessageSource` 获取翻译
