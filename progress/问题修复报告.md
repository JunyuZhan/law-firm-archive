### 1. 验证码接口异常（P0）✅

**问题描述**:
- 接口: `GET /api/auth/captcha`
- 错误: 系统繁忙，请稍后重试（500错误）
- 原因: 使用 `new Font("Arial", Font.BOLD, 25)` 在无头环境中字体不可用

**修复方案**:
- 修改 `CaptchaService.generateCaptcha()` 方法
- 使用系统默认字体 `Font.SANS_SERIF` 替代 `Arial`
- 添加异常处理，捕获字体设置失败的情况
- 添加全局异常捕获，避免500错误

**修复文件**:
- `backend/src/main/java/com/lawfirm/application/system/service/CaptchaService.java`

**修复内容**:
```java
// 修复前
captcha.setFont(new Font("Arial", Font.BOLD, 25));

// 修复后
try {
    Font font = new Font(Font.SANS_SERIF, Font.BOLD, 25);
    captcha.setFont(font);
} catch (Exception e) {
    log.warn("设置验证码字体失败，使用默认字体", e);
}
```
