## ⚠️ 存在问题的模块

### 1. 文档模板模块 ⚠️

#### 问题: 更新模板接口参数格式不匹配

**前端实现** (`frontend/apps/web-antd/src/api/document/template.ts:86-88`):
```typescript
export function updateTemplate(id: number, data: UpdateDocumentTemplateCommand) {
  return requestClient.put<DocumentTemplateDTO>(`/document/template/${id}`, data);
}
```
- 使用 PUT 请求体传递 `UpdateDocumentTemplateCommand` 对象

**后端实现** (`backend/src/main/java/com/lawfirm/interfaces/rest/document/DocumentTemplateController.java:66-76`):
```java
@PutMapping("/{id}")
public Result<DocumentTemplateDTO> update(@PathVariable Long id,
                                          @RequestParam(required = false) String name,
                                          @RequestParam(required = false) Long categoryId,
                                          // ... 多个 @RequestParam 参数
                                          ) {
    // ...
}
```
- 使用多个 `@RequestParam` 参数

**影响**: ❌ 前端无法成功调用后端更新接口

**解决方案**: 修改后端，使用 `@RequestBody UpdateDocumentTemplateCommand`
