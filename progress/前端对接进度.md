## 对接步骤

### 1. 创建API定义文件
在 `frontend/apps/web-antd/src/api/` 下创建对应的API文件：
- `system/user.ts` - 用户管理
- `crm/client.ts` - 客户管理
- `matter/matter.ts` - 项目管理
- `finance/contract.ts` - 合同管理
- 等等...

### 2. 定义TypeScript类型
根据后端DTO定义前端类型：
```typescript
export namespace ClientApi {
  export interface ClientDTO {
    id: number;
    name: string;
    // ...
  }
}
```

### 3. 实现API函数
```typescript
export async function getClientListApi(params: PageQuery) {
  return requestClient.get<PageResult<ClientApi.ClientDTO>>('/client/list', params);
}
```

### 4. 在页面中使用
```typescript
import { getClientListApi } from '#/api';

const list = await getClientListApi({ page: 1, size: 10 });
```

## 注意事项

1. **统一响应格式**：后端统一使用 `Result<T>` 格式，前端已配置拦截器
2. **分页参数**：统一使用 `page` 和 `size`，返回 `PageResult<T>`
3. **权限控制**：前端需要根据用户权限显示/隐藏功能
4. **错误处理**：统一在拦截器中处理，页面只需处理业务逻辑
5. **类型安全**：所有API都要定义TypeScript类型
