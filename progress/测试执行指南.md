## 🐛 常见问题

### 问题1：后端服务启动失败

**可能原因**：
- 数据库连接失败
- 端口被占用
- 依赖缺失

**解决方法**：
```bash
# 检查数据库连接
docker exec law-postgres psql -U law_admin -d law_firm_dev -c "SELECT 1;"

# 检查端口占用
lsof -i :8080

# 重新编译
cd backend
mvn clean install
```

### 问题2：前端无法连接后端

**可能原因**：
- 后端服务未启动
- API地址配置错误
- CORS配置问题

**解决方法**：
- 检查后端服务是否运行
- 检查 `frontend/apps/web-antd/src/api/request.ts` 中的 `baseURL` 配置
- 检查后端CORS配置

### 问题3：登录失败

**可能原因**：
- 用户名密码错误
- 数据库未初始化
- JWT配置错误

**解决方法**：
```bash
# 检查初始用户数据
docker exec law-postgres psql -U law_admin -d law_firm_dev -c "SELECT username FROM sys_user;"

# 默认账号：admin / admin123
```
