## 🔍 验证方法

### 数据库验证
```sql
-- 检查菜单是否存在
SELECT id, name, path, component, parent_id, sort_order, visible, status
FROM sys_menu 
WHERE path IN (
    '/admin/supplier',
    '/admin/overtime', 
    '/admin/asset-inventory',
    '/admin/go-out',
    '/admin/meeting-record',
    '/hr/regularization',
    '/hr/resignation'
)
ORDER BY parent_id, sort_order;

-- 检查权限分配
SELECT r.role_code, m.name, m.path
FROM sys_role_menu rm
JOIN sys_role r ON rm.role_id = r.id
JOIN sys_menu m ON rm.menu_id = m.id
WHERE m.path IN (
    '/admin/supplier',
    '/admin/overtime', 
    '/admin/asset-inventory',
    '/admin/go-out',
    '/admin/meeting-record',
    '/hr/regularization',
    '/hr/resignation'
)
ORDER BY r.role_code, m.path;
```
