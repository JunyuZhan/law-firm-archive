# 部署脚本

部署相关的脚本集合。

## 脚本列表

- `deploy.sh` - 一键 Docker 部署（推荐）
- `deploy-swarm.sh` - Docker Swarm 集群部署
- `deploy-to-server.sh` - 快速部署到服务器
- `force-update-server.sh` - 强制更新服务器代码
- `pre-deploy-check.sh` - 部署前统一检查（推荐）
- `check-production-ready.sh` - 生产环境检查（旧版）
