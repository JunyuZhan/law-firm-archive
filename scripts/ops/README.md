# 运维脚本

运维相关的脚本集合，包括备份、恢复、环境管理等。

## 脚本列表

- `backup.sh` - 完整备份
- `db-auto-backup.sh` - 数据库自动备份
- `restore.sh` - 数据恢复
- `reset-db.sh` - 数据库重置（开发用）
- `init-demo-data.sh` - 初始化示例数据
- `env-start.sh` - 环境启动
- `env-stop.sh` - 环境停止
- `env-reset.sh` - 环境重置
- `clean-docker.sh` - 清理项目Docker资源（推荐）
- `clean-law-firm-only.sh` - 仅清理律所系统资源
